### 一、 平台运营与规则 (Platform Operations & Rules)

**通用规则与概念**

* **Q0016, Q0184: 什么是对赌协议？涉及哪些方面？**

  * **A:** 对赌协议通常涉及承诺达到某个目标（如GMV），如未达到则需承担相应后果（如赔款）。这常见于融资、主播签约、代运营等方面。对于个人博主与MCN签约，较少涉及严格的对赌，但可能存在业绩目标。自己做账号则不存在对赌。

* **Q0030: IP切片是什么意思？**

  * **A:** 将一个长视频或直播内容，剪辑成多个短视频片段进行分发，以扩大覆盖面和吸引不同兴趣的用户。

* **Q0045: 公域流量和私域流量的区别是什么？**

  * **A:** 公域流量指平台分配的流量（如抖音、小红书的推荐流），难以直接掌控；私域流量指自己可控的流量池（如微信群、公众号、个人微信号），可以反复触达。

* **Q0048: MCN机构是什么？**

  * **A:** Multi-Channel Network，帮助创作者进行内容创作、推广、变现等服务的机构。

* **Q0051: 蓝V认证有什么用？**

  * **A:** 主要是企业身份的象征，对流量和限流没有直接影响。

* **Q0110: 什么是矩形号（矩阵号）？**

  * **A:** 指一个人或团队运营多个账号，形成账号矩阵，覆盖不同细分领域或内容形式，以扩大影响力和抗风险能力。

* **Q0130: 怎么判断笔记被限流了？**

  * **A:** 主要看“非粉丝阅读占比”。如果这个比例远低于账号平均水平（如低于10%甚至5%），可能被限流。另外，长时间（如超过24小时）阅读量几乎不增长也可能是迹象。

* **Q0160: 什么是“养号”？**

  * **A:** 指新账号或低权重账号在使用初期，模仿正常用户行为（浏览、点赞、评论、收藏），以提高账号活跃度和平台信任度，有助于后续内容获得更好推荐。

* **Q0178: 什么是个人IP？**

  * **A:** 指个人在特定领域建立的具有独特性、辨识度和影响力的形象或品牌。

* **Q0241: 笔记掉收录是什么意思？怎么解决？**

  * **A:** 指之前能搜到的笔记现在搜不到了。可能是平台规则调整或笔记内容问题。解决方法包括检查内容是否违规、尝试修改笔记、观察一段时间或咨询客服。

**小红书平台**

* **Q0004, Q0009, Q0152, Q0197: 小红书笔记发布后多久能看到数据反馈？一般多久会“爆”？流量高峰期是多久？**

  * **A:** 数据反馈很快，发布后几分钟就能看到阅读量。笔记没有固定的“爆”的时间，可能几小时、几天甚至几周后才开始起量。流量高峰期通常持续几天到一周左右，之后会逐渐下降。有些长尾内容的笔记可能会持续有流量。

* **Q0010: 小红书的推荐机制是怎样的？会不断推给新人吗？**

  * **A:** 平台的推荐系统复杂，会根据内容质量、用户标签、互动数据等多种因素进行分发。不是简单地只推给新人，而是基于匹配度推送给可能感兴趣的用户。

* **Q0012: 小红书上如何判断某个账号是否值得对标？**

  * **A:** 看对方的内容质量、数据表现（赞藏评）、粉丝增长趋势、商业变现情况等。选择与自己定位相似、且数据表现优秀的账号进行学习。

* **Q0020, Q0058, Q0094: 小红书个人号/非专业号可以接广告（恰饭）吗？需要报备吗？**

  * **A:** 理论上个人号不能直接接广告。通过平台（如蒲公英）接单需要专业号资质。私下接单（软广植入）存在风险，可能被判违规。建议通过官方渠道或谨慎操作。非报备广告被发现可能导致笔记限流、删帖甚至封号。

* **Q0022: 小红书笔记发布后可以修改吗？修改会影响流量吗？**

  * **A:** 可以修改，但频繁修改或大幅修改可能影响笔记权重和推荐。建议发布前仔细检查，小幅修改（如错别字）影响较小。修改图片通常比修改文字影响更大。

* **Q0025: 笔记被判定违规怎么办？可以申诉吗？**

  * **A:** 如果认为误判，可以进行申诉。提供合理解释和证据可能成功。多次违规会影响账号权重。

* **Q0026: 小红书笔记适合用夸张的标题吗？**

  * **A:** 可以适当使用吸引眼球的标题，但要避免过度夸张、题文不符或违规词汇，否则可能被判定为“标题党”或违规。真实、有价值的内容更重要。

* **Q0037: 小红书的“盒子”功能还在吗？**

  * **A:** 指的是之前的“专辑”功能，后来可能调整或改名，具体以平台当前功能为准。（信息可能过时）

* **Q0039: 小红书笔记选题如何做？**

  * **A:** 参考爆款笔记、挖掘用户痛点、结合热点、从自身专业领域出发、分析对标账号选题等。

* **Q0047, Q0096, Q0181: 小红书如何引流到私域（微信）？**

  * **A:** 这是平台的灰色地带，直接留微信号容易违规。常用方法包括：&#x20;

    * 简介引导（“私信”、“感谢关注”等暗示）。

    * 瞬间功能引导。

    * 小号评论区引导。

    * 群聊功能引导。

    * 私信沟通，用谐音、图片、分段发送等方式规避检测。

    * 在保证内容价值的前提下，吸引用户主动询问。

    * **注意：** 任何引流方式都有风险，需谨慎操作。

* **Q0060: 小红书的“专业号”有什么用？必须开吗？**

  * **A:** 专业号提供更多功能，如数据分析、营销工具（蒲公英接单）、粉丝管理等。如果想通过平台接广告或进行更专业的数据分析，建议开通。个人分享生活、不以商业变现为主要目的则非必需。

* **Q0063: 小红书笔记石沉大海（没流量）是什么原因？**

  * **A:** 可能原因：内容质量不高、选题不热门、未击中用户痛点、账号权重低、涉及违规内容、未被系统推荐等。

* **Q0064, Q0079, Q0118: 如何提高小红书笔记的互动率（点赞、收藏、评论）？**

  * **A:** 提高内容价值（干货、教程、经验分享）、优化封面和标题吸引点击、在文末设置引导性问题或互动话题、及时回复评论、参与评论区互动等。

* **Q0075: 小红书笔记配图用AI生成的图片可以吗？**

  * **A:** 可以，但要注意图片质量和版权问题。平台更鼓励真实、原创的图片。AI图片如果风格独特、与内容契合，也可能受欢迎。

* **Q0080: 小红书企业号和个人号有什么区别？**

  * **A:** 企业号需要营业执照认证，有更多商业功能（如店铺、广告投放），适合企业品牌推广；个人号门槛低，适合个人分享和IP打造。

* **Q0115: 小红书直播需要什么条件？**

  * **A:** 通常需要完成实名认证，并满足一定的粉丝数量或平台邀请等条件（具体条件可能变化，需参考平台最新规则）。

* **Q0119: 小红书笔记的“收录”是什么意思？**

  * **A:** 指笔记能被小红书的搜索功能检索到。笔记发布后需要一段时间才会被收录。未被收录的笔记无法通过搜索找到。

* **Q0125: 小红书可以多个账号互相引流吗？**

  * **A:** 可以，例如大号给小号引流，或者矩阵号之间互推，但要自然，避免过度营销感。

* **Q0134: 小红书发视频好还是图文好？**

  * **A:** 两者各有优势。图文适合深度信息、教程、攻略等；视频适合展示过程、氛围、真人出镜等。选择哪种形式取决于内容本身和目标受众偏好。可以尝试不同形式看效果。

* **Q0136, Q0210: 小红书笔记数据不好，需要删除重发吗？**

  * **A:** 不建议直接删除。删除可能影响账号权重。可以尝试修改优化（标题、封面、内容），或者隐藏后观察。如果确定内容或选题有问题，可以考虑重新创作发布新笔记。

* **Q0147: 小红书官方让修改笔记怎么办？**

  * **A:** 按照官方提示进行修改，通常是内容或形式存在不合规之处。及时修改可以避免进一步处罚。

* **Q0154: 小红书笔记审核一般多久？**

  * **A:** 审核时间不定，快的几分钟，慢的可能几小时甚至更长。敏感内容或平台审核压力大时会慢一些。

* **Q0164: 小红书上发纯文字笔记效果如何？**

  * **A:** 效果通常不如图文或视频笔记，因为小红书是视觉驱动平台。但如果文字内容极具价值或洞察，也可能获得关注。

* **Q0174: 小红书账号被封怎么办？**

  * **A:** 严重违规可能导致永久封号，一般无法解封。如果是误封，可以尝试联系客服申诉。

* **Q0188: 小红书笔记可以带第三方平台链接吗？**

  * **A:** 通常不允许直接带站外链接，特别是营销导流链接，容易违规。

* **Q0194: 如何看小红书笔记是否被收录？**

  * **A:** 复制笔记的完整标题或核心关键词，在小红书搜索框中搜索，看能否找到自己的笔记。也可以让别人搜索试试。

* **Q0201: 多个小红书账号用同一WiFi发布，会被判定为营销号吗？**

  * **A:** 有一定风险，但不绝对。平台会综合判断账号行为。如果内容正常、互动真实，风险较低。避免频繁切换账号、发布相似内容等可疑行为。

* **Q0222: 小红书可以模仿别人的爆款笔记吗？**

  * **A:** 可以学习和借鉴爆款笔记的选题、结构、封面、标题等优点，但要进行差异化和原创性改造，不能完全照搬，否则可能被判抄袭或效果不佳。

* **Q0285: 小红书直播违规，账号被限制1个月怎么办？**

  * **A:** 限制一个月属于重度违规，申诉通过概率低，但仍可尝试申诉（群资料里有申诉方法）。大部分情况只能等待限制期结束。

* **Q0286: 小红书发布语法练习册内容，应放在哪个类目？有英语资质放在英语类可以吗？**

  * **A:** 如果有英语资质，放在“教育-语言学习”或相关类目比较合适。具体分类需参考平台最新的分类体系。

**其他平台（微信、抖音等）**

* **Q0007: 视频号如何利用私域流量？**

  * **A:** 通过视频号直播、内容引导用户加入微信群、添加个人微信，或引导关注公众号，将公域流量转化为私域流量进行维护和变现。

* **Q0017: 视频号挂载公众号文章，会影响视频流量吗？**

  * **A:** 合理挂载一般不直接影响视频流量分发，但转化效果取决于文章内容与视频的契合度以及用户的兴趣。

* **Q0072: 视频号直播如何提高转化率？**

  * **A:** 优化直播内容、提升主播表现力、设计互动环节、提供限时优惠、清晰引导下单路径、利用福袋/抽奖等工具。

* **Q0123: 抖音如何运营？**

  * **A:** 这是一个非常宽泛的问题，涉及定位、内容创作、流量获取、变现等多个方面。需要根据具体目标和资源制定策略。（原回答可能指向具体课程或资料）

* **Q0144: 微信公众号文章如何提高阅读量？**

  * **A:** 优化标题、选题贴合用户需求、提高内容质量、排版美观、引导分享、多渠道推广（朋友圈、微信群、其他平台引流）等。

### 二、 内容创作 (Content Creation)

**选题与策划**

* **Q0036: 知识付费内容如何寻找选题？**

  * **A:** 分析目标用户的痛点和需求、研究竞品内容、关注行业热点和趋势、从自身专业经验和知识体系出发、通过问答社区或社群挖掘问题等。

* **Q0043: 如何判断一个选题是否适合自己做？**

  * **A:** 评估自己对该选题的兴趣度、专业度（是否有足够知识储备）、可执行性（是否有能力创作相关内容）、市场需求（用户是否关注）、以及变现潜力。

* **Q0067: 灵感枯竭怎么办？**

  * **A:** 阅读行业报告、看书、刷对标账号、与人交流、体验生活、回顾过往内容数据、利用思维导图或灵感收集工具等。

* **Q0069: 内容创作如何保持垂直度？**

  * **A:** 明确账号定位和核心主题，围绕主题进行选题和内容延伸，避免发布与定位无关的内容，保持内容风格和调性统一。

* **Q0117: 如何找到差异化的内容切入点？**

  * **A:** 细分领域、特定人群、独特视角、不同形式（如图文 vs 视频）、结合个人经历或特色、解决别人未解决的问题等。

**形式与技巧**

* **Q0011, Q0077: 如何制作吸引人的内容封面？**

  * **A:** 清晰突出主题、构图美观、色彩搭配协调、加入吸引人的文字或元素、有人物尽量用真人（看情况）、保持风格统一。可以多看爆款笔记封面找感觉。

* **Q0068: 如何写出“爆款”文案？**

  * **A:** 深入了解用户心理、使用痛点词或利益词、设置悬念或冲突、结构清晰（如总分总）、语言简洁易懂、加入金句或强观点、有明确的引导（点赞、收藏、评论、关注）。

* **Q0070: 视频口播如何更有吸引力？**

  * **A:** 准备好逐字稿、注意语速和停顿、带有情绪和表情、眼神与镜头交流、仪态自然大方、口齿清晰、可以通过后期剪辑优化节奏。

* **Q0071: 如何提升视频内容的完播率？**

  * **A:** 开头黄金几秒抓住用户、内容节奏紧凑不拖沓、信息密度高、画面吸引人、设置悬念或钩子、时长控制得当。

* **Q0073: 图文笔记和视频笔记，哪个更容易涨粉？**

  * **A:** 没有绝对答案，取决于平台特性、内容质量和领域。小红书早期图文更主流，现在视频也很重要。关键是内容能否吸引目标用户。

* **Q0074: 如何给内容增加“网感”？**

  * **A:** 使用网络流行语（适度）、结合热点梗、用轻松幽默的表达方式、排版活泼（如加emoji）、了解目标用户群体的语言习惯。

* **Q0127, Q0243: 什么是“黄金三秒”？如何应用？**

  * **A:** 指视频开头或笔记首图/前几句的关键时刻，必须快速抓住用户注意力，让他们产生继续观看/阅读的兴趣。应用方法：设置悬念、展示核心亮点、提出痛点问题、使用吸睛画面等。

* **Q0141: 如何写出好的种草文案？**

  * **A:** 描述使用场景、突出产品核心卖点和效果、加入真实体验感受、与用户建立共鸣、适当对比、给出明确购买理由。

* **Q0161: 录课程视频需要露脸吗？**

  * **A:** 不一定。露脸可以增加信任感和亲和力，但如果内容本身足够硬核，或者不露脸更符合课程风格（如软件演示），也可以不露脸。

* **Q0179: 如何找到适合自己的内容风格？**

  * **A:** 多尝试不同风格（干货、搞笑、情感、Vlog等），分析自身性格特点和擅长领域，看用户反馈，找到自己做得舒服且用户喜欢的结合点。

* **Q0282: 制作笔记中半透明的黑色背景用什么APP？**

  * **A:** 可以使用 Sketch 等设计工具。类似效果也可以用 Canva（可画）、醒图、PicsArt 等手机 App 实现。

**内容分发与优化**

* **Q0024: 发布内容应该选择什么时间？**

  * **A:** 一般选择目标用户活跃的时间段发布，如工作日中午、下班后、周末。具体最佳时间可以通过测试不同时段的数据反馈来确定。

* **Q0076: 笔记发布后，是否需要人为干预数据（如刷赞）？**

  * **A:** 不建议。平台反对刷量行为，可能被识别并处罚（限流、封号）。真实互动数据更有价值。

* **Q0078: 如何利用数据分析优化内容？**

  * **A:** 分析笔记的阅读量、互动率、涨粉数、用户评论等数据，找出表现好的内容的共性（选题、形式、标题等），总结规律，用于指导后续内容创作。同时也要分析表现不佳的原因。

* **Q0122: 内容可以多平台分发吗？**

  * **A:** 可以，有助于扩大影响力。但要注意各平台调性不同，最好根据平台特点对内容进行适当调整（如标题、格式、标签等）。

### 三、 商业模式与变现 (Business Models & Monetization)

**变现方式**

* **Q0002, Q0032, Q0042, Q0107, Q0114, Q0214: 有哪些常见的自媒体/IP变现方式？**

  * **A:** 主要包括：&#x20;

    * **广告变现:** 接品牌广告（硬广、软广植入）、平台广告分成（如公众号流量主、视频号带货）。

    * **电商带货:** 自己开店卖货、分销他人产品、直播带货。

    * **知识付费:** 卖课程、咨询服务、社群会员、电子书/资料。

    * **平台补贴/签约:** MCN签约、平台活动奖励。

    * **服务变现:** 提供代运营、账号诊断、内容制作等服务。

    * **引流到线下:** 为线下店铺或业务导流。

* **Q0003: 对标账号的变现方式如何分析？**

  * **A:** 观察其内容中是否植入广告、是否引导购买产品/课程、是否有橱窗/店铺链接、是否引导私域流量（可能在私域变现）、是否提及咨询服务等。

* **Q0006: 如何平衡内容创作和商业变现？**

  * **A:** 保持内容价值是基础，不能为了变现而牺牲用户体验。商业植入要自然、与内容调性相符。明确告知广告（如平台要求）。变现频率不宜过高。优先考虑与账号定位和粉丝需求匹配的变现方式。

* **Q0014: 知识付费产品如何定价？**

  * **A:** 参考市场同类产品价格、评估产品价值（解决问题的程度、信息稀缺性）、考虑目标用户购买力、结合自身品牌影响力、考虑交付成本。可以采用测试定价或阶梯定价。

* **Q0021: 如何找到合适的广告主（金主爸爸）？**

  * **A:** 通过平台官方渠道（如小红书蒲公英、抖音星图）、MCN机构对接、主动联系品牌方、在内容或简介中留下合作方式、积累行业人脉。

* **Q0027: 什么是CPS、CPC、CPM、CPT？**

  * **A:** 这些是常见的广告计费模式：&#x20;

    * **CPS (Cost Per Sale):** 按实际销售额付费。

    * **CPC (Cost Per Click):** 按点击次数付费。

    * **CPM (Cost Per Mille/Thousand Impressions):** 按千次展示付费。

    * **CPT (Cost Per Time):** 按时长付费（如包月、包天）。

* **Q0028: 如何提高电商带货的转化率？**

  * **A:** 选品是关键（符合粉丝需求、有竞争力）、优化内容（充分展示产品优点和使用场景）、建立信任感（真实测评、专业形象）、提供优惠或赠品、引导下单路径清晰、利用好平台电商工具（如橱窗、直播购物车）。

* **Q0108: 知识付费社群如何运营？**

  * **A:** 提供高质量的专属内容或服务、营造良好的交流氛围、定期组织活动或答疑、建立有效的管理机制、收集反馈并持续优化。

* **Q0109: 如何设计有吸引力的咨询服务？**

  * **A:** 明确服务内容和边界、突出能解决的核心问题、展示成功案例或专业背景、定价合理透明、服务流程清晰。

* **Q0187: 纯粹分享爱好，可以变现吗？**

  * **A:** 可以。当你的爱好分享吸引了足够多同好，形成了影响力和信任，就可以考虑变现。例如：接相关产品的广告、开设兴趣课程、销售周边产品、组织线下活动等。关键是找到爱好与商业的结合点。

* **Q0213: 帮别人运营账号怎么收费？**

  * **A:** 收费模式多样，常见有：固定服务费、底薪+提成（按涨粉数、GMV等）、按项目收费（如单条视频制作）、咨询费等。具体取决于服务范围、账号基础、预期目标、自身专业度等。

* **Q0281: 开店后，用户浏览了但没下单，怎么促进成交？**

  * **A:** （原回答指出问题前提不对）用户购买主要源于内容种草和需求激发，这应在开店和引流前就考虑。如果用户已经在评论区问“哪里买”，直接回复购买链接或引导至主页即可。对于浏览未下单的用户，可以通过店铺活动、优惠券、私信关怀（需谨慎，避免骚扰）等方式尝试挽回，但重点还是优化前端的内容和选品。

* **Q0283: 卖课程平台的会员（如海豚知道）属于什么变现方式？**

  * **A:** 属于知识付费的一种，也可以看作是分销或CPS，即推广平台会员资格获得佣金。

**商业思考**

* **Q0033: 如何判断一个商业模式是否可行？**

  * **A:** 评估市场需求、盈利能力（收入-成本）、可扩展性、竞争壁垒、与自身资源能力的匹配度、以及合规性。

* **Q0113: 创业初期，应该追求完美还是快速上线？**

  * **A:** 推荐“最小可行性产品”（MVP）思路，先快速上线核心功能的产品或服务，收集市场反馈，然后快速迭代优化。过度追求完美可能错失市场时机。

* **Q0170: 做自由职业/自媒体，如何解决不稳定的问题？**

  * **A:** 拓展多元化收入来源、建立稳定的客户关系或粉丝基础、持续学习提升竞争力、做好财务规划和风险储备、考虑购买社保等保障。

### 四、 账号管理与增长 (Account Management & Growth)

* **Q0001, Q0146: 如何进行有效的对标账号分析？**

  * **A:** 分析其定位、内容选题、内容形式、数据表现、涨粉策略、互动方式、变现模式等。学习优点，结合自身情况进行模仿和创新。定期分析，观察其变化。

* **Q0008: 如何有效提升账号粉丝数量？**

  * **A:** 持续产出高质量、垂直领域的内容；优化内容分发（如蹭热点、参与话题）；与其他账号互动或互推；在其他平台引流；积极与粉丝互动；研究平台推荐机制。

* **Q0018: 账号定位是什么？如何确定？**

  * **A:** 账号定位指明确你的账号是“为谁”（目标用户）提供“什么价值”（内容主题）。确定定位需结合自身兴趣/专长、市场需求和竞争情况。清晰的定位有助于吸引精准粉丝和内容创作。

* **Q0053: 多久更新一次内容比较合适？**

  * **A:** 没有固定标准，取决于领域、内容形式和个人精力。关键是保持相对稳定的更新频率（如日更、周更数次），让粉丝形成预期。质量比数量更重要。

* **Q0056: 账号遇到瓶颈期怎么办？**

  * **A:** 分析数据，找出问题所在（内容、选题、形式、互动等）；尝试新的内容方向或形式；学习新技能；与其他创作者交流；调整心态，坚持创作。

* **Q0099: 需要回复每一条评论吗？**

  * **A:** 尽量积极回复评论，特别是提问和表达认可的评论，有助于提升互动率和粉丝粘性。如果评论量过大，可以优先回复有价值的评论。

* **Q0157: 如何维护粉丝关系？**

  * **A:** 积极回复评论和私信、对粉丝的建议和反馈表示感谢、发起互动话题或活动、在内容中回应粉丝关心的问题、建立粉丝群（私域）等。

* **Q0180: 账号被限流了怎么办？**

  * **A:** 首先自查内容是否违规，进行修改；检查账号近期是否有异常操作；减少发布频率，养号一段时间；尝试发布高质量、正能量的内容；如果确认误判可申诉。

* **Q0202: 多个账号用同一身份信息认证有风险吗？**

  * **A:** 同一平台通常限制一个身份信息认证的账号数量。超量认证或被平台判定为矩阵号有一定风险，尤其是内容低质或有违规行为时。

* **Q0259: 什么是账号权重？如何提升？**

  * **A:** 账号权重是平台对账号质量和信任度的一种综合评估，影响内容推荐。提升方法包括：保持活跃度、发布高质量原创内容、提高互动数据、遵守平台规则、获得官方认证或参与活动等。

  * **A:** 关注行业网站/公众号、使用RSS订阅、利用关键词在各平台搜索、保存看到的优质内容（用稍后阅读或笔记工具）、建立自己的素材库。

    * 请谨慎使用，并确保合规。

### 六、 学习与心态 (Learning & Mindset)

* **Q0038: 如何持续学习和提升自己？**

  * **A:** 阅读书籍和行业报告、关注优质的公众号/博主、参加线上/线下课程或分享、与同行交流、复盘总结自己的实践经验、保持好奇心。

* **Q0054: 感觉做自媒体/IP很难坚持怎么办？**

  * **A:** 明确目标和初心、找到正反馈（即使是小的进步）、加入社群抱团取暖、分解目标降低难度、允许自己有低谷期、将创作融入生活习惯。

* **Q0087: 如何看待竞争对手？**

  * **A:** 将对手视为学习的对象和参照物，分析其优劣势，找到自己的差异化空间。良性竞争可以促进行业发展和自我提升。避免恶性竞争或过度焦虑。

* **Q0133: 如何面对负面评论或质疑？**

  * **A:** 区分建设性批评和恶意攻击。对于前者，虚心听取，反思改进；对于后者，可以选择忽略、拉黑或心态平和地回应（如果必要）。不必过分在意每个人的看法。

* **Q0190: 全职做自媒体/IP需要做哪些准备？**

  * **A:** 充足的资金储备（覆盖几个月甚至一年的生活成本）、明确的定位和规划、一定的技能基础（内容创作、运营等）、强大的自律性和抗压能力、了解相关法律法规和平台规则。
