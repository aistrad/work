# 不被限流的基础逻辑

内容平台，需要你提供的是：有价值的内容，或者钱

**二者必有其一，否则你一定没有流量**



有价值的内容，指的是内容不违规、不擦边、不违法，同时能得到系统推荐。

你自己认为有价值没用，必须是算法认为有价值。

有价值的粗略标准是：点赞率（点赞数/小眼睛数）不低于 5%（及格线是 3%，优秀作品是 10%）

所以如果系统只给了你 100 个小眼睛，不要说自己被限流了，先看一下点赞数。100 个人点击进来，如果只有 1～2 个点赞，那是内容不行，不是限流。

精细的评价指标是：CES 评分，评分越高越好

> CES评分是小红书推荐算法模型中的一种评分方式，用于衡量笔记的受欢迎程度。
>
>
>
> 该评分由点赞数、收藏数、评论数、转发数和关注数等因素综合计算得出，其中点赞数和收藏数各占1分，评论数和转发数各占4分，关注数占8分。CES评分高的笔记将更容易进入小红书的推荐池，获得更多的流量和曝光机会
>
>
>
> CES评分=点赞数×1分+收藏数×1分+评论数×4分+转发数×4分+关注数×8分

### 精细化运营复盘表格举例

|      | 小眼睛<br />（点击量） | 点赞+收藏数<br /> （1 分） | 评论数<br />（4 分） | 转发数<br />（4 分） | 关注数<br />（8 分） | CES 得分 |
| ---- | -------------- | ------------------ | -------------- | -------------- | -------------- | ------ |
| 笔记 A | 100            | 5                  | 1              | 1              | 0              | 13     |
| 笔记 B | 300            | 20                 | 2              | 2              | 0              | 36     |
| 笔记 C | 1000           | 45                 | 3              | 3              | 1              | 77     |

显然，CES 评分越高，小眼睛就越多，小眼睛越多，CES 评分就越高，那这有什么意义呢？变成循环论证了。

算法会被 CES 评分高的内容推流，而得到推流的内容，必然 CES 评分会继续升高。



于是我们要引入一个权重分：权重得分 = CES/小眼睛

举例，笔记 C 的权重分是 77/1000 = 0.077

这意味着，在系统帮你拿到 1000 个点击量的时候，你可以收获 77 分，77 分代表了观众对你的认同程度



这个分数的意义是：当你花钱投广告的时候，你想要多少小眼睛，就有多少小眼睛，因为你砸钱嘛。

这时候你的压力是，要在预算有限（小眼睛有限）的情况下，拿到尽可能多的观众认可。

那么最有参考价值的指标就是：权重得分 = CES/小眼睛



另外，系统在推荐你的爆款内容的时候，也在推荐其他人的爆款内容。

算法如何判断，让哪个爆款内容，变得更爆呢？（从 1k 赞变成 1w 赞），也就是从一级流量池，进入二级、三级流量池。同样是对比这个权重得分。

当然，具体的算法逻辑随时都可能微调，但是整体思路就是这样。



### 不想做自然流量，那就投钱，问题是应该投多少钱？

很多人跟我说，dontbesilent 这个账号的笔记，左下角正在卖这个 ¥99 的课程，你卖课程，平台肯定要抽成，既然平台赚钱，肯定不会给你限流吧？？



抽成不假，但是抽了多少呢？只有 2%



平台经济，是要把买卖双方生吞活剥才肯罢休的。平台希望的是，我卖一个 ¥99 的课程，毛利 ¥97，然后我花其中的 ¥96.9 去打广告，我盈利¥0.1，这是平台对我的预期



所以不要认为你挂个平台店铺就可以躺着赚钱。平台对你的广告费预期，比你的利润还要高。



你必须非常努力，把你的产品做的很好（利润高），或者是把你的广告做的非常吸引人（投流成本低），才有可能实现投流赚钱。



投流赚钱听起来是一个很性感的事情，扔进去一块钱，就给你变出来两块钱。可是，靠简单的事情赚钱，就一定难。



如果你获客环节很简单，可能流量就不太精准，变现就很难。

如果你变现环节很暴利，可能用户就不太愿意买单，获客就很难。

生意人就是负责做好平衡的。



# 养号

从平台角度讲，什么情况下，会导致你无论如何都不可能起号？

1. 你这台手机，登录过被封的账号

2. 你这个手机号，注册过被封的账号

3. 你这个 IP 地址（连接的 WiFi），出现过被封的账号



所以基本操作是：

1. 不要在二手市场买手机（可以从亲戚朋友那里收）

2. 不要从灰色渠道买 sim 卡

3. <span style="color: rgb(216,57,49); background-color: inherit">用小红书 App 的时候，永远不要连接 WiFi</span>



从平台角度讲，什么情况下，不会给你限流？

答案：你的行为看起来像是一个正常人（而不是一个疯狂薅流量的人）



这意味着你注册账号之后需要：

1. 每天至少浏览半小时（不要连 WiFi）

2. 你的手机不要太便宜（不要用 500 元的二手破安卓）

3. 看见好的内容（尤其是你希望对标的同行），点赞收藏关注一下，让系统给你推荐更多

4. 点进去一个帖子，起码浏览个 5～10 秒，不要点进去刷个赞就马上退出来，这样看起来很像机器人

5. 至少刷个三天再发笔记



你说这些操作都照做了之后，是不是平台就一定会给你流量？当然不可能。

养号成功之后，你发个玄学帖子，当场给你违规限流也很正常。

那么养号的目的是什么？

**答：确保，如果我被限流了，我可以确认是我的内容有问题，而不是账号有问题。这叫确定性归因。**

# 如何发帖

1. 不要用浏览器发帖

2. 不要连接 VPN 发帖

3. 不要连接 WiFi 发帖

4. 不要用 RPA 机器人发帖

5. 两次发帖必须间隔一小时以上

6. 不要断更，但是一天最多发帖 2～3 篇，这让你看起来像是一个正常人。如果发帖次数超过这个数字，请确保你的对标账号也是这么做的，你可以模仿。

7. 尽可能符合平台规范，写好标题、正文、tag，图片用 3:4 或者是 1:1 的，图片上面不要出现其他平台水印



有没有可能，我不按这个标准操作发帖，也可以火？非常有可能。

那按照这个规范发帖的目的是什么？

**答：确保，如果我被限流了，我可以确认是我的内容有问题，而不是账号有问题。这叫确定性归因。**

# 什么时候投钱

1. 小红书开店，可能要 1000 或者更大金额的保证金，保证金会退给你，可以随便交。

2. 如果你是纯投流打法，可以第一天就聚光开户投广告，开户充值是 2000 起步，也可以

3. 如果同时需要自然流和聚光投流，务必等你的发帖小眼睛稳定超过 500 再开户。小眼睛稳定超过 500，这个叫起号成功，如果起号失败，不要浪费钱开户

4. 投钱的前提是，你找到一个爆款素材，转化率很高。比如我花 100 元投个薯条，可以拿到 3k 个小眼睛，然后就有人买我 300 元的商品，我的投产 ROI 是 3 倍，那就没问题。所以通常我们需要让自然流量筛选我们的素材，如果你的素材连自然流量都得不到，说明观众根本不喜欢看，你投流也是浪费钱。**自然流量爆过，但是平台不再推流的素材，是好素材，可以花钱加热，投流赚钱。**
