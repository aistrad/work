# 上升星座分析

## 元数据

- **ID**: ascendant_analysis
- **名称**: 上升星座分析
- **级别**: entry
- **计费**: basic
- **描述**: 上升星座（也称为命主星座）是出生时东方地平线上的星座，代表个人的外在形象、第一印象和人生面具。本服务将根据用户的出生时间和地点，计算精确的上升星座位置，并结合十二星座特质提供深度解读，帮助用户了解自己展现给世界的形象以及人生道路的起点。

## 触发条件

**主要触发词**: 上升星座, 我的上升是什么, rising sign, ascendant

**次要触发词**: 命主星座, 东升星座, 上升点, 我的外在形象, 第一印象星座

## 服务流程 (SOP)


### Phase 1: 出生信息收集

**类型**: required

收集用户精确的出生日期、时间和地点，因为上升星座对出生时间非常敏感

**工具**: birth_data_collector
**知识查询**: 上升星座计算方法, 地平线与黄道交点

### Phase 2: 上升星座计算

**类型**: required

根据出生时间计算东方地平线与黄道的交点，确定上升星座及其度数

**工具**: ascendant_calculator, ephemeris
**知识查询**: 黄道面, 地平线交点计算

### Phase 3: 上升星座解读

**类型**: required

解读上升星座的特质、外在表现、第一印象和人生面具

**工具**: interpretation_engine
**知识查询**: 12上升星座特质, 上升星座与外在形象

### Phase 4: 区间分析

**类型**: optional

分析上升星座所在的区间（Decan），提供更精细的解读

**工具**: decan_analyzer
**知识查询**: 星座区间, decan解读
