# 土星周期

## 触发条件

**主要触发词**: 土星回归, 土星周期, 土星返照

**次要触发词**: 29岁, 58岁, 土星行运, 人生阶段

## 前置要求

- 用户出生日期（必须）
- 用户出生时间（推荐）
- 出生地点（推荐）

## 输出配置

**使用工具**:
- show_zodiac_chart
- show_insight

**交付物**: 土星周期分析报告

**预估轮次**: 3-4 轮

## 服务流程 (SOP)

### Phase 1: 信息收集

**类型**: required

收集出生信息，计算当前年龄和土星周期阶段。

### Phase 2: 计算土星位置

**类型**: required

计算关键数据：
- 本命土星位置（星座/宫位）
- 当前土星位置
- 土星周期阶段（第一/第二/第三回归）
- 土星相位（合/刑/冲）

### Phase 3: 快速扫描

**类型**: required

识别土星周期阶段：
- 0-7岁：土星四分相（第一次）
- 7-14岁：土星对分相
- 14-21岁：土星四分相（第二次）
- 21-29岁：接近第一次土星回归
- 29-30岁：第一次土星回归
- 58-60岁：第二次土星回归
- 87-90岁：第三次土星回归

### Phase 4: 主动追问

**类型**: optional

根据土星周期追问：
- "你最近是否感受到责任或限制的压力？"
- "你对自己的人生方向有什么思考？"

### Phase 5: 深度分析

**类型**: required

**使用的思维架构**:
- Robert Hand 土星周期论：土星是时间的老师
- 成熟度框架：土星周期与心理成熟的对应

**分析框架**:
```yaml

  dimensions:
    - name: "本命土星主题"
```

### Phase 7: 答疑讨论

**类型**: reactive

**常见问题处理**:
