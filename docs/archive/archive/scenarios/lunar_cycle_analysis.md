# 月亮周期影响分析

## 元数据

- **ID**: lunar_cycle_analysis
- **名称**: 月亮周期影响分析
- **级别**: standard
- **计费**: basic
- **描述**: 专注于分析月亮周期对用户生活各方面的影响，包括情绪波动、生物节律、能量变化等。帮助用户理解当前月相对其日常生活的具体影响，并提供相应的生活建议和能量调整指导。

## 触发条件

**主要触发词**: 月亮周期, 月相影响, 月亮对我的影响

**次要触发词**: 新月, 满月, 月亮能量, 情绪周期, 生物节律, 月亮与情绪

## 服务流程 (SOP)


### Phase 1: 收集用户信息

**类型**: required

获取用户出生信息以确定本命月亮位置，了解用户想分析的具体时间段

**工具**: birth_data_collector
**知识查询**: 本命月亮, 月亮星座

### Phase 2: 当前月相分析

**类型**: required

分析当前月亮周期阶段及其一般性影响

**工具**: ephemeris_calculator
**知识查询**: 月相周期, 月亮周期含义

### Phase 3: 个人化影响解读

**类型**: required

结合用户本命盘分析月亮周期对其个人的具体影响

**工具**: natal_chart_calculator
**知识查询**: 月亮过境影响, 月亮与本命盘互动

### Phase 4: 生活建议

**类型**: optional

提供基于月亮周期的生活调整建议

**工具**: 无
**知识查询**: 月相生活指南, 能量调整建议
