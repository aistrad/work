# 元素能量分析

## 元数据

- **ID**: element_analysis
- **名称**: 元素能量分析
- **级别**: entry
- **计费**: basic
- **描述**: 基于用户星盘分析四大元素（火、土、风、水）的分布与平衡状态，揭示用户的反应模式、行为特征和内在能量倾向，帮助用户理解自己的原型特质以及如何与不同元素能量和谐共处。

## 触发条件

**主要触发词**: 元素分析, 四元素, 元素分布, 元素能量, 火土风水, 火象星座, 土象星座, 风象星座, 水象星座

**次要触发词**: 元素平衡, 主导元素, 缺少什么元素, 元素特质, 火元素, 土元素, 风元素, 水元素, 模式分析, 基本宫, 固定宫, 变动宫

## 服务流程 (SOP)


### Phase 1: 获取星盘信息

**类型**: required

收集用户出生信息并生成本命星盘

**工具**: collect_zodiac_info, show_zodiac_chart
**知识查询**: 星盘计算, 出生信息

### Phase 2: 元素统计分析

**类型**: required

统计星盘中各行星、发光体所在星座的元素分布，计算火、土、风、水四元素的占比

**工具**: show_zodiac_chart
**知识查询**: 元素分类, 星座元素对应, 行星权重

### Phase 3: 主导元素解读

**类型**: required

分析主导元素对用户反应模式、行为特征的影响

**工具**: search_db, show_insight
**知识查询**: 元素特质, 原型特征, 反应模式

### Phase 4: 元素平衡建议

**类型**: optional

针对元素失衡提供调和建议，说明各元素如何在个人内在和谐运作

**工具**: show_insight
**知识查询**: 元素平衡, 元素和谐, 能量调和
