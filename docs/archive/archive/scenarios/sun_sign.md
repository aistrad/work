## 服务流程 (SOP)

### Phase 1: 信息收集

**类型**: required

确认用户出生日期，确定太阳星座。

### Phase 2: 核心特质分析

**类型**: required

分析太阳星座的核心特质：
- 元素属性（火/土/风/水）
- 模式属性（开创/固定/变动）
- 守护星及其影响
- 阴阳属性

### Phase 3: 人生主题解读

**类型**: required

解读太阳星座代表的人生主题：
- 核心自我表达方式
- 生命力来源
- 自我认同方向
- 创造力表现领域

### Phase 4: 主动追问

**类型**: optional

根据用户兴趣追问：
- 是否想了解与其他星座的互动
- 是否想深入了解月亮或上升星座

### Phase 5: 输出交付

**类型**: required

1. `show_insight` - 太阳星座核心洞察

### Phase 6: 答疑讨论

**类型**: reactive

回答用户关于太阳星座的追问。
