# 相位深度解读

## 元数据

- **ID**: aspect_interpretation
- **名称**: 相位深度解读
- **级别**: standard
- **计费**: basic
- **描述**: 详细分析用户星盘中行星之间的相位关系（合相、六分相、四分相、三分相、对分相等），揭示内在能量的和谐与冲突，帮助理解人生挑战与天赋。

## 触发条件

**主要触发词**: 相位分析, 我的星盘相位, 行星相位, 合相

**次要触发词**: 四分相, 三分相, 对分相, 六分相, 相位解读

## 服务流程 (SOP)


### Phase 1: 生成相位表

**类型**: required

计算星盘中所有主要相位

**工具**: birth_chart_calculator, aspect_calculator
**知识查询**: 相位类型, 相位容许度

### Phase 2: 识别关键相位

**类型**: required

筛选出最重要的相位配置

**工具**: 无
**知识查询**: 主要相位含义

### Phase 3: 相位图形分析

**类型**: optional

识别大三角、T三角等特殊相位图形

**工具**: pattern_detector
**知识查询**: 相位图形, 大三角, T三角

### Phase 4: 综合解读

**类型**: required

整合相位信息提供深度解读

**工具**: 无
**知识查询**: 相位组合解读
