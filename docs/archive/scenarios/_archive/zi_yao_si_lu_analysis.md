# 子遥巳禄格分析

**触发词**: 子遥巳禄, 甲子日甲子时, 遥合格局

**来源**: 三命通会.converted.md:卷六·子遥巳禄:001

## 分析流程

### Phase 1: 确认格局成立条件

- 甲子日甲子时
- 二子中癸水遥合巳中戊土
- 戊与丙同居巳宫，丙合酉中辛为甲之官

### Phase 2: 检查喜忌条件

- 喜生壬癸亥子月(印旺)
- 喜卯寅月(身旺)
- 忌见庚辛丙字明露
- 忌申酉巳字破格

### Phase 3: 检查破格因素

- 大怕有午冲子
- 大怕丑绊子
- 有丑午绊冲则减分数

### Phase 4: 判断行运喜忌

- 行官旺乡必主登科食禄
- 忌南方运
- 若生酉丑月只作正官格
