## 分析流程

### Phase 1: 确定纳音五行

- 根据年柱干支确定纳音五行
- 识别纳音的具体类型（如海中金、炉中火等）

### Phase 2: 分析纳音性质

- 判断纳音五行的盛衰状态
- 分析纳音的形质特征
- 确定纳音的功用特点

### Phase 3: 推断喜忌

- 根据纳音性质确定喜用神
- 识别纳音的忌神
- 分析与其他纳音的生克关系

### Phase 4: 综合论断

- 结合月日时纳音综合分析
- 判断纳音组合的吉凶
- 给出命理建议
