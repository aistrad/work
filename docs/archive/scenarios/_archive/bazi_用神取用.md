# 八字用神取用流程

**触发词**: 用神, 取用, 喜神, 忌神, 调候

**来源**: 滴天髓阐微.converted.md:通神论:002

## 分析流程

### Phase 1: 日主衰旺定位

- 判断日主得令失令
- 分析四柱根气强弱
- 确定日主实际衰旺

### Phase 2: 格局用神分析

- 日主旺印绶多用财星
- 日主旺官杀轻用财星
- 日主旺比劫多用食伤
- 日主弱官杀旺用印绶
- 日主弱食伤多用印绶
- 日主弱财星旺用比劫

### Phase 3: 特殊格局判断

- 从强格：旺极无制宜顺其势
- 从弱格：衰极无助宜从其势
- 两气成象：相生各半取食伤泄秀
- 独象格：一方秀气取化神为用

### Phase 4: 调候用神

- 寒金冷水需火暖
- 火炎土燥需水润
- 木嫩金坚需火攻
- 木旺金衰需土成
