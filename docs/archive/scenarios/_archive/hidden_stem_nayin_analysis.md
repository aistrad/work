# 地支藏干与甲子纳音分析

**触发词**: 藏干, 地支藏干, 纳音, 甲子纳音, 六十甲子, 人元, 海中金, 炉中火

**来源**: 东方代码启示录.converted.md:第28章:地支藏干与纳音

## 分析流程

### Phase 1: 提取地支藏干

- 子藏癸、丑藏癸辛己、寅藏甲丙己、卯藏乙
- 辰藏癸乙戊、巳藏丙庚戊、午藏丁、未藏丁乙己
- 申藏庚壬己、酉藏辛、戌藏丁辛戊、亥藏壬甲戊

### Phase 2: 藏干功能分析

- 功能一：人脉关系分析（藏干代表人元，最敏感于人脉）
- 功能二：各柱十神细节启示（从藏干十神性质和五行性质定位）
- 功能三：个性领域信息启示（如孩子数量估算）

### Phase 3: 确定纳音五行

- 查六十甲子纳音表
- 确定各柱的纳音五行属性
- 确定大运、流年的纳音五行

### Phase 4: 纳音分析应用

- 应用一：月令五行对各柱纳音的利弊（提纲克年/月/日等）
- 应用二：日干月支与各柱纳音五行关系
- 应用三：纳音与纳音之间的五行关系
- 应用四：纳音本身寓意的启示

### Phase 5: 纳音寓意解读

- 金类：海中金（内涵深度）、剑锋金（锐利锋芒）、沙中金（鱼龙混杂）、金箔金（肤浅外表）、钗钏金（脸面易折）、白蜡金（粉饰表面）
- 水类：天河水（高端气势）、涧下水（平稳低调）、大溪水（急冲高调）、泉中水（约束局限）、大海水（气势磅礴）、长流水（源远流长）
- 木类：大林木（势力广泛）、平地木（环境平稳）、桑松木（传播广泛）、杨柳木（摇摆不定）、石榴木（软化变通）、松柏木（坚硬稳固）
- 火类：炉中火（迁就应用）、山下火（迅速蔓延）、佛灯火（和谐平淡）、山头火（范围受限）、天上火（权威气势）、霹雳火（突发急躁）
- 土类：路旁土（搁置休闲）、壁上土（攀附依附）、大驿土（平缓顺畅）、城墙土（局限内部）、沙中土（基础根基）、屋上土（特殊关照）
