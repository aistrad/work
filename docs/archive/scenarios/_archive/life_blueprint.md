# 人生蓝图

## 元数据

- **ID**: life_blueprint
- **名称**: 人生蓝图
- **级别**: professional
- **计费**: premium
- **描述**: 综合分析性格、事业、财运、感情、健康等各方面，绘制完整人生蓝图

## 触发条件

**主要触发词**: 完整分析, 人生规划, 全面分析, 详细看看

**次要触发词**: 深度分析, 综合分析

## 前置要求

- 用户出生信息（必须）
- 已计算命盘数据（必须）
- 大运排盘数据（必须）

## 输出配置

**使用工具**:
- show_bazi_chart
- show_bazi_kline
- show_insight
- show_report

**交付物**: 完整人生蓝图报告

**预估轮次**: 8-15 轮

## 服务流程 (SOP)

### Phase 1: 信息收集

**类型**: required

**描述**: 收集完整出生信息

### Phase 2: 计算排盘

**类型**: required

**描述**: 计算完整命盘数据

**工具调用**:
```
calculate_bazi:
  include_dayun: true
  dayun_count: 10
  include_liunian: true
```

### Phase 3: 全面扫描

**类型**: required

**描述**: 全面识别命盘特征

**扫描维度**:
1. 日主强弱和格局
2. 十神配置和关系
3. 五行平衡状态
4. 大运整体走势
5. 红旗信号检查

### Phase 4: 主动追问

**类型**: required

**描述**: 深入了解用户情况

**追问内容**:
- 当前年龄和职业
- 感情状态
- 主要关注的方面
- 近期重大决定

### Phase 5: 深度分析

**类型**: required

**分析模块**:

1. **性格天赋**
   - 核心性格特点
   - 天赋优势
   - 需要注意的弱点

2. **事业发展**
   - 适合的行业方向
   - 事业发展周期
   - 贵人和机遇

3. **财运分析**
   - 财运格局
   - 求财方式
   - 理财建议

4. **感情婚姻**
   - 感情模式
   - 理想对象
   - 婚姻建议

5. **健康提示**
   - 五行偏枯对应的健康风险
   - 养生建议

6. **人生阶段**
   - 各大运阶段特点
   - 关键转折点
   - 发展建议

### Phase 6: 输出交付

**类型**: required

**工具调用**:
1. `show_bazi_chart` - 展示命盘
2. `show_bazi_kline` - 展示运势K线
3. `show_report(type="life_blueprint")` - 完整报告

**输出结构**:
```
【命盘总览】
{调用 show_bazi_chart}

【运势走势】
{调用 show_bazi_kline}

【人生蓝图报告】
{调用 show_report}

## 核心总结

你是{日主}日主，{格局}格局，{一句话核心定位}。

## 人生建议

1. **发展方向**: {建议}
2. **关键时机**: {时间点}
3. **注意事项**: {提醒}
```

### Phase 7: 答疑讨论

**类型**: reactive

**描述**: 针对报告内容深入讨论
