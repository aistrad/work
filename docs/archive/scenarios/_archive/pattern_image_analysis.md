# 格局形象判断

## 元数据

- **ID**: pattern_image_analysis
- **名称**: 格局形象判断
- **级别**: professional
- **计费**: premium
- **描述**: 基于滴天髓形象理论，判断八字的格局形象，包括两气成象、五气成形、独象等特殊格局

## 触发条件

**主要触发词**: 格局是什么, 我是什么格局, 八字格局, 命格分析

**次要触发词**: 两气成象, 曲直格, 炎上格, 稼穑格, 从革格, 润下格

## 服务流程 (SOP)

### Phase 1: 气势判断

**目标**: 判断八字整体气势

**执行动作**:
- 分析五行分布
- 判断是否有偏重
- 识别主导气势

### Phase 2: 形象识别

**目标**: 识别具体格局形象

**执行动作**:
- 判断是否两气成象
- 判断是否五气成形
- 判断是否独象格局

### Phase 3: 格局定性

**目标**: 确定格局类型与喜忌

**执行动作**:
- 确定格局名称
- 分析化神是否昌旺
- 判断格局喜忌运程
