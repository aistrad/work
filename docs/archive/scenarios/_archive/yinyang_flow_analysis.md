# 阴阳顺逆分析

## 元数据

- **ID**: yinyang_flow_analysis
- **名称**: 阴阳顺逆分析
- **级别**: professional
- **计费**: premium
- **描述**: 基于滴天髓干支总论，分析八字阴阳配合与气势顺逆，判断阳乘阳位、阴乘阴位的运程配合

## 触发条件

**主要触发词**: 阴阳分析, 顺逆分析, 气势分析, 运程配合

**次要触发词**: 阳乘阳位, 阴乘阴位, 寒暖配合, 燥湿调候

## 服务流程 (SOP)

### Phase 1: 阴阳判断

**目标**: 判断八字阴阳属性

**执行动作**:
- 分析天干阴阳
- 分析地支阴阳
- 判断整体阴阳偏向

### Phase 2: 寒暖分析

**目标**: 分析八字寒暖燥湿

**执行动作**:
- 判断东南阳暖还是西北阴寒
- 分析调候需求
- 确定寒暖配合方向

### Phase 3: 运程建议

**目标**: 给出运程配合建议

**执行动作**:
- 阳盛宜配阴顺之地
- 阴盛宜配阳刚之地
- 指出最佳运程方向
