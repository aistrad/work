## 分析流程

### Phase 1: 天干地支单象分析

- 天干单象：甲（硬木、梁柱、稳健）、乙（软木、花草、柔美）、丙（阳光、热烈）、丁（灯光、闪烁）、戊（硬土、大城市）、己（软土、小村镇）、庚（硬金属、强制）、辛（软金属、端庄）、壬（大海、急躁）、癸（湖泊、细致）
- 地支单象：子（北方、富贵）、丑（东北、磨难）、寅（东方、权贵）、卯（东方、暴露）、辰（东南、反复）、巳（南方、文采）、午（南方、福报）、未（西南、奔波）、申（西方、独创）、酉（西方、成熟）、戌（西北、技艺）、亥（北方、晚成）

### Phase 2: 天干合冲象分析

- 甲己合（中正之合）、乙庚合（仁义之合）、丙辛合（威制之合）、丁壬合（淫移之合）、戊癸合（无情之合）
- 甲庚冲、乙辛冲、丙壬冲、丁癸冲的飘忽性

### Phase 3: 地支合冲象分析

- 六合象：子丑合（磨炼发迹）、亥寅合（权贵威信）、卯戌合（才能显露）、辰酉合（反复成才）、巳申合（独创文采）、午未合（奔劳得报）
- 三合象：巳酉丑（文采名誉）、亥卯未（厚积薄发）、申子辰（白手创业）、寅午戌（才艺福报）
- 六冲象：子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲

### Phase 4: 重叠干支读象

- 天干五行重叠：指向所生所克的天干
- 天干十神重叠：内部指向生克冲合元素，外部影响他人八字
- 地支重叠：指向冲合元素，影响他人八字

### Phase 5: 天干十神带支读象

- 所带地支星煞的寓意
- 所带地支五行的寓意（方位、行业、长短）
- 所带地支十神的寓意
- 所带地支本性的寓意
- 干支连带角色互换

### Phase 6: 其他读象概念

- 天干十神与各柱时空人脉关系读象
- 干支柱内生克读象（支生干/支克干）
- 干支冲合对比读象
- 时空柱本位读象
- 六十甲子整体成像（意象、形象、物象）
