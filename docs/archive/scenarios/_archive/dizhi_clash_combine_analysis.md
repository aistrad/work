# 地支冲合分析

## 元数据

- **ID**: dizhi_clash_combine_analysis
- **名称**: 地支冲合分析
- **级别**: standard
- **计费**: basic
- **描述**: 基于滴天髓地支论，分析八字地支的冲合关系，判断旺衰冲拔与暗冲暗会

## 触发条件

**主要触发词**: 地支冲合, 冲克分析, 六冲, 六合, 三合, 三会

**次要触发词**: 子午冲, 卯酉冲, 寅申冲, 巳亥冲, 暗冲暗会

## 服务流程 (SOP)

### Phase 1: 冲合识别

**目标**: 识别地支冲合关系

**执行动作**:
- 检查六冲关系
- 检查六合关系
- 检查三合三会

### Phase 2: 旺衰判断

**目标**: 判断冲合双方旺衰

**执行动作**:
- 判断得令失令
- 分析旺者冲衰衰者拔
- 分析衰神冲旺旺神发

### Phase 3: 吉凶论断

**目标**: 根据冲合判断吉凶

**执行动作**:
- 生方怕动库宜开
- 败地逢冲仔细推
- 暗冲暗会尤为喜
