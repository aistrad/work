## 触发条件

**主要触发词**: 全面分析, 综合批命, 详细解读, 深度分析

**次要触发词**: 看命口诀, 系统分析, 专业批命, 完整解读

## 服务流程 (SOP)

### Phase 1: 月令提纲分析

**目标**: 先看月支有无财官，月令为命之本

**执行动作**:
- 月取支神为命
- 日取天干为主
- 月有正官偏官只以月中取
- 月令全无可用方看他格

### Phase 2: 格局用神确定

**目标**: 确定命局格局和用神

**执行动作**:
- 正官一位乃君子贵人
- 七煞一位聪明伶俐
- 财一位务要得时
- 印不论一位二位都好

### Phase 3: 五行象法分析

**目标**: 分析五行成象和配合

**执行动作**:
- 金水象不可见土
- 金土象不可见木
- 金火象不可见水
- 水火象成既济最妙

### Phase 4: 大运流年推断

**目标**: 分析大运流年吉凶

**执行动作**:
- 行好运日干伤流年祸轻
- 行不好运日干伤岁君祸重
- 逢官看财逢煞看印
- 运有生有去为福

### Phase 5: 生死兴亡论断

**目标**: 判断命运起伏和关键节点

**执行动作**:
- 始终富贵者身主专旺用神得令
- 一旦崛起者日主无力运来生扶
- 中间剥落者好运过后遇凶运
- 晚年成就者身强财官虚浮晚运补起
