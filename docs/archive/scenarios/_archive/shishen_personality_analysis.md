# 十神性格深度分析

## 元数据

- **ID**: shishen_personality_analysis
- **名称**: 十神性格深度分析
- **级别**: standard
- **计费**: basic
- **描述**: 基于十神的三层含义（人事关系、心理动力、行为模式），深度分析命主的性格特征、心理倾向和行为模式

## 触发条件

**主要触发词**: 性格分析, 我的性格, 十神性格, 心理特征, 行为模式

**次要触发词**: 比肩性格, 劫财性格, 食神性格, 伤官性格, 正财性格, 偏财性格, 正官性格, 七杀性格, 正印性格, 偏印性格

## 服务流程 (SOP)

### Phase 1: 十神配置识别

**目标**: 识别命盘中主要十神分布

**执行动作**:
- 分析四柱十神透干情况
- 判断十神旺衰
- 识别主导十神

### Phase 2: 心理动力分析

**目标**: 解读十神对应的心理动力

**执行动作**:
- 分析主导十神的心理特征
- 判断内在驱动力
- 识别潜在心理倾向

### Phase 3: 行为模式解读

**目标**: 预测命主的行为模式

**执行动作**:
- 根据十神组合判断行为倾向
- 分析过旺过弱的表现
- 给出性格优化建议
