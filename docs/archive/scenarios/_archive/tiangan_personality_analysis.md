# 天干性格分析

## 元数据

- **ID**: tiangan_personality_analysis
- **名称**: 天干性格分析
- **级别**: standard
- **计费**: basic
- **描述**: 基于《滴天髓》十天干论，分析日主天干的性格特征、优势劣势及适用边界

## 触发条件

**主要触发词**: 性格分析, 我是什么性格, 天干性格, 日主性格

**次要触发词**: 甲木性格, 乙木性格, 丙火性格, 丁火性格, 戊土性格, 己土性格, 庚金性格, 辛金性格, 壬水性格, 癸水性格

## 服务流程 (SOP)

### Phase 1: 识别日主天干

**目标**: 确定日主的天干五行

**执行动作**:
- 提取八字日柱天干
- 判断阴阳属性

### Phase 2: 性格特征分析

**目标**: 根据天干论断语分析性格

**执行动作**:
- 甲木参天主刚直
- 乙木柔韧主灵活
- 丙火猛烈主热情
- 丁火柔中主内敛
- 戊土固重主稳重
- 己土卑湿主包容
- 庚金带煞主果敢
- 辛金软弱主精致
- 壬水通河主智慧
- 癸水至弱主细腻

### Phase 3: 适用边界说明

**目标**: 说明性格分析的限定条件

**执行动作**:
- 考虑日主强弱
- 考虑五行配置
- 给出实战修正建议
