# 富贵贫贱格局判断

## 元数据

- **ID**: fugui_pianjian_analysis
- **名称**: 富贵贫贱格局判断
- **级别**: professional
- **计费**: premium
- **描述**: 基于《滴天髓》格局论，判断命局的富贵贫贱层次

## 触发条件

**主要触发词**: 我的命好不好, 富贵命, 贫贱命, 格局高低

**次要触发词**: 财气通门户, 官星有理会, 财神不真, 官星不见

## 服务流程 (SOP)

### Phase 1: 财运判断

**目标**: 判断财气是否通达

**执行动作**:
- 检查财星有根有生
- 判断身旺能否任财
- 分析食伤生财流通

### Phase 2: 贵气判断

**目标**: 判断官星是否有情

**执行动作**:
- 检查官星透而有根
- 判断有印护官或财生官
- 分析官印相生配置

### Phase 3: 综合定级

**目标**: 综合判断格局层次

**执行动作**:
- 富贵兼备为上
- 有富无贵或有贵无富为中
- 贫贱皆占为下
