# 十天干特性分析

## 元数据

- **ID**: tiangan_character_analysis
- **名称**: 十天干特性分析
- **级别**: standard
- **计费**: basic
- **描述**: 基于滴天髓天干论，分析日主天干的特性、喜忌与适宜配合

## 触发条件

**主要触发词**: 日主特性, 天干分析, 甲木特性, 乙木特性, 丙火特性, 丁火特性

**次要触发词**: 戊土特性, 己土特性, 庚金特性, 辛金特性, 壬水特性, 癸水特性

## 服务流程 (SOP)

### Phase 1: 日主识别

**目标**: 确定日主天干

**执行动作**:
- 提取日柱天干
- 判断阴阳属性
- 确定五行属性

### Phase 2: 特性解读

**目标**: 解读日主天干特性

**执行动作**:
- 阐述天干本性
- 说明天干喜忌
- 分析季节影响

### Phase 3: 配合建议

**目标**: 给出配合建议

**执行动作**:
- 说明适宜配合
- 指出忌讳配合
- 给出调候建议
