# 魁罡格局分析

**触发词**: 魁罡, 壬辰, 庚戌, 戊戌, 庚辰

**来源**: 渊海子平.converted.md:魁罡:001

## 分析流程

### Phase 1: 确认魁罡日

- 魁罡有四：壬辰、庚戌、戊戌、庚辰日
- 日位加临者众，必是福

### Phase 2: 分析格局特点

- 主人性格聪明、文章振发、临事有断
- 惟是好杀
- 运行身旺，发福百端

### Phase 3: 判断忌神

- 一见财官，祸患立至
- 四柱有财及官，或带刑杀，祸不可测
- 日独处冲者太众，必是小人

### Phase 4: 行运吉凶

- 运临财官旺处，亦防奇祸
- 身旺逢之贵异常
- 刑冲兼破害，一贫彻骨受笞鞭
