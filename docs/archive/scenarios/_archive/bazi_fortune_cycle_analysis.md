# 大运流年分析

## 元数据

- **ID**: bazi_fortune_cycle_analysis
- **名称**: 大运流年分析
- **级别**: professional
- **计费**: premium
- **描述**: 分析大运与流年对命局的影响，预测运势起伏

## 触发条件

**主要触发词**: 今年运势, 大运分析, 流年运势, 什么时候转运

**次要触发词**: 运气好不好, 几岁走好运, 明年怎么样, 运势预测

## 服务流程 (SOP)

### Phase 1: 排大运

**目标**: 根据出生信息排出大运

**执行动作**:
- 确定阳年阴年
- 确定顺行逆行
- 计算起运岁数
- 排出各步大运

### Phase 2: 判断善恶运

**目标**: 根据用神判断大运吉凶

**执行动作**:
- 利于用神者为善运
- 不利于用神者为恶运
- 被克合者平庸

### Phase 3: 分析流年

**目标**: 结合流年判断具体年份运势

**执行动作**:
- 流年与大运配合
- 流年与命局作用
- 判断具体吉凶

### Phase 4: 干支分论

**目标**: 细分天干地支的影响

**执行动作**:
- 前五年重天干
- 后五年重地支
- 干占七分支占三分或反之
