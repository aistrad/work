# 神趣八法格局分析

**触发词**: 类象, 属象, 从象, 化象, 照象, 返象, 鬼象, 伏象, 八法

**来源**: 渊海子平.converted.md:神趣八法:001

## 分析流程

### Phase 1: 判断格局类型

- 类象：天地一类，如春生人甲乙天干地支寅卯辰全
- 属象：天干甲乙木，地支寅卯未全
- 从象：日主无根，四柱纯一行
- 化象：甲己化土、乙庚化金等
- 照象：木火相照、金水相照
- 返象：月令用神引至时上绝地
- 鬼象：秋月甲乙日，地支四位纯金
- 伏象：寅午戌三合全，壬日午中丁火合壬

### Phase 2: 分析格局成败

- 类象最怕引至时为死绝之乡
- 从象只有秀气者吉，无秀气者不吉
- 化象怕逢返本，化不成反为不吉
- 照象年干有照者亦吉

### Phase 3: 行运吉凶

- 类象运至死绝则不吉
- 从象大旺运吉，死绝地凶
- 化象喜行化神旺运
- 伏象运至木火之乡皆吉，愁水旺之乡
