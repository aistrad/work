# 官印相生格局分析

## 元数据

- **ID**: guanyin_xiangsheng_analysis
- **名称**: 官印相生格局分析
- **级别**: standard
- **计费**: basic
- **描述**: 分析命盘中官印相生的格局，判断仕途发展、学历名气、权威涵养等方面的运势

## 触发条件

**主要触发词**: 官印相生, 杀印相生, 仕途分析, 官运分析

**次要触发词**: 学者型官员, 文职领导, 权贵有涵养, 化杀为权

## 服务流程 (SOP)

### Phase 1: 格局识别

**目标**: 判断是否成官印相生格

**执行动作**:
- 检查正官与正印是否同现
- 或七杀与偏印是否同现
- 判断相生是否有情

### Phase 2: 格局层次

**目标**: 评估格局高低

**执行动作**:
- 分析官印力量对比
- 判断是否有破格之神
- 评估格局纯粹度

### Phase 3: 运势预测

**目标**: 预测仕途与名气

**执行动作**:
- 分析大运流年配合
- 预测仕途发展时机
- 给出发展建议
