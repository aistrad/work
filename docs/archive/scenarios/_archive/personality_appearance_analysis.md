# 性格相貌分析

## 元数据

- **ID**: personality_appearance_analysis
- **名称**: 性格相貌分析
- **级别**: standard
- **计费**: basic
- **描述**: 基于五行旺衰分析命主的性格特征、外貌特点和行为倾向

## 触发条件

**主要触发词**: 性格分析, 我的性格, 性格特点, 相貌分析

**次要触发词**: 五行性格, 命主性情, 外貌特征, 行为特点

## 服务流程 (SOP)

### Phase 1: 五行属性判定

**目标**: 确定日主五行及其旺衰状态

**执行动作**:
- 识别日干五行
- 判断月令旺衰
- 分析四柱五行分布

### Phase 2: 性格特征推断

**目标**: 根据五行旺衰推断性格

**执行动作**:
- 木旺主仁慈恻隐
- 火旺主礼让端谨
- 土旺主忠孝诚信
- 金旺主义勇刚毅
- 水旺主智谋聪明

### Phase 3: 相貌特征分析

**目标**: 推断外貌体态特征

**执行动作**:
- 生旺主长大
- 死绝主矮小
- 结合神煞判断

### Phase 4: 综合建议

**目标**: 给出性格优化建议

**执行动作**:
- 指出性格优势
- 提示需注意之处
- 给出调整建议
