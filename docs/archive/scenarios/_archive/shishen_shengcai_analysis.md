# 食神生财格局分析

## 元数据

- **ID**: shishen_shengcai_analysis
- **名称**: 食神生财格局分析
- **级别**: standard
- **计费**: basic
- **描述**: 分析命盘中食神生财的吉格，判断以才华生财、从事餐饮教育演艺等行业、财源流通等特质

## 触发条件

**主要触发词**: 食神生财, 以才华生财, 餐饮行业, 教育行业

**次要触发词**: 演艺行业, 财源流通, 才华变现

## 服务流程 (SOP)

### Phase 1: 格局识别

**目标**: 判断是否成食神生财格

**执行动作**:
- 检查食神与财星是否相生有情
- 判断食神力量
- 评估财星旺度

### Phase 2: 行业匹配

**目标**: 匹配适合的行业

**执行动作**:
- 餐饮美食行业
- 教育培训行业
- 演艺娱乐行业
- 写作创作行业

### Phase 3: 财运预测

**目标**: 预测财运发展

**执行动作**:
- 大运流年配合
- 财运发展时机
- 给出发展建议
