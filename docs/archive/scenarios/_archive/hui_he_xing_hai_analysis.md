## 分析流程

### Phase 1: 识别会合局

- 检查三会局：亥子丑会水、寅卯辰会木、巳午未会火、申酉戌会金
- 检查三合局：巳酉丑合金、寅午戌合火、申子辰合水、亥卯未合木
- 确定成局模式：原局内部/原局与时运/纯时运

### Phase 2: 分析会合利弊

- 三会局：与会局本性相同的元素获利，土性元素受损
- 三合局：能量聚集的元素获利，其他两个元素牺牲
- 分析各元素对应的十神含义
- 解读对命主各方面的影响

### Phase 3: 识别刑害局

- 检查刑局：子卯刑、巳申寅刑、午亥酉刑、丑未辰/丑未戌/辰戌丑/辰戌未刑
- 检查自刑：午午、亥亥、辰辰、酉酉
- 检查害局：子未、丑午、寅巳、卯辰、申亥、酉戌

### Phase 4: 分析刑害影响

- 刑局含义：约束、限制、刑罚、受伤、病痛、隔膜
- 子卯刑（无礼之刑）：官非人祸
- 巳申寅刑（恃势之刑）：倚仗权势造成
- 带土相刑（无恩之刑）：残忍刻薄、伤害疾病
- 自刑：自身失误造成
- 害局含义：拖累、负累、牵连、纠缠、不和谐

### Phase 5: 给出建议

- 刑局化解：多做善事、施助他人、放生
- 害局应对：理解相互依附关系，调整期望
