## 分析流程

### Phase 1: 确定日干标尺

- 获取命主日干
- 确定以日干为标尺的星煞（贵人、禄神、羊刃、文昌）
- 确定以年支/日支为标尺的星煞（驿马、桃花、劫煞、灾煞、将星、华盖）
- 确定以月支为标尺的星煞（天医、天喜）
- 确定以年支为标尺的星煞（孤辰寡宿）
- 确定以日柱为标尺的星煞（阴差阳错、魁罡、天罗地网、四废）

### Phase 2: 定位星煞位置

- 检查八字原局四柱中的星煞分布
- 记录星煞所在柱位（年/月/日/时）
- 分析星煞与十神的关联

### Phase 3: 分析星煞影响

- 根据柱位解读时段/地域/人脉含义
- 分析星煞被冲/被合的情况
- 结合时运分析星煞的动态变化

### Phase 4: 综合解读

- 吉星（贵人、禄神、将星、文昌、学堂、天喜）的利好分析
- 凶星（劫煞、灾煞、四废）的提醒分析
- 双刃剑星煞（羊刃、桃花、华盖）的两面性分析
- 给出针对性建议
