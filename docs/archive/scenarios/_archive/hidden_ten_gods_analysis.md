## 场景描述
分析八字中表面缺失但实际��在隐性指向的十神，运用翘翘板原理、隐性指向、替代元素等专业方法。

## SOP 流程

### Phase 1: 表面缺失识别 (required)
1. 列出八字中明显出现的十神
2. 识别表面缺失的十神
3. 记录缺失十神的五行属性

### Phase 2: 翘翘板原理分析 (required)
**核心原理**: 十神之间存在对立统一关系，一方过旺则另一方受抑制

分析步骤:
1. 找出八字中最旺的十神
2. 确定其对立面十神
3. 判断对立面是否形成隐性指向

**常见翘翘板关系**:
- 印星 ↔ 财星
- 官杀 ↔ 食伤
- 比劫 ↔ 财星

### Phase 3: 隐性指向判断 (required)
判断标准:
1. 某十神过旺 → 其克制对象形成隐性存在
2. 某十神被克 → 克它的十��形成隐性需求
3. ���十神缺失 → 通过生克链条寻找替代

**隐性指向类型**:
- 直接隐性: 被克制而不显
- 间接隐性: 通过生克链条体现
- 替代隐性: 由同类十神替代

### Phase 4: 替代元素分析 (required)
1. 正偏替代: 正印可替代枭神功能
2. 同类替代: 比肩可替代劫财功能
3. 生克替代: 印星可替代官杀的部分功能

### Phase 5: 综合解读 (required)
1. 整合显性与隐性十神
2. 分析隐性十神对命主的实际影响
3. 给出针对隐性十神的调整建议

### Phase 6: 案例参考 (optional)
- 周恩来总理: 伤官配印隐性指向案例
