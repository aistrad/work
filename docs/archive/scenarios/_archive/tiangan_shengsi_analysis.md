# 天干阴阳生死分析

**触发词**: 天干, 长生, 沐浴, 冠带, 临官, 帝旺, 衰, 病, 死, 墓, 绝, 胎, 养, 阳生阴死, 阴生阳死

**来源**: 三命通会.converted.md:卷二·论天干阴阳生死:002

## 分析流程

### Phase 1: 确定日主天干

- 识别日柱天干
- 判断天干阴阳属性（甲丙戊庚壬为阳，乙丁己辛癸为阴）

### Phase 2: 定位十二宫

- 阳干顺行定十二宫位置
- 阴干逆行定十二宫位置
- 确定日主在各支的状态

### Phase 3: 分析生旺死绝

- 判断日主在月令的状态
- 分析日主在年支、日支、时支的状态
- 综合评估日主强弱

### Phase 4: 论断吉凶

- 生旺太过需制伏
- 死绝不及需生扶
- 给出调候建议
