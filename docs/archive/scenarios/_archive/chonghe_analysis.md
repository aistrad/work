# 干支冲合寓意分析流程

**触发词**: 冲合分析, 相冲, 相合, 婚姻分析, 事业稳定性, 缘分深浅

**来源**: 东方代码启示录.converted.md:第十七章:003

## 分析流程

### Phase 1: 识别冲合关系

- 天干相合：甲己、乙庚、丙辛、丁壬、戊癸
- 天干相冲：甲庚、乙辛、丙壬、丁癸
- 地支相合：子丑、亥寅、卯戌、辰酉、巳申、午未
- 地支相冲：子午、丑未、寅申、卯酉、辰戌、巳亥

### Phase 2: 判断冲合类型

- 单冲/单合：两元素直接作用
- 冲中带合：同时存在冲合元素时处于静态
- 冲者被合：合元素存在时不冲，缺失时体现冲
- 合者受冲：冲元素存在时不合，缺失时体现合
- 二合一/三合一：多合负累或波折
- 二冲一/三冲一：冲击加剧或祸端

### Phase 3: 分析冲合寓意

- 十神强弱：逢合强势，逢冲弱势
- 稳定程度：逢冲欠稳反复，逢合稳定
- 持久程度：逢冲不持久，逢合持久
- 时效程度：逢冲较迟，逢合较早
- 远近动静：冲为动/远，合为静/近
- 难易程度：逢冲波折难成，逢合顺畅容易
- 缘分深浅：逢合缘深，逢冲缘浅
- 利弊信息：逢合顺利，逢冲不顺

### Phase 4: 地支冲合对天干影响

- 承载型天干（干生支、干克支、干支同）：地支受冲则天干摇摆
- 生克型天干（支生干、支克干）：地支受冲则生克作用消失
- 地支被合：承载型天干显合特征，生克型天干作用力加强

### Phase 5: 应用指导

- 日支逢冲：配偶宜来自远方或晚婚
- 时柱逢冲：与孩子缘分浅，不宜过于缠绵
- 正官逢冲：事业多变不持久，宜远方发展
- 顺应冲合本性则顺利，违反则招致不顺
