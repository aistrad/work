# 五行时地分野吉凶分析

**触发词**: 地域分析, 出生地吉凶, 时令五行, 地域五行, 九州分野

**来源**: 三命通会.converted.md:卷四·论五行时地分野吉凶:001

## 分析流程

### Phase 1: 确定五行与地域对应

- 木(甲乙寅卯):兖州、青州为得地
- 火(丙丁巳午):徐州、扬州为得地
- 土(戊己辰戌丑未):豫州为得地
- 金(庚辛申酉):荆州、梁州为得地
- 水(壬癸亥子):冀州、雍州为得地

### Phase 2: 确定五行与时令对应

- 木:春令为得时
- 火:夏令为得时
- 土:四季月为得时
- 金:秋冬为得时
- 水:冬令为得时

### Phase 3: 分析昼夜阴晴影响

- 春木:晴则花叶敷荣,雨则寒其萌蘖
- 夏木:阴雨则吉,亢晴则忌
- 考虑昼生夜生的差异
- 结合地域判断寒暖

### Phase 4: 综合判断吉凶

- 得地得时者吉
- 失地失时者凶
- 通变权衡,不可执一
