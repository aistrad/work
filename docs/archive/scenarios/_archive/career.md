# 事业分析

## 元数据

- **ID**: career
- **名称**: 事业分析
- **级别**: standard
- **计费**: basic
- **描述**: 从八字角度分析事业发展潜力、适合的行业方向和职业建议

## 触发条件

**主要触发词**: 事业, 工作, 职业, 跳槽, 升职, 创业

**次要触发词**: 适合做什么, 工作方向, 职业规划

## 前置要求

- 用户出生信息（必须）
- 已计算命盘数据（必须）

## 输出配置

**使用工具**:
- show_bazi_chart
- show_insight

**交付物**: 事业分析报告 + 职业建议

**预估轮次**: 3-6 轮

## 服务流程 (SOP)

### Phase 1: 信息收集

**类型**: required

**描述**: 确认用户出生信息

### Phase 2: 计算排盘

**类型**: required

**描述**: 计算命盘数据

### Phase 3: 快速扫描

**类型**: required

**描述**: 快速识别事业相关的命盘特征

**扫描重点**:
1. 官杀星的状态（事业星）
2. 印星的状态（学历、贵人）
3. 食伤星的状态（才华、创意）
4. 财星的状态（财运配合）

**事业格局速判**:
| 格局 | 特征 | 适合方向 |
|-----|------|---------|
| 官杀旺 | 正官/七杀有力 | 体制内、管理层 |
| 食伤旺 | 食神/伤官有力 | 创意、技术、自由职业 |
| 财星旺 | 正财/偏财有力 | 商业、金融 |
| 印星旺 | 正印/偏印有力 | 学术、教育、文化 |

### Phase 4: 主动追问

**类型**: optional

**描述**: 了解用户当前职业状况

**追问示例**:
- "你目前从事什么行业？"
- "有跳槽或创业的想法吗？"
- "工作中最大的困扰是什么？"

**类型**: required

**分析框架**:

#### 华山八字事业分析框架（源自《东方代码启示录》）

**1. 十神格局事业定性**

| 格局 | 事业特征 | 适合方向 |
|-----|---------|---------|
| 正财生官 | 平步青云、循序渐进 | 稳定体制、大企业 |
| 杀印相生 | 功名显达、威名远扬 | 领导岗位、行业权威 |
| 伤官配印 | 名利双收、专业突出 | 技术专家、学术研究 |
| 食神制杀 | 化险为夷、以柔克刚 | 管理协调、危机处理 |
| 官制劫 | 出类拔萃、非富则贵 | 竞争性行业、脱颖而出 |
| 偏财生杀 | 投机博弈、冒险经营 | 创业、投资、风险行业 |
| 印生比 | 社交广泛、影响深远 | 公关、媒体、社交型工作 |
| 比生伤 | 时势被动、公众演艺 | 娱乐、时事、社交活跃 |

**2. 十神事业寓意速查**

| 十神 | 事业寓意 |
|-----|---------|
| 正官 | 稳定事业、功名、地位、正规职业、文职 |
| 七杀 | 权威、权力、垄断、偏门职业、武职、政职 |
| 正财 | 稳定收益、打工、以技谋生、正路经营 |
| 偏财 | 投机、资本钻营、风险投资、横财 |
| 正印 | 威望、名誉、资格、学术、靠山扶持 |
| 枭神 | 奔波、劳碌、偏能怪技、后补资历 |
| 食神 | 积累、享乐、安逸、业绩停顿、安于现状 |
| 伤官 | 权力、学术专研、技能、才艺、不服管制 |
| 比肩 | 社交、帮扶、知己贵人、好攀权贵 |
| 劫财 | 竞争、障碍、业绩压力、自力更生 |

**3. 三大衰格事业警示**

| 衰格 | 事业风险 | 化解建议 |
|-----|---------|---------|
| 伤官见官 | 事业波折、官非口舌 | 避免与上级冲突，低调行事 |
| 枭神夺食 | 多劳少获、是非周折 | 奋发自强、刻苦耐劳 |
| 偏财坏印 | 名声受损、重财轻命 | 以正道求财，保护名誉 |

#### 传统分析补充

1. **事业潜力评估**
   - 官杀星的强弱和位置
   - 是否有贵人（印星）相助
   - 格局高低判断

2. **适合行业分析**
   - 根据用神五行推荐行业
   - 根据十神配置推荐职业类型

3. **发展时机分析**
   - 当前大运对事业的影响
   - 近几年流年的事业运

4. **具体建议**
   - 发展方向建议
   - 需要注意的问题
   - 贵人方位

**行业五行对应**:
| 五行 | 适合行业 |
|-----|---------|
| 木 | 教育、文化、医疗、环保 |
| 火 | 科技、传媒、餐饮、能源 |
| 土 | 房地产、建筑、农业、矿业 |
| 金 | 金融、法律、机械、IT |
| 水 | 物流、旅游、贸易、咨询 |
