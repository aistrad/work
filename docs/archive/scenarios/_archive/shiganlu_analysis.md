# 十干禄分析

**触发词**: 禄, 建禄, 禄神, 禄位, 甲禄寅, 乙禄卯, 丙禄巳, 丁禄午, 戊禄巳, 己禄午, 庚禄申, 辛禄酉, 壬禄亥, 癸禄子

**来源**: 三命通会.converted.md:卷三·论十干禄:008

## 分析流程

### Phase 1: 确定禄位

- 甲禄在寅，乙禄在卯
- 丙戊禄在巳，丁己禄在午
- 庚禄在申，辛禄在酉
- 壬禄在亥，癸禄在子

### Phase 2: 判断禄的类型

- 福星禄：干克支为财
- 伏马禄：火土相生
- 破禄：禄位受克
- 正禄：禄位纯正
- 长生禄：禄在长生位

### Phase 3: 分析禄的组合

- 合禄：禄位与他支相合
- 拱禄：两支夹拱禄位
- 暗禄：不见禄而见合
- 库禄：禄之聚处

### Phase 4: 论断禄的吉凶

- 禄多马少使主神劳
- 禄少马多能操善负
- 禄在生旺主安逸足财
- 禄在死绝主气浊神慢
