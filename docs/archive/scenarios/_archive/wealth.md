# 财运分析

## 元数据

- **ID**: wealth
- **名称**: 财运分析
- **级别**: standard
- **计费**: basic
- **描述**: 从八字角度分析财运走势、理财方式和投资方向

## 触发条件

**主要触发词**: 财运, 投资, 赚钱, 发财, 理财

**次要触发词**: 破财, 存钱, 收入, 财务

## 前置要求

- 用户出生信息（必须）
- 已计算命盘数据（必须）

## 输出配置

**使用工具**:
- show_bazi_chart
- show_insight

**交付物**: 财运分析报告

**预估轮次**: 3-6 轮

## 服务流程 (SOP)

### Phase 1: 信息收集

**类型**: required

**描述**: 确认用户出生信息

### Phase 2: 计算排盘

**类型**: required

**描述**: 计算命盘数据

### Phase 3: 快速扫描

**类型**: required

**描述**: 快速识别财运相关特征

**扫描重点**:
1. 正财、偏财的状态
2. 食伤生财的配置
3. 比劫争财的风险
4. 财库的有无

**财运格局速判**:
| 格局 | 特征 | 财运特点 |
|-----|------|---------|
| 财星旺 | 正财/偏财有力 | 财运好，易得财 |
| 食伤生财 | 食伤生财星 | 靠才华赚钱 |
| 比劫争财 | 比劫旺 + 财弱 | 不宜合伙，防破财 |
| 财库逢冲 | 财库被冲开 | 有意外之财 |

### Phase 4: 主动追问

**类型**: optional

**描述**: 了解用户财务状况

**追问示例**:
- "你目前的收入来源是什么？"
- "有投资计划吗？"
- "最近有大额支出吗？"

### Phase 5: 深度分析

**类型**: required

**分析框架**:

1. **财运格局**
   - 正财（稳定收入）vs 偏财（意外之财）
   - 财星强弱和位置

2. **求财方式**
   - 食伤生财：靠才华、技术
   - 官杀生印：靠职位、权力
   - 比劫帮身：靠合作、人脉

3. **财运周期**
   - 当前大运财运
   - 近几年流年财运

4. **理财建议**
   - 适合的投资方向
   - 需要避免的风险

### Phase 6: 输出交付

**类型**: required

**工具调用**:
1. `show_bazi_chart` - 展示命盘（突出财星）
2. `show_insight(type="wealth")` - 财运洞察

**输出结构**:
```
【财运命盘】
{调用 show_bazi_chart}

【财运格局】
{调用 show_insight}

你的命盘显示 {财运核心结论}。

**求财方式**: {适合的赚钱方式}
**投资建议**: {适合的投资方向}
**注意事项**: {需要避免的}

当前大运{大运分析}，{财运时机建议}。
```

### Phase 7: 答疑讨论

**类型**: reactive

**常见问题**:
| 问题 | 处理 |
|-----|------|
| "适合投资吗" | 分析财星和流年 |
| "什么时候财运好" | 分析大运流年财星 |
| "会破财吗" | 分析比劫和冲克 |
