# 枭神夺食分析

## 元数据

- **ID**: xiaoshen_duoshi_analysis
- **名称**: 枭神夺食分析
- **级别**: standard
- **计费**: basic
- **描述**: 基于《滴天髓》预测论，分析枭神夺食的影响及现代修正

## 触发条件

**主要触发词**: 枭神夺食, 偏印克食神, 才华被压制, 事业受阻

**次要触发词**: 不贫则夭, 口福受损, 女命克子, 食神格

## 服务流程 (SOP)

### Phase 1: 识别枭神夺食

**目标**: 确认命局是否有枭神夺食

**执行动作**:
- 找出偏印位置
- 找出食神位置
- 判断是否形成克制

### Phase 2: 影响评估

**目标**: 评估枭神夺食的实际影响

**执行动作**:
- 食神格见偏印影响大
- 非食神格影响较小
- 检查是否有财星解救

### Phase 3: 现代修正

**目标**: 给出现代化的解读

**执行动作**:
- 古法夭字过于绝对
- 现代理解为事业受阻
- 健康风险需综合判断
