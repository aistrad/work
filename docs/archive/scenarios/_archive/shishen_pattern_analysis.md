# 十神格局定性分析流程

**触发词**: 十神格局, 格局分析, 运势分析, 时运分析, 命运分析

**来源**: 东方代码启示录.converted.md:第二十一章:001

## 分析流程

### Phase 1: 识别十神组合类型

- 确定天干十神（年干、月干、时干，不含日干）
- 判断是简单配对（两元素）还是复杂组合（三/四元素）
- 区分正类十神（正印、正���、正财、食神、比肩）和偏类十神（枭神、七杀、偏财、伤官、劫财）

### Phase 2: 简单配对定性

- 正类+正类相生：一般利好（印比相生、正财生官）
- 正类+偏类相生：主导利好带利差成分（官枭相生、杀印相生等）
- 偏类+偏类相生：利弊同存（枭劫相生、偏财生杀）
- 正类克偏类：一般利好（印克伤官、官制劫财、食神制杀等）
- 偏类克正类：一般利差（枭神夺食、偏财坏印、伤官见官、七杀克比）
- 同类克同类：利弊同存（比克正财、劫克偏财）

### Phase 3: 复杂组合定性（三元素）

- 确定终端配对（作用最终方向的配对）
- 确定期间配对（另一配对，体现因果特征）
- 判断是延续型还是逆转型
- 延续型：期间元素助推终端配对定性
- 逆转型：期间元素逆转终端配对定性

### Phase 4: 特殊规则校验

- 检查一不克二规则：一个天干不能克伤两个重叠天干，只能造成摇摆
- 注意主次层次：主导定性与次要定位分开
- 结合命主实际状况选择合适解读方向

### Phase 5: 原局与时运区分

- 原局层面：按正偏类综合强弱定性利弊
- 时运层面：按十神组合与正偏类性质定性利弊
- 时运元素插入原局构成新的十神组合
- 时运元素本身性质也构成次要影响
