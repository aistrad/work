# 偏官七煞格局分析

**触发词**: 偏官格, 七煞分析, 煞星制化, 偏官取用

**来源**: 三命通会.converted.md:卷五·论偏官:001

## 分析流程

### Phase 1: 识别偏官七煞

- 甲见��、乙见辛、丙见壬、丁见癸、戊见甲、己见乙、庚见丙、辛见丁、壬见戊、癸见己为偏官
- 隔七位而相克战，故谓之七煞
- 有制谓之偏官，无制谓之七煞

### Phase 2: 判断制化情况

- 有印绑助化:煞生印、印生身，作权贵看
- 有食神透制:一见制伏，却为贵本
- 有阳刃配合:煞无刃不显，刃煞两显威镇乾坤
- 制伏须无太过不及

### Phase 3: 分析身煞强弱

- 身强煞浅:假煞为权(吉)
- 煞重身轻:终身有损(凶)
- 身煞俱旺无制伏:虽贵不久
- 日主健旺有印绶助化为佳

### Phase 4: 检查破格因��

- 忌官煞混杂:略见一位正官，反贱
- 忌财星助煞:七煞如逢财助，其煞愈凶
- 忌三刑六害、劫亡相并、魁罡相冲

### Phase 5: 判断行运喜忌

- 原有制伏:煞出为福，喜行煞旺地
- 原无制伏:煞出为祸
- 制伏有二三处:喜行煞旺地，忌运再遇制伏
- 煞印相生:功名显达
