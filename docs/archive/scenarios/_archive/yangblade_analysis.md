# 阳刃格分析

**触发词**: 阳刃, 羊刃, 刃煞, 日刃

**来源**: 三命通会.converted.md:卷六·论阳刃

## 分析流程

### Phase 1: 识别阳刃

- 禄前一位，言旺越其分
- 惟甲丙戊庚壬五阳干有刃
- 甲见卯、丙见午、戊见午、庚见酉、壬见子

### Phase 2: 判断阳刃类型

- 劫财刃：甲见乙，不利财官格
- 护禄刃：甲见卯，大利归禄格
- 背禄刃：乙丙，大利去官留煞局

### Phase 3: 喜忌分析

- 阳刃喜偏官印绶
- 忌反吟伏吟、魁罡、三合
- 煞无刃不显，刃无煞不威
- 煞刃俱全，常人无有

### Phase 4: 运势断语

- 岁运并临，灾殃立至
- 阳刃冲合岁君，勃然祸至
- 日干无气，时逢阳刃不为凶
- 阳刃重重又见煞，大贵登甲科
