# 快速问答

## 元数据

- **ID**: quick_query
- **名称**: 快速问答
- **级别**: entry
- **计费**: free
- **描述**: 针对简单问题提供快速解答

## 触发条件

**主要触发词**: 问一下, 简单问题, 快速看看

**次要触发词**: 帮我看, 想知道

## 前置要求

- 用户出生信息（必须）
- 已计算命盘数据（必须）

## 输出配置

**使用工具**:
- show_insight

**交付物**: 简短回答

**预估轮次**: 1-2 轮

## 服务流程 (SOP)

### Phase 1: 信息收集

**类型**: required

**描述**: 确认用户出生信息和问题

### Phase 2: 计算排盘

**类型**: required

**描述**: 计算命盘数据

### Phase 3: 快速分析

**类型**: required

**描述**: 针对用户问题快速定位分析

**常见问题处理**:
| 问题类型 | 分析重点 |
|---------|---------|
| 今天运势 | 流日与命盘关系 |
| 适合做什么 | 用神五行对应行业 |
| 最近注意什么 | 流月红旗信号 |
| 贵人方位 | 用神方位 |

### Phase 4: 输出交付

**类型**: required

**输出结构**:
```
{直接回答问题，2-3句话}

{如有必要，给出简短建议}
```

**注意**: 快速问答不展开深度分析，如用户需要详细了解，引导到对应专题场景。
