# 用神取用五法分析

## 元数据

- **ID**: yongshen_wufa_analysis
- **名称**: 用神取用五法分析
- **级别**: professional
- **计费**: premium
- **描述**: 运用扶抑、病药、调候、专旺、通关五种方法综合分析八字用神，确定最佳取用策略

## 触发条件

**主要触发词**: 用神怎么取, 取用神, 用神分析, 扶抑用神, 调候用神, 通关用神

**次要触发词**: 病药取用, 专旺格, 从格, 化气格, 身强身弱

## 服务流程 (SOP)

### Phase 1: 扶抑法分析

**目标**: 判断日元强弱，确定扶抑方向

**执行动作**:
- 分析日元在月令的旺衰
- 统计四柱生扶克泄力量
- 日元强者取克泄为用
- 日元弱者取生扶为用

### Phase 2: 病药法分析

**目标**: 识别格局中的病神和药神

**执行动作**:
- 找出破坏格局的病神
- 寻找能去病的药神
- 评估病药是否得力

### Phase 3: 调候法分析

**目标**: 判断是否需要调候

**执行动作**:
- 检查金水生于冬令是否太寒
- 检查木火生于夏令是否太燥
- 确定调候用神（寒用火暖，燥用水润）

### Phase 4: 专旺法分析

**目标**: 判断是否成专旺格局

**执行动作**:
- 检查四柱气势是否偏于一方
- 判断是否可从（从财、从官、从煞等）
- 判断是否可化（化气格）
- 确定顺势取用

### Phase 5: 通关法分析

**目标**: 判断是否需要通关

**执行动作**:
- 检查是否有两神对峙
- 判断双方强弱是否均平
- 确定通关之神（如金木相战用水通关）

### Phase 6: 综合取用

**目标**: 确定最终用神

**执行动作**:
- 比较五法分析结果
- 确定主要用神和辅助用神
- 分析用神在原局的力量
