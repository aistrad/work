# 六亲关系分析

## 元数据

- **ID**: six_relatives_analysis
- **名称**: 六亲关系分析
- **级别**: standard
- **计费**: basic
- **描述**: 分析命主与父母、兄弟、配偶、子女等六亲的关系和缘分

## 触发条件

**主要触发词**: 六亲分析, 家庭关系, 父母缘分, 子女运

**次要触发词**: 兄弟姐妹, 婚姻配偶, 亲属关系, 家族运势

## 服务流程 (SOP)

### Phase 1: 六亲星位确定

**目标**: 确定六亲在八字中的代表

**执行动作**:
- 印绶为母
- 偏财为父
- 比劫为兄弟
- 正财为妻
- 官杀为子女(男命)
- 食伤为子女(女命)

### Phase 2: 宫位分析

**目标**: 分析年月日时四柱代表的六亲宫位

**执行动作**:
- 年柱为祖业父母
- 月柱为兄弟门户
- 日柱为己身配偶
- 时柱为子孙

### Phase 3: 旺衰吉凶判断

**目标**: 判断六亲星的旺衰和吉凶

**执行动作**:
- 坐长生旺库禄马贵人主富贵
- 坐刑克煞死绝主贫薄刑夭
- 有生有助主得力
- 受克受冲主不利

### Phase 4: 综合论断

**目标**: 给出六亲关系的综合分析

**执行动作**:
- 父母缘分深浅
- 兄弟是否得力
- 配偶贤愚美丑
- 子女多寡贤否
