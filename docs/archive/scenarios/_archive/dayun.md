# 大运分析

## 元数据

- **ID**: dayun
- **名称**: 大运分析
- **级别**: standard
- **计费**: basic
- **描述**: 分析十年大运周期，识别人生转折点和发展机遇

## 触发条件

**主要触发词**: 大运, 十年运, 运势周期, 人生阶段

**次要触发词**: 转运, 什么时候好, 运气变化

## 前置要求

- 用户出生信息（必须）
- 已计算命盘数据（必须）
- 大运排盘数据（必须）

**使用工具**:

- show_bazi_kline

**交付物**: 大运分析报告 + 运势K线图

**预估轮次**: 3-6 轮

### Phase 5: 深度分析

**类型**: required

**分析框架**:

#### 华山八字大运分析三步骤（源自《东方代码启示录》）

**Step 1: 十神格局分析（最重要）**
将大运干支插入原局，看天干是否构成十神生克组合：

| 格局 | 寓意 | 事业影响 |
|-----|------|---------|
| 正财生官 | 平步青云 | 循序渐进型成功 |
| 杀印相生 | 功名显达 | 权威领袖之运 |
| 伤官配印 | 名利双收 | 专业突出之运 |
| 食神制杀 | 化险为夷 | 以柔克刚之运 |
| 偏财生杀 | 投机博弈 | 冒险与机遇并存 |
| 枭神夺食 | 多劳少获 | 需防是非周折 |
| 伤官见官 | 事业波折 | 需防官非口舌 |
| 偏财坏印 | 名声受损 | 需防重财轻命 |

**Step 2: 冲合建影响**
大运干支对原局各元素的影响：
- **冲**: 动荡、变化、不稳定
- **合**: 牵绊、机遇、关系变化
- **建**: 临旺、强化、该元素特征突出

**Step 3: 大运本身解读**
- 天干正偏类定性：正类（印、官、财、食、比）为正面，偏类（枭、杀、偏财、伤、劫）需谨慎
- 地支星煞特征：贵人、禄神、桃花、羊刃等

#### 传统分析补充

1. **当前大运分析**
   - 大运五行对日主的影响
   - 大运与命局的互动
   - 当前阶段的主题

2. **历史大运回顾**
   - 过去大运的特点
   - 验证分析准确性

3. **未来大运预测**
   - 下一步大运特点
   - 转折点时间
   - 发展建议

4. **关键年份标注**
   - 大运交接年
   - 特殊流年

#### 主客体配合原则

| 组合 | 结果 |
|-----|------|
| 客体利好 + 主体强势 | 利好发挥到极致 |
| 客体利好 + 主体弱势 | 利好被摊薄 |
| 客体利差 + 主体强势 | 利差程度减轻 |
| 客体利差 + 主体弱势 | 利差发挥到极致 |
