# 伤官格分析

**触发词**: 伤官, 伤官见官, 伤官伤尽, 伤官生财

**来源**: 三命通会.converted.md:卷六·论伤官

## 分析流程

### Phase 1: 识别伤官格

- 我生彼之谓(甲见丁、乙见丙等)
- 伤官能克制官星
- 伤官格务要伤尽方作贵看

### Phase 2: 判断伤尽与否

- 月令在伤官，四柱作合结局皆在伤位
- 无冲无破，不见一点官星谓之伤尽
- 月支伤官，时上伤官，四柱无官星亦谓伤尽

### Phase 3: 五行伤官喜忌

- 火土伤官宜伤尽，忌见官星
- 金水伤官要见官
- 木火见官官要旺
- 土金官去反成官
- 水木伤官格，财官两见始为欢

### Phase 4: 用神取舍

- 伤官用印宜去财
- 用财宜去印
- 身旺者用财，身弱者用印
- 伤官最喜行财运
