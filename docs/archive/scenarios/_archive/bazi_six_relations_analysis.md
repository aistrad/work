# 六亲关系分析

## 元数据

- **ID**: bazi_six_relations_analysis
- **名称**: 六亲关系分析
- **级别**: standard
- **计费**: basic
- **描述**: 基于八字分析六亲（父母、配偶、子女、兄弟）的吉凶关系

## 触发条件

**主要触发词**: 分析我的婚姻, 看看我的姻缘, 子女运势, 父母缘分

**次要触发词**: 夫妻关系, 能生几个孩子, 兄弟姐妹, 家庭运

## 服务流程 (SOP)

### Phase 1: 确定六亲宫位

**目标**: 定位六亲在八字中的位置

**执行动作**:
- 月柱为父母宫
- 日支为配偶宫
- 时柱为子女宫

### Phase 2: 分析六亲星

**目标**: 找出代表六亲的十神

**执行动作**:
- 印绶为父母
- 财星为妻(男命)
- 官杀为夫(女命)
- 食伤为子女
- 比劫为兄弟

### Phase 3: 判断吉凶

**目标**: 分析六亲星的强弱与喜忌

**执行动作**:
- 查看六亲星是否为用神
- 分析六亲星是否受克破
- 判断六亲星的旺衰

### Phase 4: 综合论断

**目标**: 给出六亲关系的具体建议

**执行动作**:
- 结合大运流年
- 给出趋吉避凶建议
