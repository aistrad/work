# 特殊格局分析

**触发词**: 格局, 特殊格局, 飞天禄马, 倒冲禄, 壬骑龙背, 六乙鼠贵, 六阴朝阳, 拱禄拱贵

**来源**: 三命通会.converted.md:卷六·格局判断

## 分析流程

### Phase 1: 识别日主与时辰

- 确定日干是否为特定日主(庚子、壬子、辛亥、癸亥等)
- 检查时支是否符合特殊格局要求
- 判断是��为飞天禄马、倒冲禄等格局

### Phase 2: 检查格局成立条件

- 检查柱中是否无财官显露
- 检查是否有冲神多位并冲
- 检查是否有合神羁绊
- 检查是否有填实破格

### Phase 3: 判断格局纯粹程度

- 纯粹者主大贵
- 有小瑕疵者减分数
- 填实冲刑者不入格

### Phase 4: 运势分析

- 喜行身旺及伤官食神运
- 忌行财官显露之地
- 岁运同论
