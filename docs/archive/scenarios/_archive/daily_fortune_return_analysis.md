# 日禄归时格分析

**触发词**: 日禄归时, 归禄, 青云得路, 时上禄

**来源**: 三命通会.converted.md:卷六·日禄归时

## 分析流程

### Phase 1: 识别日禄归时格

- 确定是否为七日之一(甲寅、丁午、戊巳、己午、庚申、壬亥、癸子)
- 检查日主之禄是否归于时位

### Phase 2: 检查格局喜忌

- 喜日干坐旺、印绶生
- 喜月透财元、伤食
- 喜天月二德
- 忌刑冲破害、空亡死绝
- 忌劫财分禄、倒食作合、官煞克制

### Phase 3: 判断七法类型

- 青云得路：无官星，身旺印助
- 官星坐禄：官星临时支禄位
- 归禄逢二德：带天月二德
- 归禄逢印绶：印绶生身
- 归禄逢伤官：伤官去煞
- 归禄逢煞：七煞有制
- 归禄逢财：自坐财库

### Phase 4: 富贵断语

- 主大富贵
- 无财归禄亦须贫
- 有刑害其福减半
