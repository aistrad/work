# 十神格局分析 (Ten Gods Pattern Analysis)

- scenario_id: ten_gods_pattern_analysis
- level: standard

## 触发条件
- primary_triggers: ["分析格局", "十神格局", "八字格局", "命格分析", "格局是什么", "什么格局"]
- context_triggers: ["看看我的格局", "格局高低", "格局好不好"]

## 场景描述
基于十神生克关系分析八字格局类型，包含20种经典格局的识别与解读。

## SOP 流程

### Phase 1: 格局识别 (required)
1. 确定日主五行和强弱
2. 识别月令十神
3. 分析天干透出的十神
4. 判断十神之间的生克关系

### Phase 2: 格局分类 (required)
根据十神组合判断格局类型：

**印星相关格局**:
- 杀���相生: 七杀生印星，化杀为权
- 官印相生: 正官生印星，名正言顺
- 财制枭: 偏财克制枭神
- 偏财坏印: 偏财克制正印
- 官生枭: 正官生枭神

**食伤相关格局**:
- 食神制杀: 食神克制七杀
- 伤官配印: 伤官与印星配合
- 伤官见官: 伤官克制正官
- 食神生财: 食神生偏财

**财官相关格局**:
- 正财生官: 正财生正官
- 偏财生杀: 偏财生七杀
- 官制劫: 正官克制劫财
- 杀制劫: 七杀克制劫财

**比劫相关格局**:
- 劫财夺财: 劫财克制正财
- 比肩分财: 比肩分夺偏财
- 枭生劫: 枭神生劫财

### Phase 3: 格局解读 (required)
1. 分析格局的吉凶属性
2. 解读格局对性格的影响
3. 预测格局对事业、财运、感情的影响
4. 给出针对性建议

### Phase 4: 案例参考 (optional)
引用历史人物案例进行对比分析：
- 宣统皇帝 (财制枭)
- 汪精卫 (偏财坏印)
- 马占山将军 (官生枭)
- 梅兰芳 (杀印相生)
