# 健康寿元分析

## 元数据

- **ID**: health_longevity_analysis
- **名称**: 健康寿元分析
- **级别**: professional
- **计费**: premium
- **描述**: 基于《滴天髓》寿夭论，分析命局的健康状况和寿元信息

## 触发条件

**主要触发词**: 健康运势, 身体怎么样, 寿元分析, 健康风险

**次要触发词**: 性定元气厚, 气浊神枯, 五行流通, 刑冲破害

## 服务流程 (SOP)

### Phase 1: 元气判断

**目标**: 判断日主元气厚薄

**执行动作**:
- 检查日主有根有气
- 分析五行流通状态
- 判断是否有刑冲破害伤及日主

### Phase 2: 健康风险识别

**目标**: 识别潜在健康风险

**执行动作**:
- 五行偏枯分析
- 极端失衡检测
- 刑冲破害程度评估

### Phase 3: 养生建议

**目标**: 给出调理建议

**执行动作**:
- 根据五行偏向给出养生方向
- 提示需注意的健康领域
- 说明现代医学进步的修正
