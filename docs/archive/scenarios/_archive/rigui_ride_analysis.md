# 日贵日德格局分析

**触发词**: 日贵, 日德, 丁酉, 丁亥, 癸巳, 癸卯, 甲寅, 戊辰, 丙辰, 庚辰, 壬戌

**来源**: 渊海子平.converted.md:日贵日德:001

## 分析流程

### Phase 1: 确认日贵日德

- 日贵有四日：丁酉、丁亥、癸巳、癸卯
- 日德有五：甲寅、戊辰、丙辰、庚辰、壬戌日

### Phase 2: 分析格局特点

- 日贵主人纯粹、有仁德、有姿色、不傲物
- 日德主人性格慈善
- 日德俱多，福必丰厚

### Phase 3: 判断忌神

- 日贵最怕刑冲破害
- 日贵忌空亡、魁罡
- 日德忌刑冲破害、恶官星、憎财旺

### Phase 4: 行运吉凶

- 日贵运行怕空亡及太岁加会
- 日德运行身旺，大是奇绝
- 旺气已衰，运至魁罡，其死必矣
