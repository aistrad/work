# 六亲关系分析

**触发词**: 六亲, 父母, 兄弟, 妻财, 子息, 克父, 克母, 克妻, 克子

**来源**: 渊海子平.converted.md:论六亲:001

## 分析流程

### Phase 1: 确定六亲星位

- 正印为正母，偏印为偏母及祖父
- 偏财为父，亦为偏妻
- 正财为妻
- 比肩为兄弟姐妹
- 七杀为男，正官为女
- 食神为男孙，伤官为女孙及祖母

### Phase 2: 分析六亲受克

- 印绶见财，克母及祖母
- 见比劫阳刃，克妻妾及父
- 官杀多者，难为兄弟
- 伤官食神多，难为子息
- 枭印伤孙，克祖母

### Phase 3: 判断六亲吉凶

- 六亲星临生旺贵人天月德地，主有贵
- 六亲星临杀地，死他乡
- 六亲星居衰败受制墓绝之地，主平常
- 六亲星作合，主不正

### Phase 4: 岁运应期

- 以岁运见何字，则克何人
- 将冲克衰旺向背推断
- 将来者进，功成者退
- 兼看孤神寡宿、空亡
