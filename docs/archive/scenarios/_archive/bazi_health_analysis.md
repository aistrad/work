# 健康疾病分析

## 元数据

- **ID**: bazi_health_analysis
- **名称**: 健康疾病分析
- **级别**: standard
- **计费**: basic
- **描述**: 基于五行配五脏分析命主的健康状况与潜在疾病风险

## 触发条件

**主要触发词**: 健康分析, 身体状况, 容易得什么病, 健康运势

**次要触发词**: 五行健康, 疾病预防, 养生建议, 体质分析

## 服务流程 (SOP)

### Phase 1: 五行配五脏

**目标**: 建立五行与脏腑的对应关系

**执行动作**:
- 木为肝
- 金为肺
- 水为肾
- 火为心
- 土为脾

### Phase 2: 分析五行太过

**目标**: 找出八字中过旺的五行

**执行动作**:
- 统计各五行数量
- 判断哪个五行太过
- 对应脏腑可能有病

### Phase 3: 分析五行不及

**目标**: 找出八字中过弱的五行

**执行动作**:
- 统计各五行数量
- 判断哪个五行不及
- 对应脏腑可能有病

### Phase 4: 调和建议

**目标**: 给出五行调和的养生建议

**执行动作**:
- 泄其旺神
- 泻其有余
- 补其不足
- 达到五行平和
