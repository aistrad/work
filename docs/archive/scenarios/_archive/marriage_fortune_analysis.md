# 婚姻运势分析

**触发词**: 婚姻, 夫妻, 配偶, 结婚, 感情, 姻缘

**来源**: 滴天髓阐微.converted.md:六亲论夫妻:scenario001

## 分析流程

### Phase 1: 识别财星状态

- 确定日主五行属性
- 找出命局中的正财和偏财
- 判断财星是否为喜神
- 分析财星的旺衰强弱

### Phase 2: 分析财星与日主关系

- 判断财星是否逢合（财来就我）
- 检查是否有比劫争财
- 分析财星是否坏印
- 判断财星是否生杀

### Phase 3: 判断婚姻吉凶

- 喜神即是财星：妻美且富贵
- 财星逢合：妻贤淑勤能
- 乙庚化金生杀坏印：妻不贤妒悍
- 财轻劫重：主克妻
- 财重身弱无比劫：主克妻

### Phase 4: 给出建议

- 根据财星状态给出婚姻建议
- 指出有利的婚配方向
- 提示需要注意的流年大运
