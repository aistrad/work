# 不利时运应对指导

## 触发词
- 运气不好
- 走霉运
- 不利时运
- 坏运
- 倒霉
- 流年不利
- 大运不好

## SOP 流程

### Phase 1: 分析时运特征
- 类型: required
- 工具: analyze_current_luck, analyze_ten_gods
- 说明: 识别当前大运/流年的十神组合与性质

### Phase 2: 评估不利程度
- 类型: required
- 说明: 判断哪些表现可承受、哪些需避免

### Phase 3: 避重就轻策略
- 类型: required
- 说明: 主动避开大不利，迎合小不利让特征落实

### Phase 4: 制化方案
- 类型: required
- 说明: 选择制或化的方法转化不利时运

### Phase 5: 具体化解措施
- 类型: required
- 工具: generate_remedy_plan
- 说明: 从十种模式中选择具体措施
