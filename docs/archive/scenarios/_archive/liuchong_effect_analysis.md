# 六冲效应分析

## 元数据

- **ID**: liuchong_effect_analysis
- **名称**: 六冲效应分析
- **级别**: standard
- **计费**: basic
- **描述**: 基于《滴天髓》冲克论，分析命局中六冲的实际效应

## 触发条件

**主要触发词**: 六冲, 地支相冲, 冲克分析, 子午冲

**次要触发词**: 旺者冲衰, 衰者冲旺, 冲动, 冲破

## 服务流程 (SOP)

### Phase 1: 识别六冲

**目标**: 找出命局中的六冲组合

**执行动作**:
- 检查子午冲
- 检查丑未冲
- 检查寅申冲
- 检查卯酉冲
- 检查辰戌冲
- 检查巳亥冲

### Phase 2: 旺衰判断

**目标**: 判断冲克双方的旺衰

**执行动作**:
- 分析冲方在月令的状态
- 判断谁旺谁衰

### Phase 3: 效应分析

**目标**: 根据旺衰判断冲的结果

**执行动作**:
- 旺冲衰则衰方受损
- 衰冲旺则旺方被激发
- 给出具体影响说明
