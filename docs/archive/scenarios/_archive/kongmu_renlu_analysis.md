# 空墓刃禄分析流程

**触发词**: 旬空, 墓库, 羊刃, 食禄, 禄神, 空亡

**来源**: 东方代码启示录.converted.md:第十九章:001

## 分析流程

### Phase 1: 计算旬空

- 以日柱为计算主体
- 十天干配十二地支，多出两个地支为旬空
- 旬空只出现在年支、月支、时支，日支不可能旬空

### Phase 2: 分析旬空寓意

- 某类缺失：代表某方面明显缺失
- 时点强化：填空冲空时点作用力增强
- 平缓耐冲：逢冲时承受力更强更耐损
- 悬空不落实：总有令人牵挂不踏实之处

### Phase 3: 判断墓库利弊

- 确定五行墓库：丑为金库、未为木库、辰为水库、戌为火库
- 多者逢库为控（利好）：约束泛滥元素
- 少者逢墓为困（利差）：困住稀缺元素
- 正类十神落单衰弱见墓库为利差
- 偏类十神见墓库一般为利好

### Phase 4: 分析羊刃

- 计算羊刃：甲刃卯、乙刃寅、丙戊刃午、丁己刃巳、庚刃酉、辛刃申、壬刃子、癸刃亥
- 主体角度：羊刃可使身弱转身强
- 客体角度：伤官/七杀/正印配刃为权力标志
- 一般人带刃：弊多利少，防是非伤病血光
- 有刃者忌羊刃逢冲之运

### Phase 5: 分析食禄

- 计算食禄：甲禄寅、乙禄卯、丙戊禄巳、丁己禄午、庚禄申、辛禄酉、壬禄亥、癸禄子
- 食禄代表丰衣足食、享受、积蓄、回报
- 食神/偏财/正财/正官配禄为财运回报丰厚
- 禄空或禄逢冲为回禄负面信号
- 权力型八字见禄会打折扣（名利不兼收）

### Phase 6: 应用指导

- 月令旬空：心态包容但常有牵挂，婚姻宫空则姻缘贫乏
- 缺少者逢墓运：可养五行来充实墓库
- 太多元素：人为设置墓库来控制规范
- 禄者从商刃者从政，顺势而为
