# 十干化气格局分析

**触发词**: 化气, 合化, 甲己化土, 乙庚化金, 丙辛化水, 丁壬化木, 戊癸化火, 化气格

**来源**: 三命通会.converted.md:卷二·论十干化气:005

## 分析流程

### Phase 1: 确定天干合

- 甲己合化土
- 乙庚合化金
- 丙辛合化水
- 丁壬合化木
- 戊癸合化火

### Phase 2: 判断化气条件

- 日干与配合之神相合
- 月令得旺气
- 时上得旺气亦可

### Phase 3: 检查妒合

- 甲己化土忌戊字间之
- 乙庚化金忌甲字间之
- 丙辛化水忌丁字间之
- 丁壬化木忌丙字间之
- 戊癸化火忌己字间之

### Phase 4: 分析化气格局

- 正化：地支相连同气
- 转角进化：四角相顺连
- 转角退化：四角相返连
- 座下自化：如壬午、丁亥等

### Phase 5: 论断吉凶

- 化气成格主贵
- 受气临官晚年不吉
- 确定化气格的喜忌用神
