## 分析流程

### Phase 1: 确定十二宫基数

- 以日干为分析基数（分析自身相关）
- 以其他天干为分析基数（分析该十神相关）
- 以隐性天干为分析基数（分析未透出的六亲）

### Phase 2: 定位十二宫位置

- 阳干顺行：甲木长生亥、丙火长生寅、戊土长生申、庚金长生巳、壬水长生申
- 阴干逆行：乙木长生午、丁火长生酉、己土长生卯、辛金长生子、癸水长生卯
- 根据地支确定十二宫位置

### Phase 3: 五种解读方式

- 方式一：按宫位排序解读先后顺序（长生最早，绝最后，胎养为潜伏）
- 方式二：按宫位旺衰定性利弊（上升期利好，下降期利差，蓄势期平平）
- 方式三：按宫位引申含义解读
- 方式四：按时运地支套取十二宫特征
- 方式五：按隐性天干身份套取十二宫信息

### Phase 4: 解读宫位含义

- 长生：成立、启始、依靠、维持、受恩、眷顾
- 沐浴：洗浴、风流、潇洒、感情、浪漫、洗脱
- 冠带：堂皇、装饰、仪表、礼仪、社交、打扮
- 临官：升官、上任、风光、任职、荣耀、受聘
- 帝旺：鼎盛、高峰、成就、成熟、强壮、极限
- 衰：衰落、下拐、转弱、不继、后退、收缩
- 病：病痛、缺陷、弊端、问题、不足、损失
- 死：死板、僵化、不变、深入、平静、尾端
- 墓：暗晦、藏匿、受制、管束、迷茫、库存
- 绝：绝境、终结、最后、很少、消失、稀疏
- 胎：胎儿、萌芽、酝酿、构思、牵连、连累
- 养：休息、疑惑、心虚、潜伏、呵护、前期

### Phase 5: 综合分析

- 注意：十二宫只作细节层面解读，不能覆盖定性层面
- 注意：宫位作用有针对性，以谁为基数就针对谁
- 结合柱位、十神、时运综合解读
