# 月支与日干吉凶分析

**触发词**: 月令分析, 月支吉凶, 日干与月支关系, 月令用神

**来源**: 三命通会.converted.md:卷四·论十二月支得日干吉凶:001

## 分析流程

### Phase 1: 确定月支与日干关系

- 识别月支(子丑寅卯辰巳午未申酉戌亥)
- 识别日干(甲乙丙丁戊己庚辛壬癸)
- 判断月支对日干的十神关系(印绑、官、财、建禄、偏官等)

### Phase 2: 分析喜忌条件

- 若为印绑月:喜见官露印,忌坐天财伤印
- 若为官月:喜露财官、三合六合,身旺喜财官,身弱喜印旺
- 若为财月:喜露财身旺,忌坐刃透比
- 若为建禄月:身强无格可取,宜带时偏官或诸不见之形
- 若为偏官月:喜身旺有合制,忌身弱无合露

### Phase 3: 判断行运喜忌

- 身旺者喜行财官运
- 身弱者喜行印旺运
- 有偏官者喜行合偏官运,忌正官运
- 有财者忌行劫比运
