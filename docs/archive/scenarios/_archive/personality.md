# 性格简析

## 元数据

- **ID**: personality
- **名称**: 性格简析
- **级别**: entry
- **计费**: free
- **描述**: 从八字命盘分析性格特点和天赋潜能

## 触发条件

**主要触发词**: 性格, 我是什么人, 我的性格, 天赋

**次要触发词**: 脾气, 个性, 优缺点

## 前置要求

- 用户出生信息（必须）
- 已计算命盘数据（必须）

## 输出配置

**使用工具**:
- show_bazi_chart
- show_insight

**交付物**: 性格分析卡片

**预估轮次**: 1-3 轮

## 服务流程 (SOP)

### Phase 1: 信息收集

**类型**: required

**描述**: 确认用户出生信息

### Phase 2: 计算排盘

**类型**: required

**描述**: 计算命盘数据

### Phase 3: 快速扫描

**类型**: required

**描述**: 识别性格相关特征

**扫描重点**:
1. 日主五行特性
2. 十神配置（食伤、印星、比劫）
3. 五行偏枯情况

**性格速判**:
| 特征 | 性格倾向 |
|-----|---------|
| 日主旺 + 比劫多 | 独立、自信、有主见 |
| 日主弱 + 印星旺 | 内敛、善思、依赖性强 |
| 食伤旺 | 聪明、表达欲强、创意多 |
| 官杀旺 | 有责任感、压力大、守规矩 |

### Phase 4: 深度分析

**类型**: required

**分析框架**:

1. **核心性格**（日主特性）
2. **思维方式**（印星、食伤）
3. **行为模式**（比劫、官杀）
4. **情感特点**（财星、桃花）

### Phase 5: 输出交付

**类型**: required

**工具调用**:
1. `show_bazi_chart` - 展示命盘
2. `show_insight(type="personality")` - 性格洞察

**输出结构**:
```
【命盘概览】
{调用 show_bazi_chart}

【性格特点】
{调用 show_insight}

你是{日主}日主，{一句话性格概括}。

**优势**: {2-3个优点}
**注意**: {1-2个需要注意的地方}
```
