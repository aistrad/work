# 官煞去留分析

**触发词**: 官煞混杂, 去官留煞, 去煞留官, 合煞为贵

**来源**: 三命通会.converted.md:卷五·官煞去留杂论:001

## 分析流程

### Phase 1: 识别官煞混杂情况

- 月上见官时上见煞
- 月上见煞时上见官
- 四柱叠见官煞

### Phase 2: 判断官煞轻重

- 天干透者易去
- 月支所藏者难去
- 看柱中官煞孰重孰轻

### Phase 3: 确定去留方式

- 五阳日食神:能去煞又能留官
- 五阳日伤官:但能去官不能留煞，须羊刃合方成去官留煞
- 五阴日食神:能去煞却不能留官，日主自能留之
- 五阴日伤官:能去官又能留煞

### Phase 4: 分析去留结果

- 去煞留官有情而贵
- 去官留煞有情而贵
- 两停俱无扶合有破害:斟酌有力一字为用神
- 两停俱有扶合无破害:官煞混杂反为贫贱

### Phase 5: 特殊情况处理

- 贪合忘官:闲神合去官星，不为贵
- 贪合忘煞:闲神与七煞合，合煞忘贱
- 身旺无物去煞:亦能化煞为官
