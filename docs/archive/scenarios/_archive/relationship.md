# 感情分析

## 元数据

- **ID**: relationship
- **名称**: 感情分析
- **级别**: standard
- **计费**: basic
- **描述**: 从八字角度分析感情运势、桃花时机和婚姻状况

## 触发条件

**主要触发词**: 感情, 婚姻, 桃花, 恋爱, 对象

**次要触发词**: 脱单, 结婚, 离婚, 复合

## 前置要求

- 用户出生信息（必须）
- 已计算命盘数据（必须）
- 性别（必须）

## 输出配置

**使用工具**:
- show_bazi_chart
- show_insight

**交付物**: 感情分析报告

**预估轮次**: 3-6 轮

## 服务流程 (SOP)

### Phase 1: 信息收集

**类型**: required

**描述**: 确认用户出生信息和性别

### Phase 2: 计算排盘

**类型**: required

**描述**: 计算命盘数据

### Phase 3: 快速扫描

**类型**: required

**描述**: 快速识别感情相关特征

**扫描重点**:
1. 配偶星（男看财星，女看官杀）
2. 配偶宫（日支）状态
3. 桃花星位置
4. 婚姻宫刑冲情况

**感情格局速判**:
| 特征 | 感情倾向 |
|-----|---------|
| 配偶星旺 + 配偶宫稳 | 感情顺利 |
| 配偶星弱/无 | 晚婚或感情淡 |
| 桃花多 | 异性缘好，需防烂桃花 |
| 日支逢冲 | 婚姻有波折 |

### Phase 4: 主动追问

**类型**: optional

**描述**: 了解用户感情状况

**追问示例**:
- "你目前的感情状态如何？"
- "有心仪的对象吗？"
- "对另一半有什么期望？"

### Phase 5: 深度分析

**类型**: required

**分析框架**:

1. **感情格局**
   - 配偶星的强弱和位置
   - 配偶宫的状态

2. **理想对象**
   - 根据配偶星五行推断
   - 性格特点和职业方向

3. **桃花时机**
   - 当前大运桃花运
   - 近几年流年桃花

4. **婚姻建议**
   - 适合的相处方式
   - 需要注意的问题

### Phase 6: 输出交付

**类型**: required

**工具调用**:
1. `show_bazi_chart` - 展示命盘（突出感情相关星）
2. `show_insight(type="relationship")` - 感情洞察

**输出结构**:
```
【感情命盘】
{调用 show_bazi_chart}

【感情格局】
{调用 show_insight}

你的命盘显示 {感情核心结论}。

**理想对象**: {对象特点}
**桃花时机**: {桃花旺的时间}
**相处建议**: {感情建议}

当前大运{大运分析}，{感情时机建议}。
```

### Phase 7: 答疑讨论

**类型**: reactive

**常见问题**:
| 问题 | 处理 |
|-----|------|
| "什么时候能脱单" | 分析桃花流年 |
| "适合结婚吗" | 分析配偶宫和流年 |
| "会离婚吗" | 委婉分析，给出建议 |
