# 三刑分析

**触发词**: 三刑, 刑, 无恩之刑, 恃势之刑, 无礼之刑, 自刑, 寅巳申, 丑戌未, 子卯

**来源**: 三命通会.converted.md:卷二·论三刑:006

## 分析流程

### Phase 1: 识别刑的类型

- 寅巳申为无恩之刑
- 丑戌未为恃势之刑
- 子卯为无礼之刑
- 辰午酉亥为自刑

### Phase 2: 分析无恩之刑

- 寅刑巳，巳刑申，申刑寅
- 不恤所生，遥相克制
- 主人持重少语，寡欲无情

### Phase 3: 分析恃势之刑

- 丑刑戌，戌刑未，未刑丑
- 各恃旺气相刑
- 主人精神意气雄豪

### Phase 4: 分析无礼之刑

- 子刑卯，卯刑子
- 子母自相刑，无钦卑之道
- 女命见之尤为不良

### Phase 5: 判断刑的吉凶

- 有吉神相扶则刑不为害
- 带凶煞则祸不可言
- 君子不刑定不发，小人到此必为灾
