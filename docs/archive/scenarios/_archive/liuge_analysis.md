# 六格取用分析

**触发词**: 六格, 取格, 格局, 官格, 印格, 财格, 杀格, 食神格

**来源**: 渊海子平.converted.md:宝法:001

## 分析流程

### Phase 1: 确定月令格局

- 子平一法，专以日干为主
- 取提纲所藏之物为令
- 凡格用月令提纲，勿于傍求年日时为格

### Phase 2: 六格分类

- 官格、印格、财格、杀格、食神格、伤官格
- 逢官看财
- 逢杀看印
- 逢印看官

### Phase 3: 格局成败判断

- 有生有去为福
- 有助有剥为祸
- 不偏不倚、生克制化为上
- 破体囚为下运

### Phase 4: 配合年日时

- 用年日下推轻重浅深
- 月令通官，柱中遇财，财生官妙
- 但见一杀，则以杀为重
