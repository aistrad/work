# 天干合化分析

## 元数据

- **ID**: tiangan_hehua_analysis
- **名称**: 天干合化分析
- **级别**: professional
- **计费**: premium
- **描述**: 分析八字中天干五合（甲己、乙庚、丙辛、丁壬、戊癸）是否能够化气，判断化气真假，以及合化对格局的影响

## 触发条件

**主要触发词**: 天干合化, 五合, 甲己合, 乙庚合, 丙辛合, 丁壬合, 戊癸合, 化气格

**次要触发词**: 合而不化, 化气真假, 争合妒合, 合官留煞

## 服务流程 (SOP)

### Phase 1: 识别天干配合

**目标**: 找出四柱中存在的天干五合关系

**执行动作**:
- 检查年月日时四柱天干
- 识别甲己、乙庚、丙辛、丁壬、戊癸配合
- 判断合的位置（相邻或隔位）

### Phase 2: 判断能否成合

**目标**: 分析合是否有效

**执行动作**:
- 检查是否有间隔（如甲己中间有庚）
- 判断隔位远近（年干与时干为遥合）
- 分析争合妒合情况（两干合一干）

### Phase 3: 判断能否化气

**目标**: 分析合后是否能化

**执行动作**:
- 检查所临地支是否通根乘旺
- 参照月令判断化气条件
- 区分真化与假化

### Phase 4: 分析合化影响

**目标**: 评估合化对格局的影响

**执行动作**:
- 判断用神是否因合失用
- 分析忌神是否因合化吉
- 评估日主本身相合的特殊情况
