# 命运改善化解服务

## 分析流程

### Phase 1: 审时度势

- 分析八字原局的惯性特征
- 分析当前时运的特征
- 识别利好与不利的时段、方位、人脉
- 明确天时地利人和的配合点

### Phase 2: 避重就轻

- 理解轨迹惯性是特征本质而非具体表现
- 识别可承受的小不利与不可承受的大不利
- 主动避开可能引发大不利的人事
- 主动迎合小不利让惯性特征落实

### Phase 3: 强化优势

- 识别先天禀赋优势
- 优先强化优势天赋而非弥补缺点
- 在交往中表现对方的弱项以强化自身影响力

### Phase 4: 确定化解思路

- 避的思路：主动躲开不利的地域、场合、人脉
- 应的思路：主动迎合时运特征的某些表现
- 制的思路：用相克元素制约不利时运
- 化的思路：利用时运特性转化为另一种能量（首选）

### Phase 5: 选择化解模式

- 1.利用天时化解：择日、选择合适时运做事
- 2.利用环境化解：选择适合五行的居住环境
- 3.利用地域化解：选择有利方位发展
- 4.利用空间化解：个性风水布局
- 5.利用人脉化解：寻找合适的贵人帮扶
- 6.利用物件化解：佩戴寓意相应五行的物品
- 7.利用行为化解：采取相应十神特征的行为
- 8.利用色泽化解：选择相应五行的颜色
- 9.利用命名化解：名字中加入相应五行的字
- 10.利用职业化解：选择适合五行的行业

### Phase 6: 通用趋吉方法

- 施舍行善：积德气场，改善先天惯性不足
- 专注知识：进修学习，缓解不利时运
- 皈依正教：增加人生弹性与希望
- 种采冲喜：营造好开端，冲喜赶走霉运
- 放生：积累善缘
- 放下：调整心态
