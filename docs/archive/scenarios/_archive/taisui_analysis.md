# 太岁流年分析

**触发词**: 太岁, 流年, 犯太岁, 冲太岁, 本命年, 年运

**来源**: 三命通会.converted.md:卷二·论太岁:004

## 分析流程

### Phase 1: 确定太岁关系

- 识别当年太岁干支
- 判断太岁与日干关系
- 判断太岁与年支关系

### Phase 2: 分析岁君伤日

- 太岁克日干为偏官
- 如君治臣，灾祸较轻
- 上治其下，顺也

### Phase 3: 分析日犯岁君

- 日干克太岁为偏财
- 如臣犯君，灾殃必重
- 下凌上，逆也

### Phase 4: 判断有救无救

- 四柱有情则有救
- 大运相助则减凶
- 天月德解则无咎

### Phase 5: 特殊太岁格局

- 真太岁：生年与流年同干支
- 征太岁：日干支冲克太岁
- 岁运并临：流年与大运同干支
