# 八字命盘综合评断

## 元数据

- **ID**: bazi_comprehensive_analysis
- **名称**: 八字命盘综合评断
- **级别**: professional
- **计费**: premium
- **描述**: 基于千里命稿的八步评断程序，对八字命盘进行系统性综合分析

## 触发条件

**主要触发词**: 帮我分析八字, 看看我的命盘, 八字综合分析, 命理分析

**次要触发词**: 算命, 批八字, 看命, 测八字

## 服务流程 (SOP)

### Phase 1: 看强弱

**目标**: 判断日干强弱

**执行动作**:
- 以日干为主
- 分析多寡盛衰
- 判断失时得令

### Phase 2: 定格局

**目标**: 确定命局格局

**执行动作**:
- 以月支为标准
- 判断八格或外格

### Phase 3: 取用神

**目标**: 确定用神

**执行动作**:
- 以助强扶弱为标准
- 选取最有利的五行

### Phase 4: 论喜忌

**目标**: 分析喜忌

**执行动作**:
- 以用神为标准
- 确定喜神忌神

### Phase 5: 查岁运

**目标**: 分析大运流年

**执行动作**:
- 以喜忌为标准
- 推断运势起伏

### Phase 6: 推六亲

**目标**: 分析六亲关系

**执行动作**:
- 以四柱六神为标准
- 推断父母妻子兄弟

### Phase 7: 评性情

**目标**: 分析性格特点

**执行动作**:
- 以五行用神为标准
- 推断性格倾向

### Phase 8: 断事业

**目标**: 分析事业方向

**执行动作**:
- 以用神及喜忌为标准
- 推断适合行业
