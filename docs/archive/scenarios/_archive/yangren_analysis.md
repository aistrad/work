# 阳刃格局分析

**触发词**: 阳刃, 阳刃格, 日刃, 刃杀, 羊刃

**来源**: 渊海子平.converted.md:论阳刃:001

## 分析流程

### Phase 1: 确认阳刃位置

- 甲禄在寅，卯为阳刃
- 丙戊禄在巳，午为阳刃
- 庚禄在申，酉为阳刃
- 壬禄在亥，子为阳刃
- 五阳有刃，五阴无刃

### Phase 2: 分析刃杀配置

- 阳刃喜偏官七杀
- 阳刃喜印绶
- 杀无刃不显，刃无杀不威
- 刃杀俱全，非常人有之

### Phase 3: 判断忌神

- 忌反吟伏吟
- 忌魁罡
- 忌三合会刃
- 忌刑冲破害
- 忌财旺居三合

### Phase 4: 行运吉凶

- 身旺运行身旺之乡为贵
- 不要见伤官、刃旺运
- 岁运逢杀旺之乡，乃转生而反成厚福
- 岁运又逢刃杀，其祸非常
