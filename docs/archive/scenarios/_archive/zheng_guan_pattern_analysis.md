# 正官格局分析

**触发词**: 正官格, 官星分析, 正官取用, 官格吉凶

**来源**: 三命通会.converted.md:卷五·论正官:001

## 分析流程

### Phase 1: 确定正官位置

- 先看月令是否有正官(月令当时为最)
- 甲日生酉月、乙生申巳、丙生子、丁生亥、戊生卯、己生寅、庚生午、辛生寅巳、壬生午未丑、癸生辰巳戌月为正气官
- 若月令无官,看年日���支干是否有官星

### Phase 2: 判断官星纯一

- 正官止许一位,多则不宜
- 检查是否有七煞混杂(如甲见庚为混杂)
- 检查官星是否天干透出(支藏干透为佳)

### Phase 3: 检查破格因素

- 刑冲破害:如甲生酉月,见卯为冲、酉为刑、午为破、戌为害
- 伤官克官:如甲见丁为伤
- 贪合忘官:如甲见丙合辛
- 劫财分福:如甲见乙为劫

### Phase 4: 分析日主强弱配合

- 日主健旺得财印两扶为佳
- 身弱印轻要补其印
- 身旺官轻要补其官
- 官星满目日主衰弱不能负荷为凶

### Phase 5: 判断行运喜忌

- 正官格要行印乡(逢官看印)
- 行伤官运为背禄(凶)
- 行身旺运为逐马
- 运至财官旺地为发达之时
