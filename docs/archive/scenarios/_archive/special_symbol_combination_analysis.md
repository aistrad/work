# 特殊吉凶符号组合分析

**触发词**: 天地相冲, 天地相合, 天克地冲, 天生地合, 天比地冲, 地比天冲, 干支重叠, 三冲一, 二冲一, 三合一, 局运并临, 岁运并临, 人生转折

**来源**: 东方代码启示录.converted.md:第25章:特殊符号组合

## 分析流程

### Phase 1: 识别特殊组合

- 检查原局中是否存在天地相冲（天干相冲+地支相冲）
- 检查原局中是否存在天地相合（天干相合+地支相合）
- 检查天克地冲（天干相克+地支相冲）
- 检查天生地合（天干相生+地支相合）
- 检查天比地冲/地比天冲
- 检查干支重叠（逢二/逢三/逢四）
- 检查三冲一/二冲一/三合一/二合一
- 检查局运并临/岁运并临

### Phase 2: 定位组合位置

- 确定组合涉及的柱位
- 分析组合对应的时段/地域/人脉
- 判断组合是原局内部还是原局与时运构成

### Phase 3: 分析组合影响

- 天地相冲：天翻地覆、连根拔起、人生转折点
- 天地相合：亲和力最大、顺利如意、自我修复能力强
- 天克地冲：利弊程度高于一般生克
- 天生地合：利好程度高于一般生合
- 比冲组合：针对比元素的冲强化，偏负面
- 干支重叠：逢二为利好扶持，逢三为周折，逢四为偏枯
- 局运并临：元素强化剂
- 岁运并临：动与动的共振能量

### Phase 4: 给出化解建议

- 天地相冲年份：顺应纹路、主动应变、提前准备
- 岁运并临：少涉及陌生事项、以静制动、注意保养
- 识别颠覆四柱用于择日避凶
