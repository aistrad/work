# 刑冲会合解法分析

## 元数据

- **ID**: xingchong_huihe_jiefa
- **名称**: 刑冲会合解法分析
- **级别**: standard
- **计费**: basic
- **描述**: 分析八字地支的刑冲会合关系，判断会合能否解刑冲，以及刑冲能否解会合，评估对格局的影响

## 触发条件

**主要触发词**: 刑冲会合, 地支冲, 地支合, 三合, 六合, 三刑, 六冲

**次要触发词**: 解冲, 解刑, 冲破, 合化, 会局

## 服务流程 (SOP)

### Phase 1: 识别地支关系

**目标**: 找出四柱地支的刑冲会合

**执行动作**:
- 识别六冲（子午、丑未、寅申、卯酉、辰戌、巳亥）
- 识别三刑（子卯刑、寅巳申刑、丑戌未刑、辰午酉亥自刑）
- 识别六合（子丑、寅亥、卯戌、辰酉、巳申、午未）
- 识别三合（申子辰、亥卯未、寅午戌、巳酉丑）
- 识别三会（寅卯辰、巳午未、申酉戌、亥子丑）

### Phase 2: 分析会合解刑冲

**目标**: 判断会合能否解刑冲

**执行动作**:
- 检查冲支是否有合神
- 检查刑支是否有会局
- 判断合会是否有效解除刑冲

### Phase 3: 分析刑冲解会合

**目标**: 判断刑冲能否破坏会合

**执行动作**:
- 检查会合之支是否被冲
- 判断冲力是否足以破局
- 分析因解反得刑冲的情况

### Phase 4: 评估格局影响

**目标**: 判断刑冲会合对用神的影响

**执行动作**:
- 判断用神是否被冲破
- 判断忌神是否被冲去
- 评估整体格局变化
