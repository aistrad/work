# 富贵贫贱分析

## 元数据

- **ID**: bazi_wealth_status_analysis
- **名称**: 富贵贫贱分析
- **级别**: standard
- **计费**: basic
- **描述**: 分析八字中的富贵贫贱格局，判断命主的财富与社会地位

## 触发条件

**主要触发词**: 能不能发财, 有没有富贵命, 财运怎么样, 能当官吗

**次要触发词**: 命好不好, 有钱命, 贵人运, 社会地位

## 服务流程 (SOP)

### Phase 1: 分析富的条件

**目标**: 判断是否具备富命格局

**执行动作**:
- 财星生官官星卫财
- 身旺财旺有伤食或官杀
- 财为用神不遭克破

### Phase 2: 分析贵的条件

**目标**: 判断是否具备贵命格局

**执行动作**:
- 官印相生
- 杀印相生
- 食神制杀
- 正官清粹

### Phase 3: 分析贫的条件

**目标**: 判断是否有贫命特征

**执行动作**:
- 财轻官重
- 财重劫轻身弱
- 喜财而财神被合
- 财为忌神

### Phase 4: 综合判断

**目标**: 给出富贵贫贱的综合评价

**执行动作**:
- 区分富而贵或贫而贵
- 分析先富后贫或先贫后富
- 结合大运流年判断时机
