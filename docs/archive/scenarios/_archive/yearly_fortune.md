# 流年运势

## 元数据

- **ID**: yearly_fortune
- **名称**: 流年运势
- **级别**: standard
- **计费**: basic
- **描述**: 深入分析特定年份的运势走向，包括事业、财运、感情等方面

## 触发条件

**主要触发词**: 今年运势, 明年运势, 流年, 2026年运势

**次要触发词**: 运势, 今年怎么样, 明年如何

## 前置要求

- 用户出生信息（必须）
- 已计算命盘数据（必须）

## 输出配置

**使用工具**:
- show_bazi_fortune
- show_bazi_kline
- show_insight

**交付物**: 流年运势分析 + 运势趋势图

**预估轮次**: 3-5 轮

## 服务流程 (SOP)

### Phase 1: 信息收集

**类型**: required

**描述**: 确认用户出生信息和目标年份

**检查点**:
- 用户是否已有 birth_info？
- 用户想分析哪一年？（默认当前年份）

### Phase 2: 计算排盘

**类型**: required

**描述**: 计算命盘和流年数据

**工具调用**:
```
calculate_bazi:
  include_dayun: true
  target_year: {用户指定年份}
```

### Phase 3: 快速扫描

**类型**: required

**描述**: 快速识别流年关键信号

**扫描内容**:
1. 流年干支与命局的关系
2. 流年与大运的配合
3. 红旗信号检查

**关键信号**:
| 信号 | 含义 |
|-----|------|
| 流年冲太岁 | 变动年，需谨慎 |
| 流年合用神 | 贵人年，机遇多 |
| 流年克用神 | 阻力年，宜守 |
| 流年生用神 | 顺遂年，可进 |

### Phase 4: 主动追问

**类型**: optional

**描述**: 了解用户当前状况以提供更精准分析

**追问示例**:
- "你今年有什么重要计划吗？"
- "目前工作/感情状况如何？"

### Phase 5: 深度分析

**类型**: required

**描述**: 分领域深入分析流年运势

#### 华山八字流年小运四步骤（源自《东方代码启示录》）

**Step 1: 流年小运互动（最主导）**
看流年天干与小运天干是否构成十神格局：

| 格局 | 流年寓意 |
|-----|---------|
| 正财生官 | 事业顺畅、稳步发展 |
| 杀印相生 | 功名机遇、贵人相助 |
| 伤官配印 | 专业突破、名利双收 |
| 食神制杀 | 化险为夷、事业利好 |
| 偏财生杀 | 投机机遇、需防风险 |
| 枭神夺食 | 多劳少获、防是非 |
| 伤官见官 | 事业波折、防口舌 |

**Step 2: 流年插入原局**
将流年干支插入八字原局，看是否构成新的十神组合：
- 天干十神组合 → 主导定性
- 地支三合/三会/三刑 → 辅助信息

**Step 3: 冲合建影响**
流年干支对原局各元素的影响：
- **冲**: 该元素代表的人事物有变动
- **合**: 该元素代表的人事物有牵绊或机遇
- **建**: 该元素特征在本年突出

**Step 4: 本身特征解读**
- 流年天干正偏类定性
- 流年地支星煞特征（贵人、禄神、桃花、羊刃等）

#### 流年与大运配合原则

| 大运定性 | 流年定性 | 综合结果 |
|---------|---------|---------|
| 利好 | 利好 | 利好程度最大化 |
| 利好 | 利差 | 利差被大运冲淡 |
| 利差 | 利好 | 利好被大运约束 |
| 利差 | 利差 | 利差程度最大化 |

#### 传统分析维度

| 领域 | 分析重点 |
|-----|---------|
| 事业 | 官杀与流年的关系 |
| 财运 | 财星与流年的关系 |
| 感情 | 桃花星与流年的关系 |
| 健康 | 五行平衡与流年的影响 |

**知识检索**:
- 检索流年相关规则
- 匹配类似案例
