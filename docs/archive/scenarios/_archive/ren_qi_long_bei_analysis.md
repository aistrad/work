# 壬骑龙背格分析

**触发词**: 壬骑龙背, 壬辰日, 暗冲格局

**来源**: 三命通会.converted.md:卷六·壬骑龙背:001

## 分析流程

### Phase 1: 确认格局成立条件

- 壬日坐辰(壬辰日)
- 柱中辰多方能冲起
- 辰暗冲戌中丁戊(财官)

### Phase 2: 检查配合条件

- 得一寅字合住财官为妙
- 三辰一寅为冲合贵气有力
- 不宜财官显露

### Phase 3: 检查破格因素

- 忌见己官、戊煞、乙伤、丁合
- 忌北方亥子运
- 忌申子全见(当以润下格论)

### Phase 4: 判断行运喜忌

- 喜行身旺及伤官食神运
- 忌南方财官之地
- 宜见丑未为贵
