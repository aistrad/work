# 伤官见官深度分析

## 元数据

- **ID**: shangguan_jianguan_analysis
- **名称**: 伤官见官深度分析
- **级别**: professional
- **计费**: premium
- **描述**: 基于《滴天髓》预测论，深度分析伤官见官的吉凶及化解

## 触发条件

**主要触发词**: 伤官见官, 伤官克官, 官运不好, 与领导冲突

**次要触发词**: 为祸百端, 仕途受阻, 女命婚姻, 印制伤官

## 服务流程 (SOP)

### Phase 1: 识别伤官见官

**目标**: 确认命局是否有伤官见官

**执行动作**:
- 找出伤官位置
- 找出正官位置
- 判断是否同见

### Phase 2: 凶吉判断

**目标**: 判断伤官见官的实际影响

**执行动作**:
- 正官格忌伤官为凶
- 伤官格见官需看配置
- 检查是否有印制伤官

### Phase 3: 化解建议

**目标**: 给出化解或应对建议

**执行动作**:
- 有印制伤官可解
- 伤官配印见官反吉
- 给出事业婚姻具体建议
