# 口诀适用性判断

## 元数据

- **ID**: koujue_applicability_check
- **名称**: 口诀适用性判断
- **级别**: professional
- **计费**: premium
- **描述**: 基于第十二章口诀适用边界，判断古籍口诀在具体命局中的适用性

## 触发条件

**主要触发词**: 口诀验证, 断语适用, 古法口诀, 命理口诀

**次要触发词**: 经得起验证, 需要限定, 现代修正, 适用边界

## 服务流程 (SOP)

### Phase 1: 口诀分类

**目标**: 判断口诀的验证等级

**执行动作**:
- 高度验证口诀可直接用
- 中度验证口诀需看条件
- 已被修正口诀需谨慎

### Phase 2: 情境限定检查

**目标**: 检查口诀的适用条件

**执行动作**:
- 伤官见官需看是否正官格
- 枭神夺食需看是否食神格
- 财多身弱需确认身确实弱
- 杀旺无制需确认无印化杀

### Phase 3: 现代修正应用

**目标**: 应用现代实践修正

**执行动作**:
- 女命伤官非必克夫
- 羊刃逢冲非必血光
- 华盖逢空非必出家
- 桃花带煞非必淫乱
