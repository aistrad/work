# 格局成败救应分析

## 元数据

- **ID**: geju_chengbai_jiuying
- **名称**: 格局成败救应分析
- **级别**: professional
- **计费**: premium
- **描述**: 分析八字格局的成败条件，识别带忌之神，寻找救应之法，判断成中有败或败中有成

## 触发条件

**主要触发词**: 格局成败, 格局分析, 破格, 成格, 救应

**次要触发词**: 带忌, 成中有败, 败中有成, 格局高低

## 服务流程 (SOP)

### Phase 1: 判断格局类型

**目标**: 确定月令格局

**执行动作**:
- 以月令定格
- 识别正官、财、印、食神、七煞、伤官、阳刃、建禄月劫等格

### Phase 2: 分析成格条件

**目标**: 检查格局是否成立

**执行动作**:
- 官格：逢财印，无刑冲破害
- 财格：生官旺，或逢食生身强带比
- 印格：轻逢煞，或官印双全
- 食格：生财，或带煞无财弃食就煞
- 煞格：身强逢制
- 伤官格：生财，或佩印伤旺印有根
- 刃格：透官煞露财印，不见伤官
- 建禄月劫：透官逢财印，透财逢食伤

### Phase 3: 识别破格因素

**目标**: 找出败格之神

**执行动作**:
- 官格败：逢伤克刑冲
- 财格败：财轻比重，财透七煞
- 印格败：印轻逢财，身强印重透煞
- 食格败：逢枭，生财露煞
- 煞格败：逢财无制
- 伤官格败：非金水见官，生财带煞
- 刃格败：无官煞
- 建禄月劫败：无财官，透煞印

### Phase 4: 寻找救应之法

**目标**: 分析是否有救应

**执行动作**:
- 检查是否有去病之药
- 分析制忌护用之神
- 评估救应是否得力

### Phase 5: 综合判断

**目标**: 得出格局成败结论

**执行动作**:
- 判断成中有败还是败中有成
- 评估格局高低
- 预测行运吉凶方向
