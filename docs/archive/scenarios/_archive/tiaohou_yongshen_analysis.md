# 调候用神分析

## 元数据

- **ID**: tiaohou_yongshen_analysis
- **名称**: 调候用神分析
- **级别**: professional
- **计费**: premium
- **描述**: 基于《穷通宝鉴》调候表及现代修正，分析命局的调候需求

## 触发条件

**主要触发词**: 调候用神, 寒暖燥湿, 冬命需火, 夏命需水

**次要触发词**: 穷通宝鉴, 调候之神, 气候调和, 五行寒暖

## 服务流程 (SOP)

### Phase 1: 气候判断

**目标**: 判断命局的寒暖燥湿

**执行动作**:
- 确定出生月份
- 判断季节气候
- 评估五行寒暖燥湿

### Phase 2: 调候需求评估

**目标**: 判断是否需要调候

**执行动作**:
- 冬月金水日主需火调候
- 夏月火土日主需水调候
- 春秋温和月份调候需求低

### Phase 3: 现代修正应用

**目标**: 应用现代修正原则

**执行动作**:
- 格局优先再看调候
- 仅极端气候需优先调候
- 原局不足可待大运补充
