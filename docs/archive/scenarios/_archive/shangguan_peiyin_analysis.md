## 触发条件

**主要触发词**: 伤官配印, 才华有约束, 聪明内敛, 学术成就

**次要触发词**: 印星制伤官, 化解伤官之凶

## 服务流程 (SOP)

### Phase 1: 格局识别

**目标**: 判断是否成伤官配印格

**执行动作**:
- 检查伤官与印星是否同现
- 判断印星是否有力
- 评估伤官旺度

### Phase 2: 成格条件

**目标**: 判断格局是否成立

**执行动作**:
- 印星有力为要
- 伤官不太旺为宜
- 相配有情

### Phase 3: 发展建议

**目标**: 给出发展建议

**执行动作**:
- 学术研究方向
- 文化教育发展
- 以学识立身
