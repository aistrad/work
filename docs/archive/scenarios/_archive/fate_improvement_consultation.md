# 命运化解咨询

## 触发词
- 改善命运
- 化解
- 改运
- 趋吉避凶
- 如何改命
- 命不好怎么办

## SOP 流程

### Phase 1: 审时度势
- 类型: required
- 工具: calculate_bazi, analyze_daymaster, analyze_current_luck
- 说明: 分析命主八字特征与当前时运，了解惯性纹理

### Phase 2: 识别问题
- 类型: required
- 工具: analyze_ten_gods, identify_unfavorable_patterns
- 说明: 识别不利的十神组合或时运特征

### Phase 3: 选择化解思路
- 类型: required
- 说明: 根据问题性质选择避、应、制、化四种思路之一

### Phase 4: 制定化解方案
- 类型: required
- 工具: generate_remedy_plan
- 说明: 从十种化解模式中选择适合的方法

### Phase 5: 通用趋吉建议
- 类型: optional
- 说明: 补充通用改善方法
