# 女命专项分析

## 元数据

- **ID**: bazi_female_destiny_analysis

## 触发条件

**主要触发词**: 女命分析, 女性八字, 看看我的婚姻, 女人命运

**次要触发词**: 嫁得好吗, 老公怎么样, 能有孩子吗, 女命看夫

## 服务流程 (SOP)

### Phase 1: 判断日主强弱

**目标**: 确定女命日主的强弱状态

**执行动作**:
- 分析日干得令失令
- 统计帮扶与克泄
- 判断身强身弱

### Phase 2: 分析夫星

**目标**: 评估官杀（夫星）的状态

**执行动作**:
- 查看官杀的旺衰
- 分析官杀是否受伤
- 判断夫星吉凶

### Phase 3: 分析子星

**目标**: 评估食伤（子星）的状态

**执行动作**:
- 查看食伤的旺衰
- 分析食伤是否受制
- 判断子女缘分

### Phase 4: 取用神

**目标**: 根据27条女命取用法则确定用神

**执行动作**:
- 兼顾夫子自身三者
- 优先保全夫星
- 其次保全子星

### Phase 5: 综合论断

**目标**: 给出女命整体评价

**执行动作**:
- 判断夫子两全或偏废
- 分析婚姻质量
- 预测子女情况
