# 杀旺无制分析

## 元数据

- **ID**: shawang_wuzhi_analysis
- **名称**: 杀旺无制分析
- **级别**: standard
- **计费**: basic
- **描述**: 基于《滴天髓》预测论，分析七杀旺而无制的命局特征及现代修正

## 触发条件

**主要触发词**: 七杀旺, 杀旺无制, 压力大, 事业艰难

**次要触发词**: 腰驼背曲, 灾祸频发, 制杀, 化杀

## 服务流程 (SOP)

### Phase 1: 七杀状态判断

**目标**: 判断七杀的旺衰及制化

**执行动作**:
- 评估七杀力量
- 检查是否有食神制杀
- 检查是否有印绑化杀

### Phase 2: 影响分析

**目标**: 分析杀旺无制的影响

**执行动作**:
- 古法主残疾劳苦
- 现代主压力大事业艰难
- 灾祸风险评估

### Phase 3: 化解建议

**目标**: 给出应对建议

**执行动作**:
- 寻找制杀之运
- 印绑化杀为解
- 事业选择建议
