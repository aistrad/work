# 丑遥巳禄格分析

**触发词**: 丑遥巳禄, 辛丑日, 癸丑日, 遥合格局

**来源**: 三命通会.converted.md:卷六·丑遥巳禄:001

## 分析流程

### Phase 1: 确认格局成立条件

- 只有辛丑、癸丑二日
- 辛以丙为官，癸以戊为官，丙戊禄在巳
- 丑能破巳，柱中多逢丑地则丙戊之禄出

### Phase 2: 检查配合条件

- 辛丑日宜生秋月
- 癸丑日宜生冬月
- 柱中金水多方合此局
- 再见申酉合住巳字不致贵气走出为妙

### Phase 3: 检查破格因素

- 忌见子绊
- 忌未冲
- 忌巳字填实
- 辛日忌丙丁巳午，癸日忌戊己巳午

### Phase 4: 判断格局纯粹度

- 无丙丁巳午辛日之纯粹
- 无戊己巳午癸日之纯粹
- 无冲绊为人淳厚富贵双全
- 略见损伤亦主富足
