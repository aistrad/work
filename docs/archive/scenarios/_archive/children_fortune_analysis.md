## 分析流程

### Phase 1: 识别食伤状态

- 确定日主五行属性
- 找出命局中的食神和伤官
- 判断食伤是否当令
- 分析食伤的旺衰强弱

### Phase 2: 分析印绶与食伤关系

- 检查是否有印绶克食伤
- 判断印星是否不现（子多无损）
- 分析财星是否坏印（身衰印作儿）
- 判断用神是否即是子星

### Phase 3: 判断子女吉凶

- 日主旺无印绶有食伤：子必多
- 日主旺印绶重食伤轻：子必少
- 印星不现辛金明润：多子无损
- 财星坏印：多女无子
- 火旺金弱：子多损伤

### Phase 4: 给出建议

- 根据食伤状态给出生育建议
- 指出有利的生育时机
- 提示子女运势的流年大运
