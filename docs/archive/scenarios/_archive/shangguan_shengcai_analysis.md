# 伤官生财格局分析

## 元数据

- **ID**: shangguan_shengcai_analysis
- **名称**: 伤官生财格局分析
- **级别**: standard
- **计费**: basic
- **描述**: 分析命盘中伤官生财的吉格，判断以才华创业、从事技术创意咨询行业、财源偏财性质等特质

## 触发条件

**主要触发词**: 伤官生财, 以才华创业, 技术创业, 创意行业

**次要触发词**: 咨询行业, 偏财性质, 创新创业

## 服务流程 (SOP)

### Phase 1: 格局识别

**目标**: 判断是否成伤官生财格

**执行动作**:
- 检查伤官与财星是否相生
- 判断伤官力量
- 评估财星旺度

### Phase 2: 行业匹配

**目标**: 匹配适合的行业

**执行动作**:
- 技术创新行业
- 创意设计行业
- 咨询顾问行业
- 自媒体行业

### Phase 3: 创业建议

**目标**: 给出创业发展建议

**执行动作**:
- 大运流年配合
- 创业时机选择
- 给出发展建议
