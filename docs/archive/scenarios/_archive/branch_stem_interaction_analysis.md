# 干支沟通途径分析

**触发词**: 干支关系, 地支影响, 稳定性分析, 持久性, 根基分析

**来源**: 东方代码启示录.converted.md:第十章-干支的沟通途径:001

## 分析流程

### Phase 1: 干支关系类型识别

- 逐柱分析干支关系类型
- 识别支生干类型(如甲子、癸酉、丙寅等)
- 识别支克干类型(如丁亥、戊寅、庚午等)
- 识别承载型干支(非生克关系)

### Phase 2: 地支链状态分析

- 检查地支之间的相冲关系
- 检查地支之间的相合关系
- 检查地支同材(完全相同的地支)
- 识别三合局(巳酉丑、寅午戌、申子辰、亥卯未)
- 识别三会局(寅卯辰、巳午未、申酉戌、亥子丑)

### Phase 3: 地支强弱定性

- 逢相合、逢同材则地支强势稳固
- 逢相冲则地支动摇弱势
- 三合局聚力使目标地支强势
- 三会局分布使相关地支强势

### Phase 4: 天干深入程度评估

- 支生干:天干更加持久稳定顺畅
- 支克干:天干波折无法深入压力大
- 承载地支强:天干根基稳固发挥持久
- 承载地支弱:天干反复不定难以持久

### Phase 5: 综合影响解读

- 结合天干十神定性与地支影响
- 评估各十神的实际发挥程度
- 识别有始无终或持久稳固的信号
- 提出针对性的发展建议
