# 小儿命分析

**触发词**: 小儿命, 小儿, 儿童命, 幼年, 关煞

**来源**: 渊海子平.converted.md:论小儿:001

## 分析流程

### Phase 1: 以时辰为主

- 小儿之命，当论时辰为主
- 先看关煞，次看格局

### Phase 2: 判断养育难易

- 日主强，财官旺，有关无煞
- 日干弱，财官少，常病可养
- 日干弱，财官多，有关有煞，又有三合聚煞者难养

### Phase 3: 分析出身

- 八字有财官，生于富贵之家
- 偏官生于平常之家
- 伤官劫财，生于贫贱之家
- 偏官偏印偏财，主偏生庶出

### Phase 4: 关煞应期

- 偏官为关，偏财为煞
- 取生辰之数断之：水一、火二、木三、金四、土五
- 如甲日庚杀，乃四、九岁见之
