# 杀印相生格局分析

## 元数据

- **ID**: shayin_xiangsheng_analysis
- **名称**: 杀印相生格局分析
- **级别**: standard
- **计费**: basic
- **描述**: 分析命盘中杀印相生的吉格，判断化压力为动力、逆境出贵人、主武职或技术权威等特质

## 触发条件

**主要触发词**: 杀印相生, 化压力为动力, 逆境出贵人, 技术权威

**次要触发词**: 武职, 印星化杀, 化杀为权

## 服务流程 (SOP)

### Phase 1: 格局识别

**目标**: 判断是否成杀印相生格

**执行动作**:
- 检查七杀与印星是否相生
- 判断印星化杀能力
- 评估格局成立条件

### Phase 2: 特质分析

**目标**: 分析格局带来的特质

**执行动作**:
- 抗压能力强
- 逆境中成长
- 有贵人相助

### Phase 3: 发展建议

**目标**: 给出发展建议

**执行动作**:
- 武职或技术方向
- 把握逆境机会
- 善用贵人资源
