# 十神过旺过弱诊断

## 元数据

- **ID**: shishen_balance_diagnosis
- **名称**: 十神过旺过弱诊断
- **级别**: standard
- **计费**: basic
- **描述**: 诊断命盘中十神的过旺过弱状态，分析其带来的正面和负面影响，如比肩过旺好胜固执、食神过旺贪图享乐、正印过旺依赖优柔等

## 触发条件

**主要触发词**: 十神过旺, 十神过弱, 命局失衡, 十神诊断

**次要触发词**: 比肩过旺, 食神过旺, 正印过旺, 七杀过旺

## 服务流程 (SOP)

### Phase 1: 十神旺衰判断

**目标**: 判断各十神的旺衰状态

**执行动作**:
- 分析四柱十神分布
- 判断得令失令
- 评估旺衰程度

### Phase 2: 过旺过弱分析

**目标**: 分析过旺过弱的表现

**执行动作**:
- 比肩过旺-好胜固执难合作
- 食神过旺-贪图享乐精力外泄
- 正印过旺-依赖优柔缺乏行动
- 七杀过旺-压力过大灾祸频发

### Phase 3: 平衡建议

**目标**: 给出平衡调整建议

**执行动作**:
- 识别失衡十神
- 找到平衡之神
- 提供调整方向
