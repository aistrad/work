# 择日选时

## 元数据

- **ID**: timing
- **名称**: 择日选时
- **级别**: professional
- **计费**: premium
- **描述**: 根据命盘选择最佳日期和时辰

## 触发条件

**主要触发词**: 择日, 选日子, 什么时候好, 吉日

**次要触发词**: 开业日子, 结婚日子, 搬家日子

## 前置要求

- 用户出生信息（必须）
- 事项类型（必须）
- 时间范围（必须）

## 输出配置

**使用工具**:
- show_bazi_chart
- show_insight

**交付物**: 择日建议报告

**预估轮次**: 3-8 轮

## 服务流程 (SOP)

### Phase 1: 信息收集

**类型**: required

**描述**: 收集择日所需信息

**收集内容**:
| 信息 | 必填 | 说明 |
|-----|------|------|
| 出生信息 | ✅ | 年月日时 |
| 事项类型 | ✅ | 结婚/开业/搬家/签约等 |
| 时间范围 | ✅ | 希望在什么时间段内 |
| 其他参与者 | ⚪ | 如结婚需要对方信息 |

### Phase 2: 计算排盘

**类型**: required

**描述**: 计算命盘和流年流月数据

### Phase 3: 快速扫描

**类型**: required

**描述**: 识别用神和忌神

**扫描重点**:
1. 用神五行
2. 忌神五行
3. 当前流年流月特点

### Phase 4: 主动追问

**类型**: optional

**描述**: 确认具体需求

**追问示例**:
- "具体是什么事项？"
- "希望在什么时间范围内？"
- "有没有已经定下的日期？"

### Phase 5: 深度分析

**类型**: required

**分析框架**:

1. **事项五行属性**
   | 事项 | 五行 | 宜 | 忌 |
   |-----|------|---|---|
   | 结婚 | 火/木 | 桃花日、合日 | 冲日、破日 |
   | 开业 | 金/土 | 财日、贵人日 | 破财日 |
   | 搬家 | 土 | 合日、生气日 | 冲日、煞日 |
   | 签约 | 金 | 成日、定日 | 破日、危日 |

2. **吉日筛选**
   - 与用神相生相合
   - 避开忌神旺相
   - 避开个人冲克日

3. **时辰选择**
   - 吉时配合
   - 避开凶时

### Phase 6: 输出交付

**类型**: required

**工具调用**:
1. `show_bazi_chart` - 展示命盘
2. `show_insight(type="timing")` - 择日洞察

**输出结构**:
```
【命盘参考】
{调用 show_bazi_chart}

【择日分析】
{调用 show_insight}

## 推荐日期

**首选**: {日期1} {时辰}
- 原因: {分析}

**次选**: {日期2} {时辰}
- 原因: {分析}

**备选**: {日期3} {时辰}
- 原因: {分析}

## 注意事项

- {注意点1}
- {注意点2}
```

### Phase 7: 答疑讨论

**类型**: reactive

**常见问题**:
| 问题 | 处理 |
|-----|------|
| "这个日子行不行" | 分析该日与命盘关系 |
| "时辰重要吗" | 解释时辰的作用 |
| "对方日子冲怎么办" | 给出化解建议 |
