# 五行比例分析流程

**触发词**: 五行比例, 五行缺什么, 五行分析, 适合什么行业, 适合什么方位, 性格分析

**来源**: 东方代码启示录.converted.md:第十五章:001

## 分析流程

### Phase 1: 计算五行比例

- 统计八字八个字的五行属性
- 不包括任何藏干
- 计算金木水火土各有几个

### Phase 2: 判断是否特殊格局

- 检查是否有五行占4个或以上且包含月令
- 若是且与日干同五行则为旺格
- 若是且与日干不同五行则为从格
- 特���格局需调整分析原则

### Phase 3: 定位利好利差五行

- 比例最小/缺的五行为优点与禀赋所在
- 比例最多的五行为缺点与劣势所在
- 次少次多也有相应利弊但幅度较小
- 注意：从格旺格中克格局的五行即使缺也不是利好

### Phase 4: 多维度应用分析

- 性情方面：缺少为优势，多为劣势
- 能力方面：缺少为强项，多为弱项
- 地域方面：缺少为利好方位，多为利差方位
- 行业方面：缺少为利好行业，多为利差行业
- 风水方面：缺少为需求，多为防弊
- 色泽方面：缺少为有利色，多为不利色

### Phase 5: 细化领域校验

- 检查是否与具体领域十神信息矛盾
- 如事业看正官/七杀的五行及根载
- 如财运看正财/偏财的五行及根载
- 矛盾时以十神连带关系为准

### Phase 6: 补救建议

- 倾向于五行缺少的方向补救
- 将十神与五行组合应用更到位
- 如缺木印则补木印行为（求学、出版等）
- 考虑人际互补：自己多的五行对缺此五行的人有利
