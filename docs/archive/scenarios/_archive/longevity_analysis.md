# 寿夭分析

**触发词**: 寿命, 长寿, 健康, 夭折, 寿元, 身体

**来源**: 滴天髓阐微.converted.md:何知章寿夭:scenario004

## 分析流程

### Phase 1: 判断性定元神厚

- 四柱得地，五行停匀
- 所合者皆闲神，所化者皆用神
- 冲去者皆忌神，留存者皆喜神
- 无缺无陷，不偏不枯

### Phase 2: 分析元神厚薄

- 官弱逢财，财轻遇食
- 身旺而食伤发秀
- 身弱而印绑当权
- 所喜者皆提纲之神
- 所忌者皆失令之物

### Phase 3: 识别气浊神枯

- 日主失令，用神浅薄
- 忌神深重，提纲与时支不照
- 身弱印绶太重
- 身旺克泄全无
- 金寒水冷土湿，火炎土燥木枯

### Phase 4: 给出结论

- 性定元神厚：主寿
- 清而纯粹：富贵而寿
- 浊而混杂：贫贱而寿
- 气浊神枯：主夭
