# 富贵贫贱分析

**触发词**: 富贵, 贫贱, 财运, 事业, 发财, 穷困

**来源**: 滴天髓阐微.converted.md:何知章:scenario003

## 分析流程

### Phase 1: 判断财气通门户

- 身旺财弱无官：要有食伤
- 身旺财旺无食伤：须有官杀
- 身旺印旺食伤轻：财星得局
- 身旺劫旺无财印：有食伤
- 身弱财重无官印：有比劫

### Phase 2: 判断官星有理会

- 身旺官弱：财能生官
- 官旺身弱：官能生印
- 印旺官衰：财能坏印
- 印衰官旺：财不现
- 劫重财轻：官能去劫

### Phase 3: 识别财神不真

- 财重食伤多：一不真
- 财轻喜食伤而印旺：二不真
- 财轻劫重食伤不现：三不真
- 财多喜劫官星制劫：四不真
- 喜印而财星坏印：五不真

### Phase 4: 给出结论

- 财气通门户：主富
- 官星有理会：主贵
- 财神不真：主贫
- 官星不见：主贱
