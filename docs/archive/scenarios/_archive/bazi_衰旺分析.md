# 八字衰旺分析流程

**触发词**: 衰旺, 日主强弱, 身强身弱, 旺衰判断, 得令失令

**来源**: 滴天髓阐微.converted.md:通神论:001

## 分析流程

### Phase 1: 月令司令判断

- 确定月令地支
- 判断人元司令（当月何干当令）
- 分析日主是否得令

### Phase 2: 四柱根气分析

- 检查日主在年月日时的根气
- 长生禄旺为重根
- 墓库余气为轻根
- 天干比肩不如地支余气墓库

### Phase 3: 旺衰变化判断

- 旺中变弱：得时但四柱财杀过旺
- 弱中变旺：失时但四柱有根有助
- 太旺似金：旺极宜泄不宜克
- 太衰似水：衰极宜从不宜扶

### Phase 4: 用神确定

- 旺则宜泄宜伤
- 衰则喜帮喜助
- 旺极者抑之反激宜从其强
- 衰极者扶之徒劳宜从其弱
