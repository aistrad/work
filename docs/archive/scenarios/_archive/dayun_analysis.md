# 大运推断分析

**触发词**: 大运, 运程, 行运, 运势, 十年运, 起运

**来源**: 三命通会.converted.md:卷二·论大运:003

## 分析流程

### Phase 1: 确定起运方向

- 判断命主性别
- 确定出生年干阴阳
- 阳男阴女顺行，阴男阳女逆行

### Phase 2: 计算起运时间

- 阳男阴女数至未来节��
- 阴男阳女数至过去节气
- 三日折一年计算起运岁数

### Phase 3: 排列大运干支

- 从月柱起运
- 顺行或逆行排列
- 每运十年

### Phase 4: 分析运程吉凶

- 运与用神关系：损用神者凶，益用神者吉
- 运与日主关系：身弱喜运引进旺乡
- 运与原局关系：官欲运生，煞欲运制，财欲运扶，印欲运旺

### Phase 5: 论断运程事象

- 长生运主创建作新
- 临官帝旺运主兴盛发福
- 衰病运主退败破财
- 死绝运主丧事衰祸
