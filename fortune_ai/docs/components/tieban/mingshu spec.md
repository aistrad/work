## 第二部分：考刻定分（Calibration & Verification）

**设计目的：** 铁板神数最关键的一步是“考刻”，通过反推父母生肖或兄弟数量来锁定唯一的“刻分”。

**1. 考时定刻 (Verification of Birth Time)**

* **命书格式示例：**
* **基准时刻：** [子时三刻]
* **考证条文（六亲验证）：**
* *（显示第一条验证码，命主确认是否符合）*
* **条文示例：** “3788 水年母终方合此刻” 或 “9003 父属鼠”
* **条文示例：** “12930 一女三夫数定” （若用于验证婚姻状况）


* **定局结论：** 经考证，命主生于 [X] 刻，确立为 [X] 局。

## 第三部分：先天六亲断语 (Innate Fate & Kinship)

**设计目的：** 展示铁板神数最引以为傲的六亲推断（父母、兄弟、夫妻、子息）。需使用文档中的条文作为填充示例。

**1. 父母宫 (Parents)**

* **推算依据：** 乾宫考父，坤宫考母。
* **条文模板：**
* 【父命】：“12948 耳顺之年，妻死泪涟涟” （推断父亲晚年丧偶）
* 【母命】：“12937 46 0 老年丧妻之悲”
* 【刑克】：“12931 64 0 妻死之年，人之不幸”



**2. 夫妻宫 (Marriage)**

* **推算依据：** 依据《八卦加则》与《太玄配数》推算配偶属性。
* **条文模板：**
* 【姻缘】： “12942 41 42 夜月移花影，春风报佳音” （预示婚期或佳运）
* 【配偶定数】：“12936 0 0 良人有刑，再嫁庚戌” （预示婚姻变动）
* 【不幸之歎】：“12939 41 0 夫死之年，女之不幸”



**3. 子息与兄弟 (Children & Siblings)**

* **条文模板：**
* 【子息数】： “12940 52 0 数有十二子命中注定”
* 【兄弟数】： （引用类似“兄弟三人，同父不同母”的条文）
