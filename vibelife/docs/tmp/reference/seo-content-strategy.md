### 2.2 核心关键词详细清单

#### Tier 1: FAT HEAD (核心词)

| 关键词 | 月搜索量 | KD(竞争度) | 搜索意图 | 优先级 | 目标页面 |
|--------|----------|-----------|----------|--------|----------|
| AI算命 | 2,000+ | 低 | 信息查询 | P0 | 首页 |
| 八字算命 | 50,000+ | 高 | 工具使用 | P0 | 工具页 |
| 在线八字 | 10,000+ | 中 | 工具使用 | P0 | 工具页 |
| 免费八字测算 | 20,000+ | 中 | 工具使用 | P0 | 工具页 |
| 星座运势 | 100,000+ | 极高 | 信息查询 | P1 | 星座频道 |

**策略**: FAT HEAD竞争激烈，通过"AI"差异化突围

#### Tier 2: MIDDLE BODY (中级词)

**类别A: 具体年份关键词**
```
"1990年八字" (5,000+ 搜索量, KD: 低)
"1991年八字" (4,500+ 搜索量, KD: 低)
"1992年八字" (4,000+ 搜索量, KD: 低)
... (覆盖1980-2000年)
```

**类别B: 日主关键词**
```
"丙火日主" (3,000+ 搜索量, KD: 低)
"甲木日主" (2,800+ 搜索量, KD: 低)
"癸水日主" (2,500+ 搜索量, KD: 低)
... (覆盖10个日主)
```

**类别C: 功能关键词**
```
"八字合婚免费" (15,000+ 搜索量, KD: 中)
"八字起名" (8,000+ 搜索量, KD: 中)
"八字排盘" (10,000+ 搜索量, KD: 中)
"八字命运" (6,000+ 搜索量, KD: 低)
```

**类别D: AI差异化关键词**
```
"AI八字分析" (1,000+ 搜索量, KD: 极低)
"人工智能算命" (800+ 搜索量, KD: 极低)
"AI算命准吗" (500+ 搜索量, KD: 极低)
"免费AI命理" (1,200+ 搜索量, KD: 极低)
```

**策略**: MIDDLE BODY是我们的核心战场，中等竞争+高转化

#### Tier 3: LONG TAIL (长尾词)

**自动生成模式**: `{年份}年{月份}八字命理`
```
"1990年5月八字命理" (500 搜索量, KD: 极低)
"1990年6月八字命理" (450 搜索量, KD: 极低)
"1990年7月八字命理" (400 搜索量, KD: 极低)
... (1980-2000年 × 12月 = 240个页面)
```

**自动生成模式**: `{日主1}与{日主2}契合度`
```
"丙火与甲木八字契合度" (200 搜索量, KD: 极低)
"丙火与癸水八字契合度" (180 搜索量, KD: 极低)
... (10×10=100个组合)
```

**自动生成模式**: `{具体出生日期}八字`
```
"1990年5月15日出生八字" (100 搜索量, KD: 极低)
"1991年3月8日出生八字" (90 搜索量, KD: 极低)
... (理论上无限，按需生成)
```

**策略**: LONG TAIL通过AI自动生成，规模化占领
