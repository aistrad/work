```
│  文档2: Core原型 + Inner原型 + Outer原型 = 3个原型（可能各不相同）         │

│                                                                             │
```

### 2.2 概念统一：Layer 1与Layer 2的关系

**关键澄清**：Layer 1的"主原型"就是Layer 2的"Core原型"

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│                    Layer 1 与 Layer 2 的统一关系                             │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│                                                                             │
│   Layer 1 (简化视图)              Layer 2 (完整视图)                        │
│   ═══════════════════            ═══════════════════                       │
│                                                                             │
│   主原型 ─────────────────────→  Core (灵魂原型)                           │
│   (The Creator)                   你的本能能量模式                          │
│                                   当没有外在压力时，你自然会怎样            │
│                                                                             │
│   副原型 ─────────────────────→  这是Core的"调味"                          │
│   (The Outlaw)                    让你的Creator带有叛逆色彩                 │
│                                   可能来自Inner或Outer的影响               │
│                                                                             │
│   (Layer 1不展示) ────────────→  Shadow (阴影原型)                         │
│                                   你否认的另一面                            │
│                                   Core的结构性对立                          │
│                                                                             │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │                                                                     │  │
│   │   简化逻辑:                                                         │  │
│   │                                                                     │  │
│   │   主原型 = Core原型                                                 │  │
│   │   副原型 = Inner或Outer中与Core不同的最强原型                      │  │
│   │   阴影原型 = Core原型的结构性对立                                  │  │
│   │                                                                     │  │
│   │   如果 Core = Inner = Outer (三层一致):                            │  │
│   │   → 主原型 = Core, 副原型 = 该原型的"互补原型"                     │  │
│   │                                                                     │  │
│   │   如果 Core ≠ Inner 或 Core ≠ Outer (存在张力):                    │  │
│   │   → 主原型 = Core, 副原型 = Inner或Outer中最强的非Core原型        │  │

│   └─────────────────────────────────────────────────────────────────────┘  │
│                                                                             │
│                                                                             │

│   ═══════                                                                  │

│   → Layer 2: 三层高度一致，阴影是Regular(被否认的归属需求)                │
│                                                                             │
│   用户B: Core=Creator, Inner=Sage, Outer=Caregiver, Shadow=Regular         │
│   → Layer 1: Creator × Sage (Inner的影响)                                 │
│   → Layer 2: 三层各不相同，阴影触及"不够特别"的自卑感                     │
│                                                                             │
```
