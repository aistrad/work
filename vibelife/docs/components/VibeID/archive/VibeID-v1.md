### 2.1 八字排盘 → 原型映射

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│                    八字排盘分析模块                                          │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  输入：出生年月日时 + 性别                                                   │

│                                                                             │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  Step 1: 排八字四柱                                                         │
│  ─────────────────────                                                     │
│                                                                             │
│  年柱 ─┬─ 年干 (天干)                                                       │
│        └─ 年支 (地支 + 藏干)                                                │
│  月柱 ─┬─ 月干                                                              │
│        └─ 月支 (含藏干，决定格局)                                           │
│  日柱 ─┬─ 日干 (日主，代表自己)                                             │
│        └─ 日支                                                              │
│  时柱 ─┬─ 时干                                                              │
│        └─ 时支                                                              │
│                                                                             │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  Step 2: 计算十神                                                           │
│  ─────────────────────                                                     │
│                                                                             │
│  以日干为"我"，其他干支与"我"的关系：                                       │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  同我者：                                                           │   │
│  │    比肩 (同性同元素) → 自我、独立、竞争                             │   │
│  │    劫财 (异性同元素) → 冲动、争夺、魄力                             │   │
│  │                                                                     │   │
│  │  我生者：                                                           │   │
│  │    食神 (同性所生) → 才华、享受、表达                               │   │
│  │    伤官 (异性所生) → 叛逆、创意、锋芒                               │   │
│  │                                                                     │   │
│  │  我克者：                                                           │   │
│  │    正财 (异性所克) → 务实、稳定、积累                               │   │
│  │    偏财 (同性所克) → 机会、社交、变通                               │   │
│  │                                                                     │   │
│  │  克我者：                                                           │   │
│  │    正官 (异性克我) → 规矩、责任、正统                               │   │
│  │    七杀 (同性克我) → 压力、野心、突破                               │   │
│  │                                                                     │   │
│  │  生我者：                                                           │   │
│  │    正印 (异性生我) → 学习、关爱、传统                               │   │
│  │    偏印 (同性生我) → 独特、思辨、孤独                               │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  Step 3: 定格局                                                             │
│  ─────────────────────                                                     │
│                                                                             │
│  格局判断优先级：                                                            │
│  1. 月令透干 (月支藏干透出)                                                 │
│  2. 月令本气                                                                │
│  3. 八字中最旺的十神                                                        │
│                                                                             │
│  格局分类：                                                                  │
│  • 正格：正官格、正印格、正财格、食神格                                     │
│  • 偏格：七杀格、偏印格、偏财格、伤官格                                     │
│  • 特格：比劫格、从格等                                                     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
