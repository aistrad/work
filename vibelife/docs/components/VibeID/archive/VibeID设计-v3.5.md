### 1.2 v3.5设计原则

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│                    v3.5 核心设计原则                                         │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│                                                                             │
│   原则1: 建模求深，行动求简                                                  │
│   ════════════════════════════                                             │
│                                                                             │
│   • 心灵结构可以复杂（加入阴影维度）                                        │
│   • 但每日行动建议必须清晰、可执行                                          │
│   • 用户不需要理解所有理论，只需要知道"今天可以做什么"                      │
│                                                                             │
│   ───────────────────────────────────────────────────────────────────────  │
│                                                                             │
│   原则2: 理解是起点，行动是核心                                              │
│   ════════════════════════════                                             │
│                                                                             │
│   • v3.0过于强调"你是什么"                                                 │
│   • v3.5强调"知道自己是什么之后，你可以做什么"                             │
│   • 每个洞见都应该指向具体的行动可能                                        │
│                                                                             │
│   ───────────────────────────────────────────────────────────────────────  │
│                                                                             │
│   原则3: 阴影不是敌人，是资源                                                │
│   ════════════════════════════                                             │
│                                                                             │
│   • 不把"阴影日"框定为"坏日子"                                             │
│   • 阴影原型的激活是成长的邀请                                              │
│   • 整合阴影需要勇气，而勇气是可以培养的                                    │
│                                                                             │
│   ───────────────────────────────────────────────────────────────────────  │
│                                                                             │
│   原则4: 个人成长必须包含社会维度                                            │
│   ════════════════════════════                                             │
│                                                                             │
│   • 孤立的自我认知是不完整的                                                │
│   • 每个原型都有其"利己"和"利他"的表达方式                                 │
│   • 健康的成长方向是增加社会兴趣，而非强化自我中心                          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
