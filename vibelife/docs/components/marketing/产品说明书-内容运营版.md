## 一、产品定位一句话

**VibeLife = 你的 AI 人生教练 + 命理顾问 + 心灵陪伴**

24/7 陪在你身边,深度理解你这个人,主动关心你的成长。

## 二、核心价值 (3 个核心差异化)

### 1. Deep Understanding - 深度理解你这个人

❌ **不是**解决单个问题 (像 ChatGPT 一样每次从零开始)
✅ **而是**理解你是谁,记住你的目标、困惑、模式

**举例**:
- 记住你的出生时间、星盘、八字命格
- 记住你的人生愿景:"3年后在海边工作室写作"
- 记住你的卡点:"总是拖延,不知道为什么"

### 2. Proactive Agency - 主动关心

❌ **不是**等你来问问题
✅ **而是**主动提醒、推送、引导

**举例**:
- 每天早上 8 点:"早安!今天是甲子日,水气旺,适合沟通和学习"
- 周日晚上:"这周完成了 4/5 个目标,来做个复盘吧?"
- 察觉到连续 3 天未签到:"最近还好吗?感觉你有点卡住了"

### 3. Infinite Extensibility - 无限扩展能力

❌ **不是**单一功能工具
✅ **而是**可以无限添加新能力的平台

**当前 Skills**:
- 八字命理 (Bazi)
- 西方占星 (Zodiac)
- 塔罗占卜 (Tarot)
- 人生教练 (Lifecoach)
- 正念导师 (Mindfulness)
- 职业规划 (Career)

**未来可加**:风水、MBTI、九型人格、心理咨询...

## 四、核心 Skills 介绍

### 1. Bazi (八字命理) 💰付费

**一句话**: 通过出生时间,理解你的天赋、性格、运势

**核心功能**:
- 命盘排盘 + AI 深度解读
- 今日运势 (每日推送)
- 流年分析 (今年适合做什么)
- 事业/财运/感情专项分析

**内容角度**:
- "八字不是迷信,是理解自己的工具"
- "为什么有人天生适合创业?"
- "今年你的流年运势是..."

### 2. Lifecoach (人生教练) 🆓免费

**一句话**: 帮你设定目标、突破卡点、持续成长

**核心功能**:
- Dan Koe 快速重置 (10分钟设定人生愿景)
- 目标层级管理 (年 → 月 → 周 → 日)
- 每日签到 + 周复盘
- 卡点突破对话

**核心方法论**: Dan Koe (一日重置) + Covey (7 Habits)

**内容角度**:
- "10分钟重置你的人生"
- "为什么你总是拖延?可能是身份问题"
- "如何把年度目标变成每日行动"

### 3. Zodiac (西方占星) 💰付费

**一句话**: 通过星盘,理解你的性格、关系、成长方向

**核心功能**:
- 本命盘解读
- 今日星象
- 行星逆行影响

**内容角度**:
- "不只是太阳星座,完整星盘才能真正理解你"
- "水逆不是玄学,是能量周期"

### 4. Core (生命对话者) 🆓始终激活

**一句话**: VibeLife 的灵魂,负责倾听、共情、引导

**核心能力**:
- 深度倾听 + 情绪感知
- 跨对话记忆 (记住你说过的每一句话)
- 苏格拉底式提问 (引导你自己找到答案)
- 适度引导 (不说教,不评判)

**人格特质**:
- 温暖但不粘腻
- 智慧但不说教
- 好奇但不窥探
- 诚实但不伤害

## 六、产品设计哲学

### Chat-First (对话优先)

```
传统产品             VibeLife
─────────            ────────

Dashboard            💬 Chat
├─ 功能入口1         (对话是唯一核心)
├─ 功能入口2               │
├─ 功能入口3         ┌─────┴─────┐
└─ 功能入口4         ▼           ▼
                    🪞 Me      🧭 Journey
用户: 不知道点哪个   (我是谁)   (我要去哪)

                    用户: 直接说话就行
```

### Me + Journey (双页面)

**Me 页面 (我是谁)** 🪞
- 聚合所有"了解自己"的内容
- 八字命盘 + 星座星图 + 性格画像
- 今日能量状态

**Journey 页面 (我要去哪)** 🧭
- 聚合所有"目标进度"的内容
- 人生愿景 → 月度项目 → 周大石头 → 今日行动
- 一个页面看清完整的目标层级

### 极简原则

❌ 不做:
- 复杂的菜单层级
- 分散的卡片横滑
- 过多的信息密度

✅ 只做:
- Chat: 想聊什么就说
- Me: 想了解自己就点镜子
- Journey: 想看目标就点罗盘

## 十二、常见问题 FAQ

### Q1: VibeLife 是算命 App 吗?

A: 不完全是。VibeLife 用八字、占星等工具帮你理解自己,但更重要的是持续陪伴你成长。
   就像心理咨询师会用 MBTI,但核心是对话和陪伴。

### Q2: 为什么要付费?免费 AI 不是很多吗?

A: 免费 AI (如 ChatGPT) 是通用工具,不记得你是谁。
   VibeLife 深度记住你的命盘、目标、困惑,主动关心你,这是完全不同的体验。

### Q3: 我不信命理,还适合用 VibeLife 吗?

A: 当然!Lifecoach (人生教练) 是完全免费的,基于现代心理学和教练方法论。
   命理只是可选的"理解自己"工具,不是必须的。

### Q4: 和真人教练比,有什么优势?

A: • 价格: ¥29/月 vs. 真人教练 ¥500+/小时
   • 可用性: 24/7 随时对话 vs. 预约制
   • 记忆: 完美记住所有对话 vs. 笔记回顾
   • 劣势: 缺少真人的温度和直觉
