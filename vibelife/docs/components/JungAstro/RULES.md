# 心理画像分析规则

## 适用场景

用户想要了解自己的深层心理结构、性格特质、心理功能分布等。

典型问题：
- "帮我分析一下我的心理"
- "我的深层性格是什么样的？"
- "我的心理结构是怎样的？"
- "从心理学角度分析我"

## 分析框架

| 步骤 | 分析要点 | 心理意义 | 检索 Query |
|-----|---------|---------|-----------|
| 1 | 太阳星座+宫位 | 核心自我认同 (Ego) | "{sun_sign} 太阳 心理 核心自我" |
| 2 | 月亮星座+宫位 | 情感需求 (Unconscious) | "{moon_sign} 月亮 情感需求 无意识" |
| 3 | 上升星座 | 人格面具 (Persona) | "{rising_sign} 上升 人格面具 适应" |
| 4 | 水星配置 | 思维模式 (Thinking) | "{mercury_sign} 水星 思维模式" |
| 5 | 金星配置 | 价值观与关系 (Feeling) | "{venus_sign} 金星 价值观 关系" |
| 6 | 火星配置 | 驱动力与行动 (Will) | "{mars_sign} 火星 驱动力 行动" |
| 7 | 元素分布 | 心理能量类型 | "元素分布 {dominant} 心理功能" |
| 8 | 模式分布 | 行动风格倾向 | "模式分布 {dominant} 行动风格" |

## 输出结构

### 1. 核心自我 (Ego) - 太阳分析

```
☉ 核心自我
太阳 {sign} 第{house}宫

心理意义：
[太阳代表意识中心、核心身份认同、人生目的感]

你的表达：
[基于星座+宫位的具体解读]

整合状态：[健康表达]
阻塞状态：[可能的挑战]
```

### 2. 情感需求 (Unconscious) - 月亮分析

```

[月亮代表情感本能、无意识模式、安全需求]

```

[上升代表面对世界的方式、第一印象、适应策略]

你的表达：
[基于星座的具体解读]

```

### 4. 心理功能分布

```
元素分布：
火 ████░░░░ 40%  [直觉、热情]
土 ██░░░░░░ 20%  [感觉、实际]
风 ███░░░░░ 30%  [思维、沟通]
水 █░░░░░░░ 10%  [情感、共情]

主导功能：{dominant_element}
劣势功能：{inferior_element}

模式分布：
开创 ███░░░░░ 30%
固定 ████░░░░ 40%
变动 ███░░░░░ 30%
```

## 语言要求

1. **使用心理学语言**：
   - ✅ "你的核心自我认同倾向于..."
   - ❌ "你命中注定..."

2. **强调倾向而非定论**：
   - ✅ "这个配置显示一种倾向..."
   - ❌ "你就是这样的人..."

3. **成长导向**：
   - ✅ "这里有一个整合的机会..."
   - ❌ "这是你的弱点..."

## 引导方向

分析完成后，引导用户深入探索：
- "想了解你的阴影面吗？"（→ shadow-work）
- "想看看你当前的成长阶段吗？"（→ individuation）
- "想了解你的关系模式吗？"（→ relationship-dynamics）
```

# 阴影整合分析规则

## 适用场景

用户想要了解自己的阴影面、内在冲突、困难相位的意义。

典型问题：
- "我的阴影是什么？"
- "为什么我总是..."
- "我有什么内在冲突？"
- "这个困难相位是什么意思？"
- "冥王星对我有什么影响？"

## 阴影来源

在荣格心理占星中，阴影主要来自：

| 来源 | 配置 | 心理意义 |
|-----|------|---------|
| 困难相位 | 刑相 (□)、对冲 (☍) | 内在张力需要整合 |
| 冥王星 | 冥王星的星座、宫位、相位 | 被压抑或否认的部分 |
| 12宫 | 12宫内的行星 | 无意识的、隐藏的能量 |
| 土星 | 土星的配置 | 恐惧、限制、需要成熟的领域 |
| 南交点 | 南交点的星座、宫位 | 过去的模式、需要放下的 |

## 分析框架

| 步骤 | 分析要点 | 检索 Query |
|-----|---------|-----------|
| 1 | 识别困难相位 | "{planet1} {aspect} {planet2} 心理张力" |
| 2 | 分析冥王星配置 | "冥王星 {sign} {house}宫 阴影" |
| 3 | 分析12宫内容 | "12宫 {planets} 无意识" |
| 4 | 分析土星配置 | "土星 {sign} {house}宫 恐惧 成熟" |
| 5 | 综合阴影模式 | 整合以上分析 |
| 6 | 提供整合路径 | "阴影整合 {pattern} 方法" |
