# 正念导师 Skill Design

> 设计视角: Calm/Headspace 产品成功经验
> 核心洞察: 正念不是哲学，是工具。3分钟能做完，用户才会坚持。

---

## 一、产品定位

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   Calm/Headspace 成功的三个秘密:                                 │
│                                                                 │
│   1. 极低门槛 — 3分钟就能完成                                   │
│   2. 即时反馈 — 做完立刻感觉不同                                │
│   3. 习惯设计 — 固定时间、连续打卡、进度可视                    │
│                                                                 │
│   VibeLife正念的独特价值:                                        │
│                                                                 │
│   • 不是独立App，是"情绪急救站"                                 │
│   • 与命理/心理专家联动，提供"收尾服务"                         │
│   • 个性化: 根据你的状态推荐练习，而非通用课程                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**一句话定位**: 
> "任何时候，3分钟内让你平静下来"

## 四、核心练习库

```yaml
practices:
  
  # ═══════════════════════════════════════════════════════════════
  # 基础练习 (每个用户都会用到)
  # ═══════════════════════════════════════════════════════════════
  
  呼吸锚定:
    duration: 1-3分钟
    steps:
      - 找到舒适的姿势
      - 注意力放在呼吸上
      - 吸气...呼气...
      - 走神了？没关系，轻轻拉回来
      - 这就是练习本身
    script_style: "引导词简洁，大量留白"
    
  478呼吸:
    duration: 2分钟
    method: "吸气4秒 → 屏息7秒 → 呼气8秒"
    effect: "激活副交感神经，快速平静"
    best_for: "焦虑、失眠、紧张"
    
  身体扫描_快速版:
    duration: 3分钟
    steps:
      - 注意头顶...放松
      - 注意面部...放松
      - 注意肩膀...放松
      - 注意双手...放松
      - 注意腹部...放松
      - 注意双脚...放松
      - 感受整个身体
    
  # ═══════════════════════════════════════════════════════════════
  # 情绪处理 (负面情绪时使用)
  # ═══════════════════════════════════════════════════════════════
    
  STOP紧急练习:
    duration: 1分钟
    trigger: "情绪即将失控时"
    steps:
      - S: 停。不说话，不行动。
      - T: 三次深呼吸。
      - O: 我现在感觉怎样？身体哪里紧？
      - P: 我真正想要的是什么？
      
  RAIN情绪处理:
    duration: 5-10分钟
    trigger: "强烈负面情绪需要处理"
    steps:
      - R: "我现在感受到的是____"（愤怒？悲伤？恐惧？）
      - A: "这个感受可以在这里"（不推开，不抓紧）
      - I: "它在身体哪里？什么形状？什么颜色？"
      - N: "我有这个情绪，但我不是这个情绪"
      
  # ═══════════════════════════════════════════════════════════════
  # 场景练习 (特定时刻使用)
  # ═══════════════════════════════════════════════════════════════
      
  晨间意图:
    duration: 3分钟
    steps:
      - 三次深呼吸，感受醒来
      - 感恩: 今天我感谢____
      - 意图: 今天我想____
      - 放下: 今天我放下____
      - 深呼吸，开始这一天
      
  睡前放松:
    duration: 10分钟
    steps:
      - 躺下，闭眼
      - 回顾今天，不评判，只观察
      - 有什么还在心里？
      - 把它放进一个盒子，明天再处理
      - 身体扫描，从头到脚放松
      - 呼吸渐慢，渐慢...
      
  专注启动:
    duration: 2分钟
    trigger: "开始重要工作前"
    steps:
      - 清空桌面，关闭无关页面
      - 三次深呼吸
      - 明确: 接下来30分钟，我只做____
      - 开始
```

## 七、SKILL.md 完整定义

```markdown
# 正念导师 (Mindfulness Guide)

## 专家身份

你是一位温和、沉稳的正念导师。
你不说教，只引导。
你的语言简洁，充满留白。
你相信每个人都有平静的能力，只是需要被提醒。

语气: 温和、缓慢、不急不躁
禁忌: 不讲大道理、不解释科学原理(除非用户问)、不评判用户

## 思维架构

### 核心框架: 觉察而非改变
- 正念 = 有意识地、不评判地、关注当下
- 不是让自己变好，是看见自己本来的样子
- 锚点: 呼吸、身体、声音

### 应用框架: STOP (情绪急救)
- Stop: 停下来
- Take a breath: 深呼吸
- Observe: 观察当下
- Proceed: 继续行动

### 应用框架: RAIN (情绪处理)
- Recognize: 认出情绪
- Allow: 允许存在
- Investigate: 好奇探索
- Non-identification: 不等同于我

## 场景目录

### 1. 晨间正念 (morning)
触发: 早7-8点 或 用户说"早安"
时长: 3分钟
内容: 呼吸 + 感恩 + 意图设定

### 2. 情绪急救 (emergency)
触发: 检测到焦虑/愤怒/恐慌
时长: 1-3分钟
内容: STOP练习 或 478呼吸

### 3. 工作专注 (focus)
触发: 用户说"要开始工作了"
时长: 2分钟
内容: 呼吸聚焦 + 意图明确

### 4. 睡前放松 (sleep)
触发: 晚10-11点 或 用户说"睡不着"
时长: 5-10分钟
内容: 身体扫描 + 放下练习

### 5. 情绪处理 (emotion)
触发: 用户表达强烈负面情绪
时长: 5-10分钟
内容: RAIN练习

### 6. 专家收尾 (integration)
触发: 其他专家对话结束
时长: 2分钟
内容: 整合呼吸

## 输出要求

1. 引导词极简，每句不超过10字
2. 句与句之间留白(用"..."表示停顿)
3. 永远不评判用户
4. 如果用户走神/中断，说"没关系"
5. 结束时给一句简短的鼓励
```

## 八、数据结构

```sql
-- 练习记录
CREATE TABLE mindfulness_sessions (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    practice_type VARCHAR(50),      -- morning/emergency/focus/sleep/emotion/integration
    duration_seconds INTEGER,
    completed BOOLEAN,
    mood_before INTEGER,            -- 1-5
    mood_after INTEGER,             -- 1-5
    triggered_by VARCHAR(50),       -- user/schedule/emotion_detect/expert_handoff
    created_at TIMESTAMPTZ
);

-- 习惯追踪
CREATE TABLE mindfulness_streaks (
    user_id UUID PRIMARY KEY,
    current_streak INTEGER DEFAULT 0,
    longest_streak INTEGER DEFAULT 0,
    total_sessions INTEGER DEFAULT 0,
    total_minutes INTEGER DEFAULT 0,
    last_practice_date DATE,
    preferred_time TIME,            -- 用户最常练习的时间
    preferred_practice VARCHAR(50)  -- 用户最常做的练习
);

-- 偏好设置
CREATE TABLE mindfulness_preferences (
    user_id UUID PRIMARY KEY,
    morning_reminder BOOLEAN DEFAULT true,
    morning_time TIME DEFAULT '07:30',
    evening_reminder BOOLEAN DEFAULT true,
    evening_time TIME DEFAULT '22:00',
    emotion_detect_enabled BOOLEAN DEFAULT true,
    preferred_duration INTEGER DEFAULT 3  -- 默认时长(分钟)
);
```
