## 十、文案检查 (v1.3 新增)

### 10.1 文案理念验证

- [ ] **核心理念贯彻: 从"命定"到"选择"，从"认知"到"行动"**
- [ ] 避免"命运""算命""预测""占卜"等表达
- [ ] 避免"你是..."固定身份表达，使用"你当前呈现为..."
- [ ] 阴影描述为"未被开发的潜能"而非"缺陷"

### 10.2 推荐表达检查

- [ ] "你当前呈现为..." (而非 "你是...")
- [ ] "一种生活策略" (而非 "一种命运")
- [ ] "成长的机会" (而非 "需要改正的问题")
- [ ] "了解自己，是为了更好地选择" (而非 "了解命运")
- [ ] "阴影是未被开发的潜能" (而非 "阴影是缺陷")
- [ ] "四维不一致是丰富" (而非 "四维不一致是冲突")
- [ ] "需要勇气" (而非 "需要运气")
- [ ] "你可以选择" (而非 "你注定如此")

### 10.3 避免表达检查

- [ ] ✗ "你的命运是..." / "你天生就是..."
- [ ] ✗ "好日子" / "坏日子"
- [ ] ✗ "你的弱点是..." / "你的缺陷是..."
- [ ] ✗ "这个原型比那个原型好"
- [ ] ✗ "算命" / "预测" / "占卜"
- [ ] ✗ "100%准确"
- [ ] ✗ 过度承诺 ("彻底改变你的人生")
- [ ] ✗ 制造焦虑 ("不了解自己会...")
