## 📊 总结：体系展现策略

| 用户阶段 | 展现深度 | 核心目的 |
|---------|---------|---------|
| **首页（未登录）** | 体系全景 | 建立差异化认知，激发好奇 |
| **Onboarding P1** | 三体系暗示 | 植入"不只是算命"的心智 |
| **Onboarding P2** | 三重计算可视化 | 展示专业性，建立信任 |
| **Onboarding P3** | 来源解释 + 心理学解读 | 让用户理解"为什么准" |
| **Onboarding P4** | 体系深度功能预览 | 创造持续探索的期待 |
| **首页（已登录）** | 每日三体系融合报告 | 提供持续价值，增加留存 |
| **深度页面** | 完整八字/星盘/原型详解 | 满足深度用户需求 |
