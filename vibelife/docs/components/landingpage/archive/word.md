### 2.1 Hero 首屏

| 元素 | 原文案 | 优化文案 | 优化理由 |
|------|--------|----------|----------|
| 主标题 | 你是谁的12种可能 | **了解自己，是为了更勇敢地活** | 点明"认知→行动"的核心价值 |

## 三、全局文案语气指南

### 3.1 应该用的表达

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│                    推荐表达 ✓                                                │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  • "你当前呈现为..." (而非 "你是...")                                       │
│  • "一种生活策略" (而非 "一种命运")                                         │
│  • "成长的机会" (而非 "需要改正的问题")                                     │
│  • "了解自己，是为了更好地选择" (而非 "了解命运")                           │
│  • "阴影是未被开发的潜能" (而非 "阴影是缺陷")                               │
│  • "四维不一致是丰富" (而非 "四维不一致是冲突")                             │
│  • "需要勇气" (而非 "需要运气")                                             │
│  • "你可以选择" (而非 "你注定如此")                                         │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────────────────────┐
```

| 体系定位 | 不是算命，是成长伙伴 | 三种智慧，一个完整的你 |

---

这套文案体系的核心改变：
1. **从被动到主动**：用户是选择者，不是被定义者
2. **从认知到行动**：洞见是起点，勇气行动才是目的
3. **从标签到旅程**：这是一段持续的自我探索，不是一次性的测试
4. **阴影的正名**：阴影不是缺陷，是未被开发的潜能
