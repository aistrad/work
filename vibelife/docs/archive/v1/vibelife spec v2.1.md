Visual Design System: LUMINOUS PAPER

设计理念：在保留 VELLUM OS 克制美学的基础上，增加"生命感"和"个人化"

核心元素：
• 光纸背景 (Luminous Paper) — Vellum 底色 + 五行色 glow
• 呼吸光晕 (Breath Aura) — 根据用户五行属性的微弱呼吸动画
• 个人化印章 (Insight Seal) — 基于八字生成的唯一徽记
• Skill 形态语言 — 八字(方形柱)、星座(圆形连线)、MBTI(光谱条)、依恋(嵌套圆环)

品牌核心色：
• Ink: #1C1A17 (墨色 - 主文字、CTA)
• Antique Gold: #B88A44 (古金 - 强调、徽章、印章)
• Vellum: #FBF7F1 (纸底 - 全局背景)

字体系统：
• Display: Fraunces + Noto Serif SC (情感标题)
• UI: Bricolage Grotesque + Noto Sans SC (正文界面)

# 八字分析

## Overview

帮助用户通过八字看见自己——天生的性格特质、生命的节奏周期。
用温暖、现代的语言解释传统概念，强调命理是"了解自己"而非"预测命运"。

## Workflow

1. 收集信息
   - 必需：出生年月日（公历）
   - 可选：出生时辰（提供更准确分析）

2. 排盘分析
   - 使用 `scripts/calculator.py` 排出八字
   - 判断日主强弱和格局
   - 分析十神关系

3. 输出结果
   - 命盘可视化
   - 性格特点解读（温暖、不评判）
   - 运势周期分析
   - 具体建议

## Persona Guidelines

语言风格：
- 用"你"而非"您"或"用户"
- 说"我发现"而非"分析结果显示"
- 说"你天生..."而非"根据八字理论..."
- 温暖、像朋友在聊天，不是专家在讲课

示例对比：
❌ "根据八字分析，您的日主为甲木，生于子月，水旺木相..."
✓ "你是甲木人，像一棵向上生长的树。冬天出生的你，
   内心有一种'等待春天'的坚韧——现在可能感觉有些压抑，
   但你的能量正在积蓄。"

## References

- [八字理论](references/theory.md)
- [常见格局](references/patterns.md)
- [解读指南](references/interpretations.md)

## Proactive Triggers

运势节点提醒：
- 月运交接：每月提前 1-2 天提醒
- 流年交接：每年提前 3-5 天提醒
- 大运交接：提前 1 个月提醒

提醒模板见 `assets/reminders.json`

║   • 自然融入对话 — 不打断用户                                                ║
║   • 可展开可忽略 — 用户有控制权                                              ║

```typescript
interface InsightCard {
  id: string
  type: 'discovery' | 'pattern' | 'timing' | 'growth'

  // 内容
  title: string
  content: string
  evidence?: string[]  // 支撑证据

  // 元数据
  confidence: number
  skillSource: string  // 产生洞察的 Skill
  createdAt: Date

  // 向量嵌入 (支持跨 Skill 语义检索)
  embedding?: number[]  // 3072维向量

  // 交互
  actions: {
    primary: { label: string; action: string }
    secondary: { label: string; action: string }
  }

  // 状态
  status: 'pending' | 'shown' | 'expanded' | 'dismissed'
  userReaction?: 'positive' | 'negative' | 'neutral'
}

interface InsightEngine {
  // LLM 驱动的洞察判断
  // 不再使用硬编码规则，让 LLM 理解语境
  maybeGenerateInsight(
    userId: string,
    skillId: string,
    conversationId: string,
    portrait: string,        // 用户画像 (Hot Memory)
    conversation: Message[], // 本次对话
    recentInsights: InsightCard[]  // 最近洞察 (避免重复)
  ): Promise<InsightCard | null>

  checkCooldown(
    userId: string,
    cardType: string,
    topic?: string
  ): Promise<boolean>

  // 检查每日上限
  checkDailyLimit(userId: string): Promise<boolean>

  generateEmbedding(content: string): Promise<number[]>

// Reminder Service
class ReminderService {
  // Daily cron job
  async processUpcomingEvents() {
    const upcomingEvents = await this.getEventsInRange(
      new Date(),
      addDays(new Date(), 7)
    )
    
    for (const event of upcomingEvents) {
      const affectedUsers = await this.getAffectedUsers(event)
      
      for (const user of affectedUsers) {
        const shouldRemind = await this.checkUserSettings(user, event)
        if (!shouldRemind) continue
        
        const content = await this.generateContent(event, user)
        await this.scheduleNotification(user, content, event)
      }
    }
  }
  
  // Generate personalized content
  async generateContent(
    event: FortuneEvent, 
    user: UserProfile
  ): Promise<ReminderContent> {
    const template = this.getTemplate(event.type)
    
    // Calculate personal relevance
    const personalAnalysis = await this.calculatePersonalImpact(
      event,
      user.birthData
    )
    
    // Generate with LLM following Persona Guidelines
    const content = await this.llm.generate({
      template,
      event,
      personalAnalysis,
      userHistory: user.recentConversations,
      style: 'warm, friendly, not formal'
    })
    
    return content
  }
  
  // Get users who should receive this reminder
  async getAffectedUsers(event: FortuneEvent): Promise<UserProfile[]> {
    switch (event.type) {
      case 'bazi_month':
      case 'bazi_year':
        // All users with birth data
        return this.usersWithBirthData()
      
      case 'mercury_retrograde':
        // All zodiac users, prioritize affected signs
        return this.zodiacUsers()
      
      default:
        return []
    }
  }
}

export async function GET(request: Request) {
