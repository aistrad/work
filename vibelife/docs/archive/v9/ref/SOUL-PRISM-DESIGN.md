# Soul Prism 灵魂棱镜 - 统一设计文档

> **Version**: 1.0 | 2026-01-22
> **融合**: Soul Snapshot + Identity Prism → Soul Prism
> **基于**: VibeID 四层架构 + 12 原型系统

---

## 一、设计理念

### 1.1 为什么要合并？

| 原设计 | 问题 | 合并后价值 |
|--------|------|-----------|
| Soul Snapshot (首次激活) | 只展示单一原型，缺乏深度 | 首次即展示四维人格 |
| Identity Prism (日常仪表盘) | 与首次体验割裂 | 统一的「灵魂棱镜」隐喻 |

**核心洞察**：首次相遇和日常相处应该是同一个「镜子」的不同阶段，而不是两个不同的功能。

### 1.2 灵魂棱镜的隐喻

```
一束白光（你的生辰）
        ↓
    进入棱镜
        ↓
    ┌─────────────────────────────────────┐
    │           Soul Prism                │
    │                                     │
    │    ┌─────┐                         │
    │   /       \   Core (本质)           │
    │  /  Inner  \  Inner (内在)          │
    │ /   Outer   \ Outer (外在)          │
    │/    Shadow   \Shadow (阴影)         │
    │               │                     │
    └─────────────────────────────────────┘
        ↓
折射出你的灵魂光谱
    🔥 火象开拓者 (Core)
    🌙 情感守护者 (Inner)
    ⚡ 行动叛逆者 (Outer)
    💎 隐藏智者 (Shadow)
```

**一个人不是一个标签，而是四个维度交织的独特存在。**

## 二、VibeID 四层架构

### 2.1 四层定义

| 层次 | 英文 | 定义 | 数据来源 | 权重 |
|------|------|------|---------|------|
| **Core** | 本质 | 最深层的灵魂驱动力 | 八字 70% + 太阳星座 30% | 最重要 |
| **Inner** | 内在 | 处理情绪和私密体验 | 月亮星座 75% + 金星 25% | 亲密关系 |
| **Outer** | 外在 | 展示给世界的形象 | 上升星座 70% + 火星 30% | 社交面具 |
| **Shadow** | 阴影 | 被压抑/否认的潜能 | 对立原型 + 对话发现 | 成长方向 |

### 2.2 四层的体验价值

```
Core   → 「我看到你是...」     → 被看穿的震撼
Inner  → 「在亲近的人面前...」 → 被理解的安慰
Outer  → 「别人眼中的你...」   → 被承认的满足
Shadow → 「你内心深处...」     → 被接纳的释然
```

## 四、数据融合算法

### 4.1 Core 层融合（八字主导）

```
Core 原型 = 八字原型 × 70% + 太阳星座原型 × 30%
```

**八字 → 原型映射**（格局 × 用神）：

| 格局 | 用神为财 | 用神为官杀 | 用神为印 | 用神为食伤 | 用神为比劫 |
|------|---------|----------|---------|----------|----------|
| 正官格 | Ruler+Caregiver | Ruler+Hero | Sage+Ruler | Creator+Ruler | Regular+Ruler |
| 七杀格 | Hero+Explorer | Outlaw+Hero | Hero+Sage | Outlaw+Creator | Hero+Regular |
| 正印格 | Caregiver+Sage | Sage+Ruler | Sage+Innocent | Creator+Sage | Caregiver+Regular |
| 食神格 | Creator+Lover | Magician+Creator | Sage+Creator | Creator+Jester | Jester+Creator |
| 伤官格 | Creator+Outlaw | Outlaw+Hero | Magician+Creator | Creator+Outlaw | Outlaw+Explorer |

**太阳星座 → 原型映射**：

| 星座 | 主原型 | 次原型 |
|------|--------|--------|
| ♈ 白羊座 | Hero | Explorer |
| ♉ 金牛座 | Lover | Regular |
| ♊ 双子座 | Jester | Magician |
| ♋ 巨蟹座 | Caregiver | Innocent |
| ♌ 狮子座 | Ruler | Creator |
| ♍ 处女座 | Sage | Caregiver |
| ♎ 天秤座 | Lover | Ruler |
| ♏ 天蝎座 | Magician | Outlaw |
| ♐ 射手座 | Explorer | Sage |
| ♑ 摩羯座 | Ruler | Hero |
| ♒ 水瓶座 | Outlaw | Sage |
| ♓ 双鱼座 | Innocent | Magician |

### 4.2 一致性处理

| 情况 | 处理方式 |
|------|---------|
| 八字与星座指向同一原型 | +15% 置信度奖励 |
| 指向相近原型（同象限） | 中等置信度 |
| 指向对立原型 | 以八字为准，星座指向作为 Shadow 线索 |

## 六、情感设计要点

### 6.1 首次揭晓的三个「打击点」

| 打击点 | 设计 | 情感效果 |
|--------|------|---------|
| **第一击：原型揭晓** | 大图腾 + 昵称 + Slogan | 「哇，这就是我」 |
| **第二击：三个特质** | 优点 + 优点 + 隐秘恐惧 | 「你怎么知道？」 |
| **第三击：AI 开口** | 我看到你...但我也看到... | 「我被理解了」 |

### 6.2 四层的情感节奏

```
Core   →  震撼   → 「我被看穿了」
Inner  →  温暖   → 「原来我的内心是这样」
Outer  →  认可   → 「别人眼中的我」
Shadow →  释然   → 「原来这是可以接受的」
```

### 6.3 语言温度规范

| 避免 | 使用 |
|------|------|
| 「你的命盘显示...」 | 「我看到你...」 |
| 「分析结果如下...」 | 「你是...」 |
| 「你的阴影面是...」 | 「你内心深处，有一个...」 |
| 「建议你...」 | 「也许你可以试着...」 |

      zodiac: {
        sunSign: string;
        archetype: Archetype;
        weight: 0.30;
      };
    };
    consistencyBonus: boolean;      // 一致性奖励
  };

  // 内在原型（Inner）
  inner: {
    archetype: Archetype;
    confidence: number;
    sources: {
      moonSign: string;
      venusSign: string;
    };
    emotionalTraits: string[];      // 情感特质
    loveStyle: string;              // 爱情风格
  };

  // 外在原型（Outer）
  outer: {
    archetype: Archetype;
    confidence: number;
    sources: {
      risingSign: string;
      marsSign: string;
    };
    socialStyle: string;            // 社交风格
    firstImpression: string;        // 第一印象
  };

  // 阴影原型（Shadow）
  shadow: {
    archetype: Archetype;
    tension: string;                // "信任 vs 掌控"
    growthDirection: string;        // 成长方向
    discovered: boolean;            // 是否已通过对话发现
  };

  // 三个核心特质
  coreTraits: [string, string, string];  // [优点, 优点, 隐秘恐惧]

  // 生成的解读
  interpretation: {
    coreSummary: string;            // Core 层解读
    innerSummary: string;           // Inner 层解读
    outerSummary: string;           // Outer 层解读
    shadowSummary: string;          // Shadow 层解读
    aiFirstWords: string;           // AI 第一句话
  };

  // 计算元数据
  meta: {
    calculatedAt: string;
    version: string;
  };
}

interface Archetype {
  id: string;                       // "Hero"
  name_cn: string;                  // "英雄"
  nickname: string;                 // "搞事业的命"
  emoji: string;                    // "⚔️"
  slogan: string;                   // "要么出众，要么出局"
  coreDrive: string;
  coreFear: string;
  description: string;
  superpowers: string[];
  growthPoints: string[];
}
