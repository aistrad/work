# VibeLife 页面布局设计文档 v7.1

> **版本**: 7.1 | 2026-01-17
> **目标读者**: AI Agent / 前端开发
> **设计规范**: VibeLife-frontend-v7.md + VibeID设计-v5.md + V6.8系统架构
> **设计风格**: Linear 精致度 + 温暖东方美学

---

## 设计原则

### 视觉语言
- **色调**: 暖色调为主，降低饱和度增加高级感
- **阴影**: 多层微妙阴影，带暖棕色调
- **边框**: 最小化使用，用阴影分隔
- **动效**: Spring physics，温和节奏
- **间距**: 精心调校，respecting whitespace

### 响应式策略
- **Mobile First**: 先设计移动端，再扩展到PC
- **断点**: sm(640px) / md(768px) / lg(1024px) / xl(1280px) / 2xl(1536px)
- **布局切换**: md(768px) 以下为移动布局，以上为PC布局

### 性能优先 (Vercel Best Practices)
- `bundle-dynamic-imports`: 重组件动态加载
- `rerender-memo`: 静态组件 memoization
- `async-suspense-boundaries`: Suspense 流式加载
- `client-swr-dedup`: SWR 请求去重

## 3. 新用户引导 (/onboarding)

### 功能定位
- 收集用户出生信息（必需：年月日；可选：时分、地点）
- 计算 VibeID
- 展示首次 VibeID 结果
- 引导进入主应用

### 步骤流程

```
  Step 1          Step 2          Step 3          Step 4
  Landing    →    Info Input   →   Loading     →    Reveal
  (Welcome)       (Birth Data)     (Calculating)    (VibeID)
```
