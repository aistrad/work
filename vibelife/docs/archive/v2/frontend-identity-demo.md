## 推荐：混合方案 (B + A)

```
响应式策略:

┌─────────────────────────────────────────────────────────┐
│                                                         │
│   PC 端 (≥1024px)           移动端 (<1024px)           │
│   ═══════════════           ═══════════════            │
│                                                         │
│   使用 Side Panel           使用 Bottom Sheet          │
│   (方案 B)                  (方案 A)                   │
│                                                         │
│   • 洞察在右侧展开           • 从底部滑出               │
│   • 聊天保持可见             • 支持半屏/全屏            │
│   • 支持 A2UI 切换           • 手势拖拽关闭             │
│                                                         │
└─────────────────────────────────────────────────────────┘

代码实现:

  if (isMobile) {
    return <VaulDrawer>{insightContent}</VaulDrawer>
  } else {
    return <SidePanel>{insightContent}</SidePanel>
  }

```
