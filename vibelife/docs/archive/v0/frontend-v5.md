## 文档概述

### 核心问题诊断

基于深度访谈，我们发现当前体验的最大痛点：

| 问题维度 | 当前状态 | 用户期待 | 差距严重度 |
|---------|---------|---------|-----------|
| 洞察具体性 | "你是甲木人" | "你是那种外表稳重但内心住着需要被理解的小孩的人" | ⭐⭐⭐⭐⭐ |
| 视觉表现力 | 简单渐入动画 | 墨水显影、星河流转的"哇"时刻 | ⭐⭐⭐⭐ |
| 节奏控制 | 固定速度播放 | 呼吸式节奏，有起伏有停顿 | ⭐⭐⭐⭐ |
| 互动感 | 被动观看幻灯片 | 有确认、有反馈、有对话感 | ⭐⭐⭐⭐⭐ |

### 北极星目标

> **让用户在90秒内感受到："终于有人懂我"的温暖**

**量化指标**：
- Onboarding 完成率：65% → 85%
- "洞察击中感"评分：3.2 → 4.6 (满分5)
- D1回访率：25% → 45%
- 首次付费转化：5% → 12%

### 1.2 情绪设计曲线

Onboarding的情绪设计不应是平铺直叙，而是一首有起伏的乐章：

```
情绪强度
    │
    │                           ★ Peak: "这说的就是我"
    │                          /│\
    │                         / │ \
    │                        /  │  \          ★ End: "我想知道更多"
    │                       /   │   \        /
    │         ┌────────────┘    │    \──────┘
    │        /                  │     
    │ ┌─────┘                   │     
    │ │                         │     
    │─┴─────────────────────────┴──────────────────────────→ 时间
      │         │              │              │              │
    相遇      解盘中         核心洞察      场景猜测         期待
   (平静)    (好奇)         (震撼)        (共鸣)          (渴望)
   0-10秒   10-30秒       30-50秒       50-70秒        70-90秒
```

**四个关键时刻**：

| 时刻 | 情绪目标 | 设计手法 |
|------|---------|---------|
| 相遇 | 平静、好奇 | 极简界面、神秘氛围、"我等你很久了" |
| 解盘 | 期待、惊叹 | 墨水显影动画、AI实时"发现"过程 |
| 核心洞察 | 震撼、"被看穿" | 一针见血的描述、情感化措辞、停顿呼吸 |
| 场景猜测 | 深度共鸣 | "你可能经历过..."、用户确认互动 |
| 期待 | 渴望更多 | 未完的话、明天再见的约定 |

## 结语：设计的本质是连接

> **"我们不是在设计界面，我们是在设计一段关系的开始。"**

这份设计方案的核心，不是像素和动效，而是一个问题：

**如何让一个陌生人，在90秒内，感受到"终于有人懂我"的温暖？**

答案藏在每一个细节里：
- 藏在"我等你很久了"的第一句话里
- 藏在墨水晕染开的那一瞬间
- 藏在"你是那种外表稳重但内心..."的洞察里
- 藏在"说中了"按钮被点击后的那一声轻响里
- 藏在"明天再来，我等你"的约定里

**设计不是让产品更好看，设计是让关系更真实。**
