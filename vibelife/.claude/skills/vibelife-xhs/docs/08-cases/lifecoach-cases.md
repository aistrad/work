## 案例1：关于"自律"的真相

### 内容信息

| 维度 | 内容 |
|-----|------|
| 钩子类型 | C - 反常识宣言 |
| 目标 | 高收藏 + 高转发 |
| Skill关联 | lifecoach - dankoe |

### 标题

```
"自律"是这个时代最大的骗局之一
```

### 正文

```
"自律"是这个时代最大的骗局之一。

我以前也信这套。
买了无数本自律书，下了无数个习惯养成App。
每次坚持不到一周就放弃，然后开始自我攻击：
"我怎么这么没毅力？"

直到我观察了身边那些真正厉害的人。
我发现一件事——

他们从来不觉得自己在"坚持"什么。
他们只是没办法不那样做。

*区别在哪？*

你在"自律"，是因为你还没找到那个让你无法不做的理由。

我用了一个方法——每天早上问自己：如果今天只能做一件事，是什么？
听起来简单，但真的问了才发现，大部分"要做的事"其实不重要。
一周下来，焦虑少了很多，因为不再被"我有很多事没做"的感觉压着。

健身教练不需要"自律"去健身——因为那是他的身份。
作家不需要"自律"去写作——因为不写他会难受。

所以问题不是"如何更自律"。
问题是"你到底想成为什么样的人"。

*当成为那个人的渴望，超过了躺平的舒适——你根本不需要自律。*

如果你现在还在"坚持"什么，可能说明：
你内心深处，并不真正想要它。

这不是在责怪你。
这是在帮你看清。

看清之后，你就可以选择：
要么找到真正想要的，要么承认不想要。
两种都比假装"在努力"强。

*真正的改变，从看清自己开始。*

如果你也想找到那个让你"没办法不做"的理由，
评论"重置"，试试10分钟人生重置。

#自律 #自我成长 #认知升级 #人生规划 #目标
```

### 金句提取

- 「他们从来不觉得自己在"坚持"什么，他们只是没办法不那样做」
- 「问题不是"如何更自律"，问题是"你到底想成为什么样的人"」
- 「当成为那个人的渴望，超过了躺平的舒适——你根本不需要自律」

## 案例2：凌晨三点的焦虑

### 内容信息

| 维度 | 内容 |
|-----|------|
| 钩子类型 | B - 自我揭露 |
| 目标 | 高共鸣 + 引流 |
| Skill关联 | lifecoach + psych |

### 标题

```
凌晨三点睡不着的人，我懂你
```

### 正文

```
上周有个姑娘凌晨3点给我发消息。

她说：「我知道想也没用，但就是停不下来。」

说实话，我太懂这种感觉了。

三年前我也是这样。
每天凌晨躺在床上，脑子像一台停不下来的机器。
明天的会议、上周说错的话、下个月的房租、那个没回的消息。
一个念头接一个念头。

我试过很多方法。
数羊、喝牛奶、听白噪音。
结果呢？数着数着开始想别的，牛奶喝完开始想其他的。

后来我才明白一件事——

*焦虑不是因为你想太多，是因为你不知道怎么和想法相处。*

大多数人的本能是"别想了"。
但"别想了"这三个字，本身就是一个想法。
越压制，越反弹。

真正有用的，不是"不想"，是"换一种方式看"。

把自己想象成一个观众，坐在电影院里。
那些焦虑的念头是屏幕上的画面。
你看着它们，但你不是它们。

*你有想法，不代表你是那个想法。*

那个姑娘后来告诉我，她试了这个方法。
焦虑从每天都有，变成了偶尔一次。

不是焦虑消失了。
是她学会了和焦虑相处。

如果你也是凌晨睡不着的人，
试试这个：不是"别想了"，而是"我注意到我在想这个"。

就这一句话，可能会改变很多。

*你值得被理解，不是被纠正。*

如果你也是凌晨三点还在想很多的人，
你需要的不是"别想了"，而是学会和想法相处。

评论"对话"，试试10分钟自我对话。

#焦虑 #失眠 #凌晨 #情绪管理 #自我成长
```

### 金句提取

- 「焦虑不是因为你想太多，是因为你不知道怎么和想法相处」
- 「你有想法，不代表你是那个想法」
- 「不是焦虑消失了，是她学会了和焦虑相处」

## 案例3：关于"拖延"的真相

### 内容信息

| 维度 | 内容 |
|-----|------|
| 钩子类型 | A - 反向认同 |
| 目标 | 高互动 |
| Skill关联 | lifecoach - dankoe |

### 标题

```
你不是懒，你是怕
```

### 正文

```
你大概率今天又拖延了。

没关系，我也是。

研究显示，95%的人都有拖延问题。
所以你不是一个人。

但问题不在于你不够自律。
问题在于——没有人告诉过你拖延的真正原因。

*拖延不是懒，是保护。*

你的大脑有一套自我保护系统。
当它感知到"威胁"，就会启动逃避模式。

什么是威胁？
- 害怕做不好
- 害怕被评判
- 害怕成功后的责任
- 害怕失败后的失望

所以你拖延的那件事，不是因为它太难。
而是因为它对你太重要。

*你越在乎，越拖延。*

这解释了为什么：
- 小事你能马上做，大事总是拖
- 帮别人的事很快，自己的事很慢
- 不重要的任务效率很高，重要的总是"明天再说"

所以下次你发现自己在拖延的时候，别骂自己"太懒"。

问自己一个问题：
「我在害怕什么？」

我第一次认真问自己这个问题的时候，
发现自己怕的不是失败，是做好之后别人期望更高。
看清这点后，拖延就开始松动了。
不是我变自律了，是我不再和那个恐惧对抗了。

答案可能会让你意外。
可能是害怕做不好。
可能是害怕做好之后，别人对你期望更高。
也可能是害怕承认——这件事其实你并不想做。

*看见恐惧，拖延就开始松动。*

想看清你在害怕什么？评论"拖延"，我帮你拆解。

#拖延 #自我成长 #认知升级 #焦虑 #目标
```

### 金句提取

- 「拖延不是懒，是保护」
- 「你越在乎，越拖延」
- 「看见恐惧，拖延就开始松动」

```
你知道最可怕的是什么吗？

不是失败。
不是被拒绝。
不是赚不到钱。

是你5年后醒来，发现自己还在做同样的事。
想着同样的事。
抱怨同样的事。

不是没有机会，是你每天都在主动选择不改变。

去年这个时候，你说要健身。
今年这个时候，你还在说。

去年这个时候，你说要学新技能。
今年这个时候，你还在收藏。

去年这个时候，你说要改变。

*我们总以为还有时间。*

但时间不会等你准备好。
它只是静静流过，不带任何提醒。

这不是在制造焦虑。
这是在问你一个问题：

如果明年这个时候，你还是这样——
你能接受吗？

如果不能，那今天可以做的最小一步是什么？

不是改变人生。
只是一小步。

*一小步，反复做，就是改变。*

*一小步，反复做，就是改变。*

收藏这篇，明年回来看看你变了没。

如果你不想再等一年，
评论"重置"，从今天的10分钟开始。

#改变 #自我成长 #人生规划 #时间管理 #目标
```

## 案例5：王阳明的答案（东方智慧重写版）

### 内容信息

| 维度 | 内容 |
|-----|------|
| 钩子类型 | D - 直击痛点 |
| 目标 | 深度内容 + 差异化 |
| Skill关联 | lifecoach - yangming |

### 标题

```
"知道但做不到"？500年前就有人解决了
```

### 正文

```
如果你也有过这种感觉——

明明知道应该早起，但就是赖床。
明明知道应该少刷手机，但就是控制不住。
明明知道应该开始了，但就是拖到明天。

那这篇是写给你的。

500年前有个叫王阳明的人，他提出了四个字：
**知行合一**。

大多数人理解错了。

以为是"先知道，再去做"。
但王阳明说的是：*真正的知道，本身就包含了做到。*

什么意思？

你说你"知道"应该早起。
但你还是赖床。
王阳明会说：你其实并不"真知"。

*真知 = 知道 + 感受到 + 认同*

你"知道"熬夜不好（信息层面）。
但你"感受"到熬夜带来的即时快乐（感受层面）。
你"认同"这种快乐（价值层面）。

所以你的行为跟随的是感受和认同，而不是"知道"。

怎么从"知道"变成"真知"？

**1. 体验后果**
不是别人告诉你熬夜不好，而是你亲身体验第二天的疲惫，然后把这种感受记住。

**2. 找到更大的意义**
早起不是为了"应该"，而是因为它能帮你实现更重要的东西——比如健康、比如时间、比如成为你想成为的人。

**3. 从小开始验证**
不是一下子早起2小时。而是先早起10分钟，感受那个变化。让身体和心理都"知道"：这是好的。

*你不是做不到，你是"知"得还不够深。*

下次你说"我知道但做不到"的时候，
问自己：我真的"知道"吗？还是只是"听说过"？

*你不是做不到，你是"知"得还不够深。*

我用了一年才真正理解这句话。
现在每次"知道但做不到"的时候，我会问自己三个问题。

想知道是哪三个问题？评论"知行"。

#王阳明 #知行合一 #自我成长 #认知升级 #哲学
```

### 金句提取

- 「真正的知道，本身就包含了做到」
- 「你不是做不到，你是"知"得还不够深」
- 「不是"知道但做不到"，是"听说过但没真正理解"」

## 内容生产模板 v2.0

### Dan Koe 风格模板

```
【第一段：钩子】（选择ABCD类型之一，3秒抓住人）
- 反常识宣言
- 自我揭露
- 反向认同
- 直击痛点

【第二段：共鸣】（50-80字，建立"你懂我"的感觉）
我以前也[同样的问题]。
试过[无效方法]。
结果[失败描述]。
直到我明白[转折]。

【第三段：洞察】（核心观点，先破后立）
大多数人以为[错误认知]。
但真相是[正确理解]。
*金句*

【第四段：支撑】（具体例子、场景、数据）
这解释了为什么...
比如...
观察一下身边...

【第五段：应用】（可执行的小步骤）
所以下次你[场景]的时候，
试试这个：[具体做法]

【第六段：赋权收尾】（不是命令，是邀请）
如果这对你有帮助，收藏它。
你值得[正向结果]。
```
