# 金句模板库

> 每篇内容至少3句可摘抄的金句

---

## 一、通用金句模板

### 身份认知类

```
如果你想要{结果}，你必须先成为{会产生结果的人}。
```
- 如果你想要自律，你必须先成为一个真正想要那个目标的人。
- 如果你想要被爱，你必须先成为一个能接受爱的人。

```
你不是没有{能力}，你是{真正原因}。
```
- 你不是没有自律，你是还没找到真正想要的东西。
- 你不是没有方向，你是方向太多。
- 你不是不够好，你是还没学会看见自己的好。

```
{问题}不是{常见归因}，是{真正原因}。
```
- 拖延不是懒，是你在保护自己免受某种恐惧。
- 焦虑不是病，是内心冲突的信号。
- 内耗不是想太多，是行动太少。

### 认知升级类

```
真正的{目标}不是{表面做法}，而是{深层原则}。
```
- 真正的改变不是控制行为，而是升级欲望。
- 真正的自律不是强迫自己，而是找到不需要强迫的事。
- 真正的放下不是忘记，而是不再期待。

```
{常见概念}只是{表象}，{深层概念}才是{本质}。
```
- 努力只是入场券，认知才是分水岭。
- 方法只是工具，心态才是根基。
- 技巧只是皮毛，洞察才是内核。

```
问题不在于你{表面问题}，而在于你{深层问题}。
```
- 问题不在于你缺乏什么，而在于你想要什么。
- 问题不在于你做了什么，而在于你为什么做。
- 问题不在于你知道多少，而在于你相信什么。

### 反直觉洞察类

```
你越{努力方向}，就越{反效果}。
```
- 你越想控制焦虑，焦虑就越控制你。
- 你越想被喜欢，就越容易迷失自己。
- 你越想完美，就越不敢开始。

```
{看似缺点}的背面，是{隐藏优点}。
```
- 敏感的背面，是共情力。
- 焦虑的背面，是责任感。
- 拖延的背面，是完美主义。

```
那些{负面特质}的人，其实是{正向解读}。
```
- 那些"想太多"的人，其实是愿意为未来负责的人。
- 那些"太敏感"的人，其实是能感知到别人忽略的信号的人。
- 那些"不合群"的人，其实是有自己判断标准的人。

### 时间维度类

```
{短期行为}是{短期效果}，{长期行为}才是{长期效果}。
```
- 逃避是麻醉剂，面对才是解药。
- 舒适是陷阱，不适才是成长。
- 确定感是幻觉，不确定才是常态。

```
你今天的{选择}，决定了你{时间后}的{结果}。
```
- 你今天的认知，决定了你五年后的生活。
- 你今天的习惯，决定了你十年后的身体。
- 你今天的选择，决定了你一生的方向。

## 二、Skill专属金句

### psych 心理探索

**认知扭曲主题**：
- *你以为是世界对你不友好，其实是你的想法在欺负你。*
- *大脑最擅长的事，是用想象的痛苦折磨真实的你。*
- *你不是在想事情，是事情在想你。*
- *思维是工具，不是主人。你可以使用它，不必被它使用。*

**依恋类型主题**：
- *你在感情里的不安全感，不是这段感情的问题，是童年的回声。*
- *我们不是在找伴侣，是在找小时候缺失的那块拼图。*
- *你以为你在爱TA，其实你在治疗自己。*
- *安全感不是别人给的，是你自己长出来的。*

**情绪管理主题**：
- *情绪不是敌人，是信使。杀死信使，不会让消息消失。*
- *你不需要控制情绪，你需要理解情绪想告诉你什么。*
- *崩溃不是软弱，是积压太久的信号终于被听见。*
- *允许自己不好，是变好的第一步。*

**内耗主题**：
- *内耗的本质是：你在用想象的行动，代替真实的行动。*
- *想得多不是问题，问题是想完不动。*
- *你的大脑在加班，但你的人生在停滞。*
- *治疗内耗最好的药：做一个5分钟就能完成的小事。*

### zodiac 西方占星

**星座性格主题**：
- *星座不是给你贴标签，是帮你理解为什么你是这样的人。*
- *你的星盘不是命运，是倾向。倾向可以被意识改变。*
- *了解自己的星座，不是为了找借口，是为了更精准地成长。*

**星座季主题**：
- *每个星座季都是一次能量邀请，你可以接受，也可以旁观。*
- *宇宙不会强迫你改变，只会在合适的时机轻轻推你一把。*

### lifecoach 人生教练

**目标设定主题**：
- *目标不清晰，努力就是自我感动。*
- *你不是缺乏动力，是目标不够性感。*
- *真正的目标不是"应该做"，是"不得不做"。*

**习惯养成主题**：
- *习惯的本质不是重复，是身份认同。*
- *你不是在培养习惯，你是在成为一个新的人。*
- *不要问"我能坚持多久"，要问"我想成为什么样的人"。*

**迷茫主题**：
- *迷茫不是没有方向，是方向太多。*
- *你不需要找到答案，你需要开始行动。答案会在路上出现。*
- *迷茫是特权——说明你还有选择的余地。*

### tarot 塔罗占卜

- *塔罗不是预测未来，是照见当下被忽视的部分。*
- *牌不会告诉你该怎么做，它只是让你看见你已经知道的答案。*
- *你选的牌，就是你内心深处最想被看见的那个部分。*

### bazi 八字命理

- *八字不是宿命，是你的出厂设置。设置可以被优化。*
- *五行不是迷信，是古人对人格类型的朴素描述。*
- *了解自己的命盘，不是认命，是知道该往哪个方向用力。*

## 三、结尾金句模板

### 赋权型结尾

```
这不是什么高深的道理，但大多数人一辈子都不会真正理解它。
而你，现在已经知道了。
```

```
改变不是一夜之间发生的，但它确实是从某一刻开始的。
也许，就是现在这一刻。
```

```
你不需要变成另一个人。
你只需要更了解这一个。
```

### 行动型结尾

```
如果你看到这里，我建议你做一件事：
[具体行动]
不需要很久，[时间]就够了。
但这[时间]可能会改变你接下来的[更长时间]。
```

### 思考型结尾

```
最后，留一个问题给你：
[问题]
不用急着回答。
有时候，好问题比好答案更有价值。
```

## 四、金句创作公式

### 公式1：先破后立

```
{常见认知}是错的。真正的{主题}是{正确认知}。
```

### 公式2：反直觉对比

```
{A}和{B}的区别，决定了你能走多远。
```

### 公式3：重新定义

```
{概念}不是{常见定义}，是{新定义}。
```

### 公式4：身份转换

```
你不是{负面标签}，你是{正向重构}。
```

### 公式5：因果倒置

```
不是因为{常见原因}才{结果}，是因为{真正原因}。
```

## 五、使用指南

### 金句密度标准

| 内容长度 | 最少金句数 |
|---------|----------|
| 300字以下 | 2句 |
| 300-500字 | 3句 |
| 500字以上 | 4-5句 |

### 金句放置位置

1. **开头**：作为钩子的一部分
2. **每个核心观点后**：作为总结
3. **结尾**：作为收束

### 金句标注方式

用 *斜体* 标注金句，让读者一眼能找到可摘抄的内容。
