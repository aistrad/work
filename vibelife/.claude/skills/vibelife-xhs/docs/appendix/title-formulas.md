# 50+标题公式

## 数字型公式

### 1. 数字+痛点+解决方案
```
[N]个让你[解决痛点]的方法
```
- 3个让你快速平静的方法
- 5个让你停止焦虑的技巧
- 7个让你不再内耗的习惯

### 2. 数字+时间+结果
```
[时间]改变[X]的[N]个方法
```
- 21天改变一个坏习惯的方法
- 3分钟让自己平静下来
- 1个月养成早起习惯

### 3. 数字+身份+特征
```
[身份]的[N]个[特征/秘密]
```
- 天蝎座的5个真相
- 高敏感人群的7个特征
- 内向者的10个优势

### 4. 数字+清单
```
[N]个[主题]，建议收藏
```
- 15种认知扭曲，你中了几种？
- 12星座本月运势，一图看懂
- 7个习惯，让你越来越好
