# 发帖前检查清单

## 发布前必查（每篇）

### 内容检查

- [ ] **标题**
  - [ ] 长度15-25字
  - [ ] 包含数字/关键词
  - [ ] 无敏感词
  - [ ] 与封面配合（不重复）

- [ ] **正文**
  - [ ] 排版清晰（空行、emoji、序号）
  - [ ] 无错别字
  - [ ] 无敏感词/违禁词
  - [ ] 无其他平台引流信息
  - [ ] 结尾有CTA（评论引导）

- [ ] **图片/封面**
  - [ ] 尺寸正确（3:4或1:1）
  - [ ] 清晰度足够
  - [ ] 无其他平台水印
  - [ ] 品牌水印位置正确
  - [ ] 首图吸引眼球

- [ ] **标签/话题**
  - [ ] 添加3-5个相关标签
  - [ ] 包含热门话题
  - [ ] 与内容强相关

### 引流检查

- [ ] 评论区置顶话术已准备
- [ ] 私信回复模板已准备
- [ ] 引流链接已准备且可用

### 技术检查

- [ ] 未连接WiFi（使用移动数据）
- [ ] 距上次发帖间隔>1小时
- [ ] 发布时间合适（非凌晨）

## 发布后操作

### 立即执行（5分钟内）

- [ ] 发布评论区置顶
- [ ] 检查笔记是否正常展示
- [ ] 检查是否收到违规提醒

### 1小时后检查

- [ ] 查看初始数据（小眼睛、点赞）
- [ ] 回复评论（如有）
- [ ] 确认无异常限流迹象

### 24小时后复盘

- [ ] 记录数据（小眼睛、点赞、收藏、评论）
- [ ] 计算点赞率
- [ ] 分析数据异常原因（如有）

## 常见问题排查

### 发布后0流量

| 可能原因 | 排查方法 | 解决方案 |
|---------|---------|---------|
| 内容违规 | 检查是否收到违规通知 | 修改或删除重发 |
| 敏感词 | 逐句检查正文 | 替换敏感词 |
| 账号问题 | 发其他内容测试 | 养号或换号 |
| 图片问题 | 检查水印/违规元素 | 更换图片 |

### 流量突然下降

| 可能原因 | 排查方法 | 解决方案 |
|---------|---------|---------|
| 被举报 | 检查是否收到通知 | 申诉 |
| 内容同质化 | 对比近期内容 | 调整内容方向 |
| 发布时间 | 对比历史数据 | 调整发布时间 |
| 平台算法调整 | 观察其他账号 | 等待恢复 |

## 敏感词速查

### 医疗健康类（高风险）

```
抑郁症、焦虑症、诊断、治疗、处方、药物、
躁郁症、精神分裂、PTSD、用药、开药
```

**替代说法**：
- "抑郁症" → "情绪低落"
- "焦虑症" → "焦虑情绪"
- "治疗" → "调节/改善"

### 绝对化用语（中风险）

```
100%、一定、绝对、永远、从不、最好、最差
```

**替代说法**：
- "100%有效" → "很多人反馈有效"
- "一定能" → "大概率能"

### 引流相关（中风险）

```
加微信、加群、私聊、免费领取、限时免费、
点击下方、链接在、二维码
```

**替代说法**：
- "加微信" → "私信聊"
- "免费领取" → "评论关键词私发"

### 命理占卜类（中风险）

```
算命、命中注定、天命、劫难、灾祸、破财、
必定发生
```

**替代说法**：
- "命中注定" → "能量趋势"
- "灾祸" → "挑战"
- "必定发生" → "可能出现"

## 发布SOP

```
Step 1: 内容最终检查
        ↓
Step 2: 断开WiFi，使用移动数据
        ↓
Step 3: 确认距上次发帖>1小时
        ↓
Step 4: 发布笔记
        ↓
Step 5: 立即发布评论置顶
        ↓
Step 6: 5分钟后检查是否正常展示
        ↓
Step 7: 记录发布时间和初始数据
```
