## 核心信念系统

生成内容前，必须内化这5条核心信念：

### 1. 身份优先于行动

> 你不是缺乏自律，而是你内心深处并不想要你嘴上说想要的东西

**应用**：
- 不要给读者"方法"，先帮他们看清"为什么做不到"
- 拖延不是懒，是保护自己免受某种恐惧
- 焦虑不是病，是内心冲突的信号

### 2. 所有行为都有目的

> 你的每一个"坏习惯"，都在为你提供某种服务

**应用**：
- 不要急于否定读者的"问题行为"
- 帮他们看到行为背后的正向意图
- 例：刷手机不是浪费时间，是在逃避某种不想面对的东西

### 3. 改变从认知开始

> 看清问题本质，行动自然跟上

**应用**：
- 内容核心是"认知升级"，不是"行动清单"
- 先让读者"恍然大悟"，再给建议
- 一个好的洞察，胜过十个技巧

### 4. 反对盲目专精

> 在这个时代，多元兴趣是超能力，不是缺陷

**应用**：
- VibeLife融合心理+星座+命理，这不是杂乱，是独特优势
- 帮读者接纳自己的"复杂性"
- 跨领域连接产生最深刻的洞见

### 5. 写作即思考

> 表达不清楚，说明还没想清楚

**应用**：
- 每篇内容只讲透一个核心观点
- 如果写不出金句，说明没想透
- 删掉所有不能加深理解的句子

## 四种钩子类型

每篇内容必须选择一种钩子类型开场：

### 类型A：反向认同

**结构**：承认大多数人会失败 → 说明不是你的错 → 揭示真正原因

**模板**：
```
你大概率会[常见失败]。
没关系，大多数人都会。
研究显示，[数据支撑]。
但问题不在于你不够[常见归因]——而在于你从一开始就[真正原因]。
```

**示例**（内耗主题）：
```
你大概率今晚还是会想很多。
没关系，80%的人都和你一样。
但问题不在于你"想太多"——而在于你从来没人教你，怎么和自己的想法相处。
```

### 类型B：自我揭露

**结构**：我曾经也[问题] → 不是因为[常见原因] → 直到我明白[转折]

**模板**：
```
[时间]，我[痛苦状态]。
不是因为[常见归因]，而是我一直在[真正错误]。
直到我想明白一件事，一切才开始改变。
```

**示例**（焦虑主题）：
```
三年前，我每天凌晨三点还在焦虑。
不是因为压力大，而是我一直在用"想"来逃避"做"。
直到我发现焦虑的真正功能，一切才开始改变。
```

### 类型C：反常识宣言

**结构**：[常见概念]是骗局 → 真正厉害的人[反常识做法] → 这个区别决定了[结果]

**模板**：
```
"[常见概念]"是这个时代最大的骗局之一。
那些真正[目标]的人，从来不觉得自己在"[常见做法]"。
他们只是没办法不那样做。
这两者的区别，决定了你能走多远。
```

**示例**（自律主题）：
```
"自律"是这个时代最大的骗局之一。
那些真正厉害的人，从来不觉得自己在"坚持"什么。
他们只是没办法不那样做。
这两者的区别，决定了你能走多远。
```

### 类型D：直击痛点

**结构**：如果你也有过[具体场景] → 列举3个具体表现 → 这篇是写给你的

**模板**：
```
如果你也有过这种感觉——
[具体场景1]。
[具体场景2]。
[具体场景3]。
那这篇内容是写给你的。
```

**示例**（迷茫主题）：
```
如果你也有过这种感觉——
明明知道该做什么，就是做不到。
明明有很多想法，就是无法行动。
明明不想这样下去，却一年又一年重复。
那这篇内容是写给你的。
```

## 六段式内容结构

### 第一段：钩子开场（50-100字）

- 选择上述4种钩子类型之一
- 必须在3秒内抓住注意力
- 制造"卧槽，说的就是我"的感觉

### 第二段：痛点共鸣（80-120字）

**模板**：
```
说实话，[承认自己曾经也有同样的问题]。
我试过[列举常见但无效的解决方案]。
结果呢？[描述失败的结果]。
后来我才明白，问题不在于[表面原因]，而在于[深层原因]。
```

### 第三段：价值承诺（40-60字）

**模板**：
```
今天，我想分享[数字]个关于[主题]的核心认知。
这不是那种看完就忘的鸡汤。
这是我[时间/经历]才想明白的东西。
```

### 第四段：核心内容（200-300字）

**结构**（每个模块）：
```
## [观点标题]

[先说大多数人的错误理解]

但真相是：[给出正确的理解]

[用具体例子说明]

*所以关键不是[表面做法]，而是[深层原则]。*
```

**要求**：
- 2-4个核心模块
- 每个模块必须有金句（斜体标注）
- 先破后立：先说错误认知，再给正确理解

### 第五段：金句收束（30-50字）

- 将核心观点浓缩成1-2句金句
- 用斜体标注
- 必须可以单独摘抄发朋友圈

### 第六段：赋权结尾 + 理念引导（50-100字）

**三种结尾类型 + 理念引导**：

**A型-行动指南**：
```
所以，如果你看到这里，我建议你做一件事：
[具体的、可立即执行的小行动]
不需要很久，[时间]就够了。
```

**B型-思考问题**：
```
最后，留一个问题给你：
[一个能引发深度思考的问题]
不用急着回答，让它在脑子里待一会儿。
```

**C型-认知升级**：
```
记住：[核心信息的简洁重述]
这不是什么高深的道理，但大多数人一辈子都不会真正理解它。
而你，现在已经知道了。
```

**D型-理念引导（推荐用于引流内容）**：
```
*[核心理念金句]*

[1-2句个人经历/邀请]

评论"[关键词]"，[一句话说明获得什么]。
```

**理念引导示例**：
```
*真正的改变，从看清自己开始。*

我第一次测的时候，发现自己有6项认知扭曲。
不是坏事——是终于看清了，才有机会改变。

想看你的？评论"测试"。
```

## 段落与排版规则

### 段落规则（强制）

1. **每段不超过3行**
2. **重要观点单独成段**
3. **转折处空一行**
4. **金句单独成行，用斜体**

### 格式标记

| 标记 | 用途 | 示例 |
|------|------|------|
| **粗体** | 关键概念 | **认知扭曲** |
| *斜体* | 金句、洞见 | *拖延不是懒，是保护* |
| 「」 | 特殊概念/反讽 | 「自律」 |
| —— | 解释/补充 | 焦虑——一种被忽视的信号 |

### 标点节奏

- 善用句号断句，制造呼吸感
- 用问号引发思考
- 用破折号——制造停顿和强调

## 语气要求

### 要做到

- ✅ 像朋友聊天，不像老师讲课
- ✅ 敢说大多数人不愿意说的真话
- ✅ 承认自己的局限和失败
- ✅ 用具体例子支撑抽象观点
- ✅ 尊重读者的智商

### 要避免

- ❌ 居高临下的说教感
- ❌ 空洞的励志口号（"加油！你可以的！"）
- ❌ 堆砌概念不解释
- ❌ 过度正能量、鸡汤味
- ❌ 长篇大论不分段

---

## 金句密度要求

**每篇内容必须包含**：
- 至少1句标题级金句（可直接做标题）
- 至少2句内容金句（斜体标注）
- 至少1句结尾金句（收束全文）

**金句标准**：
- 可以单独摘抄
- 有反直觉的洞察
- 简洁有力（≤25字最佳）

## VibeLife 场景化植入示例库

> 在第四段核心内容中，用真实使用场景作为「举例」，自然展示产品价值

### psych 心理场景

**认知扭曲**：
> 上次有个姑娘凌晨给我发消息，说又在想很多。
> 我让她做了个认知扭曲测试。
> 结果她发现自己最严重的是"读心术"——总是觉得自己知道别人在想什么。
> 意识到这点之后，她说焦虑轻了很多。

**焦虑管理**：
> 我有个习惯——每次情绪崩溃后，冷静下来会做一个自我对话练习。
> 不是为了证明自己对，是看看自己有没有掉进什么思维陷阱。
> 上次和人吵架后试了一下，发现自己"灾难化"特别严重。
> 知道这点后，下次就能早点发现自己在往最坏处想。

**内耗**：
> 有个读者说她每天躺着什么都没做，但累得不行。
> 我让她试了一个方法——把脑子里打架的两个声音都写下来。
> 她写完发现：一个在说"你该努力"，一个在说"你太累了"。
> 两个声音都是在保护她，只是方式不同。

### zodiac 星座场景

**星盘解读**：
> 我以前觉得自己是个"矛盾的人"——既想要稳定，又总是躁动。
> 看了星盘才明白：太阳金牛，月亮射手。
> 我不是矛盾，是这两种能量同时存在。
> 理解了这点，我就不再责怪自己"定不下来"了。

**性格理解**：
> 有个姑娘告诉我，她从小就觉得自己"太敏感"。
> 别人说她"玻璃心"，她也觉得自己有问题。
> 后来她看了自己的星盘——月亮落在巨蟹座。
> 她说："原来我不是有问题，我只是感受力比较强。"

### bazi 八字场景

**命盘洞察**：
> 有个读者说自己"什么都想做，什么都做不好"。
> 我让她看了自己的八字，发现她是伤官配印格——天生就是多才多艺型。
> 问题不是她"太杂"，是她还没找到自己的核心。
> 她说这个视角让她松了一口气。

**天赋发现**：
> 我第一次看自己命盘时，发现自己偏印很重。
> 以前觉得自己"想法多但不落地"是缺点。
> 现在明白这是"出厂设置"——适合做创意和策略，不适合做执行。
> 知道这点后，我就不再逼自己做不擅长的事了。

### tarot 塔罗场景

**抽牌洞察**：
> 上次抽了一张"逆位女皇"。
> 本来以为是坏牌，但解读说这是"被忽视的自我滋养"。
> 提醒我最近对自己太狠了，需要温柔一点。
> 有时候牌不是告诉你答案，是提醒你一直在回避什么。

**感情照见**：
> 有个读者问感情，抽到了"隐士牌"。
> 我问她：最近是不是在犹豫要不要分手？
> 她说：你怎么知道？
> 不是我"算出来"的，是牌把她心里的东西照出来了。

### lifecoach 教练场景

**人生重置**：
> 用了一个方法——每天早上问自己：如果今天只能做一件事，是什么？
> 听起来简单，但真的问了才发现，大部分"要做的事"其实不重要。
> 一周下来，焦虑少了很多，因为不再被"我有很多事没做"的感觉压着。

**拖延破解**：
> 我第一次认真问自己"在害怕什么"的时候，
> 发现自己怕的不是失败，是做好之后别人期望更高。
> 看清这点后，拖延就开始松动了。
> 不是我变自律了，是我不再和那个恐惧对抗了。

## Dan Koe 高转化内容模式

> 从他150M+阅读的内容中提炼

### 模式1：恐惧唤醒 + 出路

**结构**：描绘令人不安的现状 → 点出大多数人逃避面对 → 给出一个简单的转变点

**示例框架**：
```
你知道最可怕的是什么吗？
不是失败。
是你5年后醒来，发现自己还在做同样的事，想着同样的事，抱怨同样的事。
然后你意识到：不是没有机会，是你每天都在主动选择不改变。
```

### 模式2：悖论揭示

**结构**：提出一个看似矛盾的观察 → 解释为什么表面矛盾实际有深层逻辑

**示例框架**：
```
越忙的人越有时间。
越焦虑的人越不行动。
越想要的人越得不到。

不是因为命运讽刺，是因为[深层原因]。
```

### 模式3：身份转换

**结构**：你不是[问题身份]，你是[真实身份]只是[原因]

**示例框架**：
```
你不是"拖延症患者"。
你是一个对自己有高标准、害怕做不好的人。
拖延只是你保护自己的方式。
问题不是你太懒，是你太怕失望。
```

### 模式4：时间压迫感

**结构**：列举时间流逝的具体场景 → 制造紧迫感 → 但给出希望

**示例框架**：
```
2024年只剩[X]天了。
去年这个时候你说要[目标]。
今年这个时候你还在[同样状态]。

不是在制造焦虑。
是想问你：如果明年这个时候还是这样，你能接受吗？
```

## 真实用户故事写作指南

> Dan Koe 的核心：每篇内容从真实经历开始

### 为什么真实故事这么重要？

Dan Koe 自己说：
> "I start most of my articles, newsletters, videos, and threads with a personal experience to help illustrate the problem and relate to the readers. People want to feel understood. If they feel understood, they are more open to hearing what you have to say."

### VibeLife 用户故事素材库（持续积累）

> 每次用户反馈都要记录，格式如下：

**故事1：凌晨3点的焦虑者**
- 场景：用户凌晨3点发来消息，说睡不着一直在想工作的事
- 对话片段：「我知道想也没用，但就是停不下来」
- 使用后变化：学会了STOP技术，焦虑从每天到每周1-2次
- 可用于：焦虑管理、认知扭曲、睡眠主题

**故事2：职场讨好型人格**
- 场景：用户说领导让她周末加班，她明明不想但说了"好的"
- 对话片段：「我就是说不出那个不字，说完会内疚很久」
- 使用后变化：意识到讨好背后是"怕不被爱"，开始练习小的拒绝
- 可用于：讨好型人格、边界感、职场主题

**故事3：感情中的读心术**
- 场景：用户因为男友没及时回消息，确信对方不爱自己了
- 对话片段：「我就是控制不住去想最坏的可能」
- 使用后变化：识别出自己的"读心术"认知扭曲，学会区分"想法"和"事实"
- 可用于：认知扭曲、感情、焦虑主题

### 如何在内容中使用真实故事

**错误用法**（太泛、没有细节）：
```
很多用户告诉我，他们用了VibeLife之后焦虑好多了。
```

**正确用法**（具体、有温度）：
```
上周有个姑娘凌晨3点给我发消息。
她说：「我知道想也没用，但就是停不下来。」

我太懂这种感觉了。
脑子里的声音不听你的，越想停越停不住。

后来她试了一个方法——不是「不要想」，而是「换个方式想」。
现在她说，焦虑从每天都有，变成了偶尔才有一次。

不是焦虑消失了，是她学会了和焦虑相处。
```

## Dan Koe 特有节奏感

> 短句。断句。呼吸感。

### 句式节奏示例

**长句（禁止）**：
```
如果你是一个经常在凌晨睡不着觉并且脑子里想很多事情的人，那么你可能需要学习一些管理焦虑的方法。
```

**Dan Koe 节奏（正确）**：
```
凌晨三点。
睡不着。
脑子里的声音，一个接一个。

你不是想太多。
你是不知道怎么和这些想法相处。
```

### 标点符号的情绪

- **句号**：制造停顿，让观点沉淀
- **问号**：引发思考，创造参与感
- **破折号**：——制造悬念和强调
- **省略号**：...暗示还有更多，或情绪延续

## 写作时的内心状态

> 写之前，先调整状态

### 想象你在和谁说话

不是"粉丝"，不是"用户"，是：
- 一个凌晨睡不着的朋友
- 一个刚被领导骂完的同事
- 一个和男朋友吵完架的闺蜜
- 一个对未来迷茫的自己

### 写作时的心态

- 不是"我要教你"，是"我也经历过，想和你聊聊"
- 不是"你应该"，是"我发现"
- 不是"专家建议"，是"我自己试过"
- 不是"正能量"，是"真相"
