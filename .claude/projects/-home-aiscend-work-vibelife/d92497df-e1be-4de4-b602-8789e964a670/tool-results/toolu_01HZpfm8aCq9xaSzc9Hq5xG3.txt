   400→## 十一、资产制作检查清单
   401→
   402→### 11.1 SVG 优化要求
   403→
   404→- [ ] 使用 SVGO 压缩
   405→- [ ] 移除多余元数据
   406→- [ ] 合并相同路径
   407→- [ ] 使用 currentColor 支持颜色继承
   408→- [ ] viewBox 设置正确
   409→
   410→### 11.2 图片优化要求
   411→
   412→- [ ] 提供 WebP 格式
   413→- [ ] 提供 2x 分辨率版本
   414→- [ ] 文件大小 < 100KB
   415→- [ ] 正确的 alt 描述
   416→
   417→### 11.3 动画优化要求
   418→
   419→- [ ] Lottie 文件 < 50KB
   420→- [ ] 帧率 ≤ 60fps
   421→- [ ] 提供 reduced-motion 静态替代
