   416→  const sizes = { default: 'px-6 py-3 text-body', large: 'px-8 py-4 text-large' }
   417→  return (
   418→    <motion.div whileHover={{ y: -2 }} whileTap={{ scale: 0.98 }}>
   419→      <Link href={href} className={`${base} ${variants[variant]} ${sizes[size]}`} style={{ boxShadow: 'var(--shadow-card)' }}>
   420→        {children}
   421→      </Link>
   422→    </motion.div>
   423→  )
   424→}
   425→```
