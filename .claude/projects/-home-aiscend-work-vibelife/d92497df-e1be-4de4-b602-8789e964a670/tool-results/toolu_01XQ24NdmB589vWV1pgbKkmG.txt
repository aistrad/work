   359→
   360→### 3.1 应该用的表达
   361→
   362→```
   363→┌─────────────────────────────────────────────────────────────────────────────┐
   364→│                                                                             │
   365→│                    推荐表达 ✓                                                │
   366→│                                                                             │
   367→├─────────────────────────────────────────────────────────────────────────────┤
   368→│                                                                             │
   369→│  • "你当前呈现为..." (而非 "你是...")                                       │
   370→│  • "一种生活策略" (而非 "一种命运")                                         │
   371→│  • "成长的机会" (而非 "需要改正的问题")                                     │
   372→│  • "了解自己，是为了更好地选择" (而非 "了解命运")                           │
   373→│  • "阴影是未被开发的潜能" (而非 "阴影是缺陷")                               │
   374→│  • "四维不一致是丰富" (而非 "四维不一致是冲突")                             │
   375→│  • "需要勇气" (而非 "需要运气")                                             │
   376→│  • "你可以选择" (而非 "你注定如此")                                         │
   377→│                                                                             │
   378→└─────────────────────────────────────────────────────────────────────────────┘
   379→```
   380→
   381→### 3.2 应该避免的表达
   382→
   383→```
   384→┌─────────────────────────────────────────────────────────────────────────────┐
   385→│                                                                             │
   386→│                    避免表达 ✗                                                │
   387→│                                                                             │
   388→├─────────────────────────────────────────────────────────────────────────────┤
   389→│                                                                             │
   390→│  • "你的命运是..." / "你天生就是..."                                        │
   391→│  • "好日子" / "坏日子"                                                      │
