     1→# 场景生成 Prompt 模板
     2→
     3→> 此 Prompt 在 Claude Code 中手动执行，不调用外部 LLM API。
     4→
     5→## 使用方法
     6→
     7→1. 从 knowledge_chunks 表中读取待分析的知识内容
     8→2. 查询已有场景列表
     9→3. 将内容填入下方模板
    10→4. 在 Claude Code 中执行 prompt
    11→5. 将结果手动写入 scenario_candidates 表
    12→
