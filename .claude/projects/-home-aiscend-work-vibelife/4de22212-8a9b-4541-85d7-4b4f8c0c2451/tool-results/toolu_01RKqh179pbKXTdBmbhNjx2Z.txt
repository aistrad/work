     3→## Minimal Skill (Single File)
     4→
     5→```markdown
     6→---
     7→name: example-skill
     8→description: Does X and Y. Use when user mentions Z or asks about W.
     9→---
    10→
    11→# Example Skill
    12→
    13→## Quick Start
    14→
    15→[Most common use case - 3-5 lines]
    16→
    17→## Workflow
    18→
    19→1. Step one
    20→2. Step two
    21→3. Step three
    22→
    23→## Examples
    24→
    25→**Input**: [concrete example]
    26→**Output**: [expected result]
    27→```

    29→## Standard Skill (With References)
    30→
    31→```
    32→skill-name/

    35→    └── details.md
    36→```

    38→**SKILL.md**:
    39→```markdown
    40→---
    41→name: skill-name

    43→---
    44→
    45→# Skill Name
    46→
    47→## Quick Start

    62→## Complex Skill (Full Structure)
    63→
    64→```
    65→skill-name/
    66→├── SKILL.md
    67→├── reference/
    68→│   ├── api.md
    69→│   └── troubleshooting.md
    70→├── scripts/
    71→│   ├── validate.py
    72→│   └── process.py
    73→└── examples/
    74→    └── sample-input.json
    75→```
    76→
    77→**SKILL.md**:
    78→```markdown
    79→---
    80→name: skill-name
    81→description: [Comprehensive description with trigger phrases]
    82→---
    83→
    84→# Skill Name
    85→
    86→## Quick Start
    87→
    88→Run: `python scripts/process.py input.json`
    89→
    90→## Workflow
    91→
    92→### Step 1: Validate
    93→python scripts/validate.py input.json
    94→
    95→### Step 2: Process
    96→python scripts/process.py input.json output.json
    97→
    98→### Step 3: Verify
    99→Check output.json for expected structure.
   100→
   101→## Reference
   102→- [API Documentation](reference/api.md)
   103→- [Troubleshooting Guide](reference/troubleshooting.md)
   104→
   105→## Examples
   106→See [examples/](examples/) for sample inputs.
   107→```
   108→
   109→## Checklist Before Finalizing
   110→
   111→- [ ] name: lowercase, hyphens, max 64 chars
   112→- [ ] description: third person, what + when, max 1024 chars
   113→- [ ] SKILL.md body: < 500 lines
   114→- [ ] Quick start: covers most common case
   115→- [ ] References: max 1 level deep
   116→- [ ] Examples: concrete, not abstract
   117→- [ ] Terminology: consistent throughout
   118→
