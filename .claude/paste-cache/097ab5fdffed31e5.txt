Q4.1: 半屏浮层 (Half-Modal) 方案                                                                                                              
                                                                                                                                                
  是否可以考虑:                                                                                                                                 
  - Chat 页面全屏显示                                                                                                                           
  - 点击进入 Protocol 后,从右侧滑入一个 Modal                                                                                                   
  - Modal 覆盖部分 Chat 区域(比如占 50-70% 宽度)                                                                                                
  - 背景 Chat 模糊可见                                                                                                                          
  - 完成后 Modal 滑出                                                                                                                           
                                                                                                                                                
  这样的好处:                                                                                                                                   
  - 保持 Protocol 的完整 UI 体验                                                                                                                
  - 不需要重构右侧栏架构                                                                                                                        
  - 用户能感知到"正在做子任务"                                                                                                                  
                                                                                                                                                
  Q4.2: 内联嵌入方案                                                                                                                            
                                                                                                                                                
  或者:                                                                                                                                         
  - Protocol 不独立展示                                                                                                                         
  - 而是作为特殊的 Chat 卡片嵌入对话流                                                                                                          
  - 进度条 + 当前问题直接显示在聊天区域                                                                                                         
  - 用户在原地完成流程                                                                                                                          
                                                                                                                                                
  这样的体验是否符合你的预期?
