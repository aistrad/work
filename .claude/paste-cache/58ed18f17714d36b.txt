Q2: LLM 个性化生成 - 选择题 + 可选文本追加
────────────────────────────────────────────────────────────────
示例（甲木日主）:
"作为甲木日主的你，在关系中是否常常感到：
 [ ] 需要扎根的安全感
 [ ] 向上生长的空间
 [ ] 被滋养和支持

想补充说说吗？（可选）___________"

→ LLM 根据 Q2 回答生成 Q3

Q3: LLM 追问 - 开放文本
────────────────────────────────────────────────────────────────
示例:
"你提到在感情中需要安全感。
 最近有什么事情让你感到不安全吗？"

___________________________________
```

**核心设计**:
- 问题要「触动人心」，让用户感到被理解
- 问题基于命盘特征个性化生成
- 逐步深入，第三个问题引导用户敞开心扉
```
