  当前实现的流程

  用户访问 vibelife.app
         ↓
  ┌─────────────────────┐
  │ 是否有 Cookie/子域名？│
  └─────────┬───────────┘
            ├─ 否 → Skill 选择器（5个卡片）
            │       ↓ 选择
            │       保存 Cookie + 跳转
            │
            └─ 是 → 直接跳转 /bazi 或 /zodiac
                    ↓
            ┌──────────────┐
            │ Landing Page │  ← 服务介绍 + CTA
            └──────┬───────┘
                   ↓ 点击"开始对话"
            ┌──────────────┐
            │   Chat Page  │  ← 主对话界面
            └──────────────┘
